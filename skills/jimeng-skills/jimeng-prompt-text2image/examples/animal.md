# Animal — 动物

## Instructions

动物类提示词的关键是**物种特征精准**和**姿态生动**。即梦对常见动物（猫、狗、马、鸟、鱼）的理解较好，对稀有物种可能偏差。核心策略：用具体物种名称 + 特征细节 + 自然环境或纯色背景。

## Scenario Patterns

| 子场景 | 配方侧重 | 典型触发词 |
|--------|---------|-----------|
| 野生动物 | 物种+栖息地+动态+自然光 | 野生, 自然, 捕猎, 栖息 |
| 宠物 | 品种+表情+互动+居家/户外 | 猫, 狗, 可爱, 萌, 宠物 |
| 鸟类 | 羽色+飞行/栖息+环境 | 鸟, 飞翔, 羽毛, 树枝 |
| 海洋生物 | 物种+水下光线+环境 | 鱼, 鲸, 珊瑚, 海底 |
| 昆虫/微距 | 物种+微距细节+植物环境 | 蝴蝶, 蜜蜂, 微距, 露珠 |
| 幻想生物 | 物种组合+奇幻特征+风格 | 龙, 独角兽, 神话 |

## Examples

### Example: 雪豹

**用户需求**：帮我写一个雪豹在山岩上的野生动物摄影提示词

**公式拆解**：
- 主体：雪豹 + 灰白毛+黑色斑纹 + 浅蓝眼睛 + 尾巴环绕
- 场景：雪山岩石
- 姿态：休息/警觉远望
- 风格：自然摄影
- 光线：自然散射光
- 构图：微距特写
- 画质：毛发细节分毫毕现

**最终提示词**：
```
一只雪豹在雪山的岩石上休息，灰白色的毛发上有黑色斑纹，尾巴环绕身体，
浅蓝色的眼睛警觉地望向远方，微距特写，毛发细节分毫毕现，自然摄影风格
```

**为何这样写**：物种特征（灰白毛+黑斑纹+浅蓝眼）是雪豹的识别关键；"尾巴环绕身体"增加画面构图的闭合感

### Example: 蜂鸟悬停

**用户需求**：写一个蜂鸟在花前采蜜的动态瞬间

**公式拆解**：
- 主体：蜂鸟 + 虹彩羽毛 + 翅膀扇动
- 场景：红色花朵前
- 动作：悬停（翅膀高速扇动虚化）
- 风格：长焦微距
- 光线：阳光照射羽毛闪烁
- 构图：极浅景深 + 背景完全虚化
- 画质：极致动态捕捉

**最终提示词**：
```
一只蜂鸟悬停在红色花朵前，翅膀高速扇动产生虚影效果，虹彩般的羽毛在阳光下闪烁，
金黄绿色为主，长焦微距，极浅景深，背景完全虚化，极致动态捕捉
```

**为何这样写**：翅膀虚影是蜂鸟区别于其他鸟类的核心特征；虹彩羽毛需要"在阳光下闪烁"来表现金属光泽

### Example: 萌宠猫咪

**用户需求**：想要一张可爱的布偶猫在窗边晒太阳的照片

**公式拆解**：
- 主体：布偶猫 + 蓝色眼睛 + 长毛 + 慵懒姿态
- 场景：窗边 + 阳光洒入
- 风格：日系清新宠物摄影
- 光线：温暖自然光
- 构图：浅景深
- 画质：毛发蓬松质感

**最终提示词**：
```
一只布偶猫慵懒地躺在窗边的垫子上，蓝色的大眼睛半眯着，长毛在阳光下呈现蓬松质感，
温暖的自然光从窗户洒入，照亮毛发边缘，日系清新宠物摄影风格，浅景深，毛发细节丰富
```

### Example: 锦鲤

**用户需求**：池塘里的锦鲤，要有东方美学

**公式拆解**：
- 主体：锦鲤 + 红白黑三色 + 鳞片闪光
- 场景：池塘 + 莲叶+荷花
- 视角：俯拍
- 细节：水面波纹+倒影 + 水下通透
- 风格：自然摄影
- 光线：阳光穿透水面

**最终提示词**：
```
池塘中游动的锦鲤，红白黑三色相间，鳞片在阳光下闪闪发光，水面的波纹和倒影，
周围有莲叶和盛开的荷花，俯拍构图，通透清澈的水下细节
```

## Category-Specific Tips

1. **物种名称要准确** — "布偶猫"比"好看的猫"效果好得多，即梦对常见品种识别度高
2. **动物表情靠姿态传达** — 不要写"开心的狗"，写"吐着舌头、尾巴翘起、耳朵竖起"
3. **毛/羽/鳞的质感词是加分项** — "毛发蓬松""羽毛有光泽""鳞片反光"等质感描述显著提升效果
4. **飞翔/跳跃/奔跑等动态词放在动作位置** — 放在场景描述之前，让模型优先理解姿态
5. **昆虫类务必加"微距"** — 否则即梦可能生成全景而非特写
