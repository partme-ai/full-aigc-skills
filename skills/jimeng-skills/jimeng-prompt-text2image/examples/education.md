# Education — 教育/校园/学习

## Instructions

教育类提示词的核心是**学术氛围和求知感**。图书馆的古籍书香、教室的晨光洒在课桌上、实验室的精密仪器、毕业典礼的激动——每个场景都承载着知识与成长的情感。光线通常柔和温暖，构图经典稳重。

## Scenario Patterns

| 子场景 | 配方侧重 | 典型触发词 |
|--------|---------|-----------|
| 教室/课堂 | 课桌+黑板+学生+晨光 | 教室, 上课, 课堂, 学校 |
| 图书馆/阅读 | 书架+书桌+台灯+静谧 | 图书馆, 阅读, 自习, 书 |
| 毕业/学位 | 学士服+学位帽+校园+抛帽 | 毕业, 学位, 毕业照 |
| 科学实验 | 试管+显微镜+实验台+专注 | 实验, 化学, 物理, 生物 |
| 大学校园 | 草坪+钟楼+学生+自行车 | 校园, 大学, 象牙塔 |
| 考试/学习 | 笔记+笔+深夜+台灯+咖啡 | 学习, 考试, 复习, 自习 |

## Examples

### Example: 图书馆阅读

**用户需求**：写一个古老图书馆中的阅读场景

**公式拆解**：
- 场景：古老大学图书馆 + 高书架+木梯
- 主体：学生在书桌前阅读
- 光线：穹顶采光+绿色台灯
- 氛围：静谧求知的殿堂
- 风格：纪实摄影

**最终提示词**：
```
古老大学图书馆的圆形阅览室，高耸的木质书架从地面延伸到穹顶，一架木梯靠在书架上，
学生坐在橡木书桌前低头阅读一本厚重的旧书，绿色玻璃台灯发出温暖的光照亮书页，
穹顶采光窗透入柔和的自然光，空气中漂浮着古老的纸墨香，静谧而神圣的求知殿堂，
纪实摄影风格
```

### Example: 毕业典礼

**用户需求**：写一个大学毕业典礼的激动瞬间

**公式拆解**：
- 主体：毕业生 + 黑色学士服+学士帽
- 动作：抛起学士帽 + 欢呼
- 场景：大学校园草坪 + 古老建筑
- 光线：夏日阳光
- 氛围：激动喜悦
- 构图：仰拍

**最终提示词**：
```
大学毕业典礼上，一群穿着黑色学士服戴着学士帽的毕业生将帽子抛向天空，
仰拍视角捕捉学士帽在蓝天下飞舞的瞬间，年轻的脸庞上洋溢着激动和喜悦的笑容，
背景是校园古老的红砖钟楼，夏日阳光灿烂，毕业季纪实摄影，青春洋溢
```

### Example: 科学实验

**用户需求**：写一个化学实验室中学生做实验的场景

**公式拆解**：
- 场景：化学实验室 + 试管架+烧杯+试剂
- 主体：学生 + 白大褂+护目镜
- 动作：专注滴入试剂 + 溶液变色
- 光线：实验室白色顶光
- 氛围：科学探索
- 构图：中景

**最终提示词**：
```
化学实验室中，一位穿白大褂戴护目镜的学生正在专注地进行滴定实验，手持滴定管
向锥形瓶中的溶液逐滴加入试剂，透明的溶液正在转变为粉红色，实验台上整齐排列着
试管架和各种试剂瓶，干净的实验室白色灯光，科学探索的专注氛围
```

## Category-Specific Tips

1. **图书馆光线=穹顶光+台灯暖光** — 这两种光的混合是图书馆场景的经典布光
2. **教室要有"时间的痕迹"** — 斑驳的黑板边缘、磨损的课桌、泛黄的墙壁，让画面有真实历史感
3. **毕业=蓝天下+抛帽** — 这是毕业摄影的经典符号，几乎必不可少
4. **实验室要写器材细节** — "精确刻度的量筒""玻璃棒搅拌""蓝色酒精灯火焰"，科学感来自精密
5. **校园四季分明** — 春天的樱花、夏天的梧桐、秋天的银杏、冬天的雪景，季节定校园氛围
