# Food & Product — 美食/产品

## Instructions

美食/产品类提示词的核心是**质感诱惑**。这类图片的目的通常是商业展示或社交分享，所以质感、光泽、新鲜度是重中之重。关键是让材质"可感知"——酥脆、丝滑、多汁、晶莹——以及精确的布光描述。

## Scenario Patterns

| 子场景 | 配方侧重 | 典型触发词 |
|--------|---------|-----------|
| 高级西餐 | 摆盘+酱汁+微焦点缀+俯拍 | 西餐, 牛排, 法餐, fine dining |
| 甜品烘焙 | 层次+糖粉+奶油+侧光 | 甜品, 蛋糕, 烘焙, 甜点 |
| 中式料理 | 锅气+红油+热气+色彩 | 中餐, 炒菜, 火锅, 家常菜 |
| 饮品 | 冰块+水珠+玻璃器皿+逆光 | 咖啡, 鸡尾酒, 茶, 饮品 |
| 产品摄影 | 材质+高光+纯色背景+微距 | 手表, 香水, 电子产品, 电商 |
| 静物 | 摆放+光影+质感+意境 | 静物, 花卉, 书本, 器物 |

## Examples

### Example: 高级西餐

**用户需求**：写一个惠灵顿牛排切开瞬间的美食摄影

**公式拆解**：
- 主体：惠灵顿牛排 + 金黄酥皮层次 + 粉红内里
- 摆盘：白瓷盘 + 酱汁弧线 + 微型蔬菜+金箔
- 构图：俯拍
- 风格：美食摄影
- 光线：柔和顶光
- 画质：极致质感

**最终提示词**：
```
一份精致的惠灵顿牛排被切开，酥皮金黄层次分明，内里的菲力牛排呈现完美粉红色，
酱汁在白色瓷盘上画成弧线，点缀微型蔬菜和金箔，俯拍构图，美食摄影质感
```

**为何这样写**：切开瞬间展示内部层次是关键；金黄酥皮vs粉红内里的色彩对比制造食欲；酱汁弧线增加画面动感

### Example: 法式甜品

**用户需求**：写一个草莓千层酥的甜品摄影

**公式拆解**：
- 主体：草莓千层酥 + 起酥层次 + 草莓+卡仕达酱夹层
- 动态细节：糖粉飘落
- 配景：卡布奇诺
- 光线：左侧柔光
- 构图：侧拍展示层次
- 风格：美食杂志

**最终提示词**：
```
法式甜点草莓千层酥，起酥薄脆的层次，新鲜草莓和卡仕达酱夹层，糖粉如雪飘落表面，
旁边有一杯卡布奇诺，柔和自然光从左侧打来，侧拍展示层次，美食杂志风格
```

**为何这样写**："糖粉飘落"是一个动态细节，让静态摄影有了瞬间感；侧拍是为了展示千层的截面层次

### Example: 中式炒菜

**用户需求**：写一个刚出锅的宫保鸡丁

**公式拆解**：
- 主体：宫保鸡丁 + 红油光亮 + 鸡丁嫩滑 + 花生酥脆
- 配料：干辣椒+花椒点缀
- 动态：升腾热气
- 器皿：白瓷盘
- 光线：顶光突出光泽质感
- 风格：中式美食摄影
- 色彩：鲜艳真实

**最终提示词**：
```
一锅刚出锅的宫保鸡丁，红油光亮，鸡丁嫩滑，花生酥脆，干辣椒和花椒点缀，升腾的热气，
白瓷盘承载，顶光拍摄突出光泽和质感，中式美食摄影，色彩鲜艳真实
```

**为何这样写**："升腾的热气"传达"刚出锅"的时效感；红油光亮是中式炒菜的核心视觉特征

### Example: 手表微距

**用户需求**：瑞士机械表的微距特写，要高级感

**公式拆解**：
- 主体：瑞士机械表 + 精密齿轮游丝
- 细节：蓝宝石镜面蓝光反射 + 皮质表带缝线
- 光线：深色背景+单一光源
- 构图：极浅景深微距
- 风格：产品广告摄影

**最终提示词**：
```
瑞士机械表的微距特写，精密齿轮和游丝清晰可见，蓝宝石镜面反射出迷人的蓝光，
皮质表带的缝线纹理，深色背景中单一光源打亮表盘，极浅景深，产品广告摄影风格
```

## Category-Specific Tips

1. **食物要"动"起来** — 热气、糖粉飘落、酱汁流淌、切开瞬间，动态细节让食物摄影不再死板
2. **光泽=食欲** — "油光""光亮""晶莹""水润"是食物摄影的高频词
3. **产品背景要干净** — 纯色背景或深色背景，不抢产品风头。可用"干净简洁的深灰背景""纯白无缝背景"
4. **材质词即卖点** — 产品摄影中，"拉丝金属""碳纤维纹理""意大利真皮缝线"等材质描述比外观更重要
5. **构图角度决定展示重点** — 俯拍=展示全貌和摆盘，侧拍=展示层次，45度=模拟人眼视角最自然
