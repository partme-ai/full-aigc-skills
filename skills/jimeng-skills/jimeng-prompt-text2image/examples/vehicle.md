# Vehicle — 交通工具

## Instructions

交通工具类提示词的核心是**机械美感和速度感**。汽车摄影强调漆面光泽、车身线条、轮毂细节；飞机摄影强调云层之上的壮丽；摄影中的关键是用光来表现金属和玻璃的质感——棚拍用渐变柔光，动态用追焦。

## Scenario Patterns

| 子场景 | 配方侧重 | 典型触发词 |
|--------|---------|-----------|
| 超跑/豪车 | 车身线条+漆面反射+轮毂+棚拍/山路 | 超跑, 跑车, 法拉利, 兰博基尼 |
| 经典老车 | 复古车型+铬饰条+傍晚光+情怀 | 老爷车, 经典车, vintage car |
| 摩托车 | 机车+骑手+皮衣+公路+自由 | 摩托, 哈雷, 机车, 骑行 |
| 飞机/航空 | 客机+云层+跑道+蓝天 | 飞机, 客机, 起飞, 云上 |
| 游艇/帆船 | 船体+海面+阳光+奢华 | 游艇, 帆船, 出海 |
| 自行车 | 自行车+骑手+山路/城市+健康 | 骑行, 自行车, 山地车 |

## Examples

### Example: 超跑棚拍

**用户需求**：写一辆红色法拉利在棚拍中的广告级大片

**公式拆解**：
- 主体：红色法拉利超跑 + 流线型车身+五辐轮毂
- 场景：专业汽车影棚
- 光线：渐变柔光+漆面高光+地面倒影
- 构图：45度前侧+低角度
- 风格：汽车广告摄影
- 画质：8k

**最终提示词**：
```
红色法拉利超跑在专业汽车影棚中，流畅的车身线条从前翼子板延伸到尾部，
深红色金属漆面在渐变柔光下呈现出丰富的高光和暗面过渡，五辐合金轮毂后的
红色刹车卡钳醒目，纯黑地面倒映出车身的完美镜像，45度前侧低角度构图，
汽车广告摄影风格，8k极致细节
```

### Example: 哈雷摩托车

**用户需求**：写一个哈雷摩托车在公路上的自由画面

**公式拆解**：
- 主体：哈雷巡航摩托 + 镀铬发动机+V型双缸
- 骑手：皮衣+半盔+墨镜
- 场景：旷野公路 + 日落+远山
- 氛围：自由奔放
- 风格：机车旅行摄影

**最终提示词**：
```
一辆黑色哈雷巡航摩托车行驶在旷野的公路上，镀铬的V型双缸发动机闪着金属光泽，
骑手穿着黑色皮衣戴着半盔和墨镜，公路笔直延伸向远方红色的砂岩山脉，
壮丽的日落天空在背景中展开，机车旅行摄影风格，自由奔放的美式公路精神
```

### Example: 私人游艇

**用户需求**：写一艘白色豪华游艇在碧海上的画面

**公式拆解**：
- 主体：白色豪华游艇 + 流线船体+飞桥
- 场景：蔚蓝海面 + 晴朗天空
- 光线：正午阳光 + 水面波光
- 构图：侧面全景
- 风格：游艇生活摄影

**最终提示词**：
```
白色豪华游艇在蔚蓝的地中海上航行，流线型船体劈开海面留下白色尾迹，
飞桥上有人在享受阳光和海风，晴朗天空下海面闪烁着星点般的波光，
侧面全景构图，游艇生活摄影风格，奢华而自由的海洋生活方式
```

## Category-Specific Tips

1. **车型要写具体** — "保时捷911 Turbo S""哈雷戴维森Fat Boy"比"跑车""摩托车"精准
2. **漆面和金属反光是质感关键** — "金属漆面在光线下呈现渐变层次""镀铬部件闪着冷光"
3. **场景定调** — 影棚=精致广告感，山路=运动性能，公路=旅行自由，城市=奢华生活
4. **动态要有追焦感** — "背景拖出运动模糊""轮毂旋转虚化"传达速度
5. **飞机用云端制造壮丽感** — "穿过云层""云海之上""夕阳在机翼上反光"
