# Night Photography — 夜景/暗光摄影

## Instructions

夜景类提示词的核心是**光与暗的博弈**。暗光环境下，光源本身成为构图的主体——霓虹灯、车灯轨迹、星空、烛光、烟花。关键是明确主光源和它的色彩温度，以及暗部的层次处理。

## Scenario Patterns

| 子场景 | 配方侧重 | 典型触发词 |
|--------|---------|-----------|
| 城市夜景 | 建筑灯光+车轨+倒影+蓝调时刻 | 夜景, 城市夜景, blue hour |
| 霓虹灯街 | 霓虹招牌+湿润地面+冷暖对比 | 霓虹, 赛博朋克, 霓虹灯 |
| 星空/银河 | 银河+前景+长曝光+极暗环境 | 星空, 银河, 星轨, 夜空 |
| 烟花 | 烟花绽放+城市/水面+倒影 | 烟花, 烟火, 跨年 |
| 烛光/暗调 | 烛光+暗部细节+温暖氛围 | 烛光, 暗调, 氛围, 温馨 |
| 光绘/车轨 | 光轨+长曝光+城市道路 | 光绘, 车轨, 光轨, 慢门 |

## Examples

### Example: 蓝调时刻

**用户需求**：写一个蓝调时刻的城市天际线

**公式拆解**：
- 时间：日落后蓝调时刻（blue hour）
- 场景：城市天际线 + 建筑灯光开始亮起
- 水面：河流 + 倒影
- 色彩：深蓝天空 vs 暖黄城市灯光
- 风格：城市风光摄影
- 构图：全景 + 长曝光

**最终提示词**：
```
蓝调时刻的城市天际线，深蓝色的暮光天空与城市建筑的暖黄灯光形成冷暖对比，
摩天楼群的窗户星罗棋布地亮着，河流平滑如镜完美倒映着建筑灯光，
城市风光摄影风格，全景长曝光，宁静而壮丽
```

### Example: 烟花大会

**用户需求**：写一个跨年烟花绽放的壮观画面

**公式拆解**：
- 主体：大型烟花 + 多色绽放+拖尾
- 场景：城市夜空 + 水面倒影
- 前景：人群剪影仰望
- 光线：烟花照亮夜空
- 风格：节庆摄影
- 氛围：壮观喜庆

**最终提示词**：
```
跨年烟花在城市夜空绽放，金色、红色和紫色的烟花在夜幕中形成巨大的花朵图案，
烟花拖尾的光迹交织重叠，河面完美倒映着烟花的绚烂光影，
前景人群的剪影在烟花光芒中仰望天空，节庆摄影风格，壮丽喜庆，长曝光捕捉烟花轨迹
```

### Example: 烛光氛围

**用户需求**：写一个多支蜡烛的温馨暗调静物

**公式拆解**：
- 主体：多支蜡烛 + 烛光摇曳
- 场景：深色木质桌面
- 点缀：散落的花瓣 + 旧书
- 光线：烛光为唯一光源
- 氛围：温暖静谧
- 构图：俯拍 + 暗调

**最终提示词**：
```
深色木质桌面上摆放着多支点燃的蜡烛，烛光摇曳在周围投下微弱的暖色光芒，
桌面散落着干枯玫瑰花瓣和一本翻开的旧书，蜡烛的暖光照亮书页边缘，暗调摄影风格，
极简构图，温暖静谧的氛围，暗部保留细微纹理
```

## Category-Specific Tips

1. **蓝调时刻只有约20分钟** — 日落后天空呈深蓝色的短暂窗口，是城市夜景最佳拍摄时间
2. **长曝光是关键** — 车轨、星轨、烟花轨迹、丝滑水面都需要"长曝光"关键词
3. **暗部要保留细节** — 写"暗部保留纹理"而非"全黑"，避免死黑
4. **光源色温决定氛围** — 暖光（2000-3500K）=温馨，冷光（5000K+）=科技/孤独
5. **水面倒影=双倍光效** — 夜景中加入水面反射，等于拥有了两个光源层
