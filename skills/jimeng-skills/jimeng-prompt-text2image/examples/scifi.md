# Sci-Fi — 科幻/未来

## Instructions

科幻类提示词的核心是**科技感的可信度**。金属材质、全息投影、能量武器、外星环境——这些元素需要具体的技术细节来描述。关键是未来科技和当下可辨识元素的平衡，让科幻世界既超前又可感知。

## Scenario Patterns

| 子场景 | 配方侧重 | 典型触发词 |
|--------|---------|-----------|
| 太空/星际 | 飞船+星云+行星+宇航员+浩瀚 | 太空, 宇宙, 星际, 星舰 |
| 外星世界 | 异星地貌+双太阳+奇异植物+异色 | 外星, 异星, 外星球 |
| 机器人/AI | 金属机体+发光传感器+拟人/非人 | 机器人, AI, 机甲, 人形机 |
| 赛博朋克 | 霓虹+义体+全息+雨夜+巨企 | 赛博朋克, cyberpunk, 义体 |
| 废土/末世 | 废墟+黄沙+破烂装备+生存 | 废土, 末世, 核战后, 荒原 |
| 未来都市 | 飞行器+垂直城市+全息广告+悬浮 | 未来城, 未来都市, 科幻城 |

## Examples

### Example: 星际战舰

**用户需求**：写一个巨型星舰在星云中航行的史诗画面

**公式拆解**：
- 主体：巨型星际战舰 + 金属装甲+炮塔阵列+引擎蓝焰
- 场景：紫色星云 + 远处恒星
- 点缀：小型护航舰
- 光线：星云反射光+引擎光+恒星远光
- 构图：全景+体积对比
- 风格：科幻概念艺术

**最终提示词**：
```
巨型星际战舰在紫色星云中缓缓航行，深灰色金属装甲上布满武器阵列和信号灯，
尾部离子引擎喷出蓝色火焰，舰体表面反射着星云的紫红色光芒，数十艘小型护航舰
如鱼群般环绕，远处一颗白矮星发出冷白光，科幻概念艺术风格，史诗般的太空歌剧
```

### Example: 赛博朋克义体人

**用户需求**：写一个赛博朋克风格的义体改造人

**公式拆解**：
- 主体：义体改造人 + 半张脸是机械 + 发光义眼
- 细节：颈部金属脊椎接口 + 手臂机械纹路
- 场景：雨夜霓虹街道
- 光线：霓虹蓝紫+全息投影光
- 风格：赛博朋克写实
- 氛围：冷峻

**最终提示词**：
```
赛博朋克风格的义体改造人特写，半张脸被精密的银色机械替代，橙色的义眼发出微光，
颈部暴露出金属脊椎接口和发光的数据线，皮肤上有电路纹身发出蓝光，站在雨夜的
霓虹灯街道上，霓虹蓝紫光和全息广告的光影投射在金属面庞上，赛博朋克写实风格，超写实
```

### Example: 废土末世

**用户需求**：写一个核战后废土世界的幸存者

**公式拆解**：
- 场景：废墟城市 + 黄沙掩埋+生锈钢筋
- 主体：幸存者 + 拼接护甲+防毒面罩+改装步枪
- 天空：昏黄色 + 辐射云
- 风格：废土末世
- 氛围：苍凉绝望

**最终提示词**：
```
核战争后的废土世界，曾经的城市只剩断壁残垣被黄沙半掩，生锈的钢筋从混凝土中伸出
如骨骸，天空被辐射尘埃染成昏黄色，一名幸存者穿着轮胎和铁片拼接的护甲，
戴着老旧的防毒面罩，手持改装的步枪穿越废墟，废土末世风格，苍凉而绝望的氛围
```

## Category-Specific Tips

1. **科技要写具体材质和形态** — 不写"高科技"，写"碳纤维编织的装甲""量子计算的蓝色光路""磁悬浮的反重力引擎"
2. **太空要有尺度感** — 星舰旁加护航舰、宇航员旁加巨大飞船，让观者感知规模
3. **赛博朋克=霓虹+雨夜+义体** — 这三个元素组合基本就是赛博朋克
4. **废土=锈迹+破烂+黄沙** — 废土的美学核心是"破败的美"
5. **外星环境要反常** — 紫色天空、双太阳、蓝色植被、环形星环，打破地球日常经验
