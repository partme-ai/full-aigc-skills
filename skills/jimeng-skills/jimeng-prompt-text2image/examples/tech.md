# Technology — 科技/数码

## Instructions

科技类提示词的核心是**精密感和未来感**。产品摄影强调材质（铝合金、玻璃、碳纤维）、精密排布的元件、以及干净富有科技感的布光。概念科技则更大胆——透明屏幕、全息交互、纳米材料。

## Scenario Patterns

| 子场景 | 配方侧重 | 典型触发词 |
|--------|---------|-----------|
| 手机/平板 | 超薄机身+屏幕+摄像头模组+棚拍 | 手机, iPhone, 平板 |
| 电脑/笔记本 | 铝合金外壳+键盘背光+屏幕显示 | 笔记本, MacBook, 电脑 |
| 芯片/半导体 | 晶圆+电路+针脚+显微+蓝光 | 芯片, CPU, 晶圆, 半导体 |
| 智能穿戴 | 手表+耳机+眼镜+佩戴场景 | 手表, 耳机, AR眼镜, 穿戴 |
| 概念科技 | 全息投影+透明屏幕+未来交互 | 概念, 未来科技, 全息 |
| 数据/网络 | 服务器+光纤+数据中心+蓝光 | 数据中心, 服务器, 云 |

## Examples

### Example: 芯片微距

**用户需求**：写一个CPU芯片的微距特写

**公式拆解**：
- 主体：CPU芯片 + 晶圆+针脚阵列+电路纹理
- 光线：蓝紫色侧光 + 硅晶反光
- 构图：极微距 + 纯黑背景
- 风格：科技产品摄影
- 画质：显微级细节

**最终提示词**：
```
CPU芯片的极微距特写，硅晶圆表面密布着纳米级的电路纹理，金色针脚阵列如城市天际线
般精密排列，蓝紫色侧光照亮晶圆表面呈现出彩虹般的硅晶反光，纯黑背景，
显微级细节，科技产品摄影风格，半导体工业之美
```

### Example: 概念全息

**用户需求**：写一个未来全息交互界面的概念场景

**公式拆解**：
- 主体：全息投影界面 + 半透明蓝色UI+手势交互
- 用户：剪影+手在虚空中滑动
- 场景：暗调科技空间
- 光线：全息蓝光为主要光源
- 风格：科技概念设计

**最终提示词**：
```
未来科技空间中悬浮的全息交互界面，半透明的蓝色UI面板漂浮在空中，数据流动的光点
粒子如星河流转，一只手在虚空中做出滑动和捏合的手势操控界面，全息蓝光映在
操作者的面部和手掌上，暗调科技背景，未来科技概念设计风格，科幻而优雅
```

### Example: 智能手表

**用户需求**：写一个智能手表的户外生活场景

**公式拆解**：
- 主体：智能手表 + 钛金属表壳+运动表带
- 场景：晨跑中 + 手腕佩戴
- 光线：清晨户外光
- 细节：屏幕上显示跑步数据
- 风格：科技生活方式摄影

**最终提示词**：
```
智能手表佩戴在跑步者的手腕上，钛金属表壳在晨光中泛着微光，OLED屏幕显示着实时的
心率数据和跑步里程，橙色运动表带贴合手腕，背景虚化为清晨的林间跑道，
科技生活方式摄影风格，浅景深，运动与科技的融合
```

## Category-Specific Tips

1. **芯片/半导体要写光色** — 蓝紫光（紫外光刻的联想）是芯片摄影的首选光源色
2. **全息/概念科技用透明+发光** — "半透明""悬浮""粒子光点""数据流动"是未来科技的视觉标志
3. **数码产品要写材质** — "钛金属""陶瓷""微晶玻璃""芳纶纤维"，材质决定高级感
4. **科技产品=极简背景+精密光** — 黑色/白色背景，光源要干净没有多余的散射
5. **数据中心用蓝-绿-白LED阵列** — 服务器的指示灯矩阵是数据中心摄影的标志性视觉
