# Underwater — 水下/海洋

## Instructions

水下类提示词的核心是**光线在水中的行为**——折射、散射、体积光（耶稣光在水下）。蓝色/青色是主色调，关键是描述水下的能见度、气泡、漂浮颗粒和生物的悬浮感。

## Scenario Patterns

| 子场景 | 配方侧重 | 典型触发词 |
|--------|---------|-----------|
| 珊瑚礁 | 珊瑚色彩+热带鱼+阳光穿透+浅海 | 珊瑚, 热带鱼, 浅海, 潜水 |
| 深海/沉船 | 暗蓝+沉船遗迹+大型生物+幽暗 | 深海, 沉船, 鲸, 鲨鱼 |
| 自由潜水 | 潜水员+气泡+光束+悬浮感 | 自由潜, 美人鱼, 水下摄影 |
| 水下洞穴 | 钟乳石+光束+幽蓝+探险 | 水下洞穴, 蓝洞 |
| 水母群 | 水母+透明+发光+漂浮 | 水母, 发光, 浮游 |
| 极地冰下 | 冰层+冰裂缝+光线+极地生物 | 冰下, 北极, 南极 |

## Examples

### Example: 珊瑚礁

**用户需求**：写一个热带珊瑚礁的水下画面

**公式拆解**：
- 场景：浅海珊瑚礁 + 阳光穿透水面
- 生物：五彩热带鱼群 + 海龟
- 光线：水面投射的波光+光束（耶稣光）
- 色彩：蓝绿色主调 + 珊瑚暖色点缀
- 细节：水中浮游微粒+气泡
- 风格：水下自然摄影

**最终提示词**：
```
热带浅海珊瑚礁的壮丽水下景观，阳光从水面穿透形成波动的光束，鹿角珊瑚和脑珊瑚
在海底铺展，橙色、紫色和绿色的珊瑚形成丰富的色彩层次，一群蓝黄色的热带鱼游过，
一只海龟悠闲地滑翔，水中漂浮着微小的浮游颗粒和气泡，水下摄影风格，通透的海水质感
```

### Example: 自由潜水

**用户需求**：写一个自由潜水员在光束中的唯美画面

**公式拆解**：
- 主体：自由潜水员 + 长脚蹼+流畅泳姿
- 场景：深蓝海水 + 垂直光束
- 光线：上方投下的耶稣光柱
- 氛围：静谧悬浮感
- 构图：仰拍 + 剪影效果
- 风格：水下艺术摄影

**最终提示词**：
```
自由潜水员在深蓝的海水中优雅下潜，长脚蹼和流线型身姿在水中划出流畅的弧线，
头顶上方投下数道金色的耶稣光柱穿透海水，潜水员的剪影悬浮在光束中，
气泡缓缓上升，水下艺术摄影风格，静谧超凡，仰拍视角突出人的渺小与海的深邃
```

### Example: 水母群

**用户需求**：写一个深海发光水母群的梦幻画面

**公式拆解**：
- 主体：发光水母群 + 透明伞状体+荧光触手
- 场景：深暗海水
- 光线：水母自身的生物荧光
- 色彩：暗蓝+荧光粉/紫
- 氛围：奇幻神秘
- 风格：水下微距摄影

**最终提示词**：
```
深海中漂浮的水母群，数十只透明伞状体的水母在幽暗的海水中发出柔和的粉色和紫色
生物荧光，细长的触手如丝带般在水中飘舞，光点散布在深蓝的背景下如星空，
水下微距摄影，奇幻神秘的深海世界
```

## Category-Specific Tips

1. **海水颜色要分层** — 浅处蒂芙尼蓝/绿 → 中层湛蓝 → 深处墨蓝/黑
2. **光束（耶稣光）是水下灵魂** — "阳光穿透水面形成的光束"是水下摄影的标志性元素
3. **气泡和浮游微粒增加真实感** — "缓缓上升的气泡""水中的浮游微粒"让水下场景可信
4. **生物要写出品种名** — "小丑鱼在海葵中"比"鱼在珊瑚中"精准
5. **水下能见度是可调的** — "通透的海水"vs"略带浑浊的绿色海水"决定氛围
