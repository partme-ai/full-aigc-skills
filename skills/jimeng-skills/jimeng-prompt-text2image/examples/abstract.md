# Abstract & Creative — 抽象/概念/艺术创意

## Instructions

抽象/创意类提示词是**最自由也最难写好**的类别。因为没有"像照片一样"这个锚点，需要自己建立视觉逻辑。核心策略：先确定一个**概念或情绪** → 找到可视觉化的**隐喻/符号** → 选择**风格和色彩体系**来承载。

## Scenario Patterns

| 子场景 | 配方侧重 | 典型触发词 |
|--------|---------|-----------|
| 超现实主义 | 不可能的组合+写实质感+梦境逻辑 | 超现实, 梦境, 荒诞, 魔幻 |
| 概念艺术 | 抽象概念+象征符号+极简表达 | 概念, 象征, 极简, 隐喻 |
| 奇幻世界 | 架空世界+幻想物种+史诗感 | 奇幻, 魔法, 异世界, 龙 |
| 科幻概念 | 未来科技+外星+太空+未知 | 科幻, 外星, 太空, 未来 |
| 情绪可视化 | 情绪→色彩→形状→质感 | 孤独, 焦虑, 喜悦, 宁静 |
| 几何抽象 | 几何形状+色彩+构图+节奏 | 几何, 抽象, 色块, 构成 |

## Examples

### Example: 超现实梦境

**用户需求**：写一个超现实的梦境画面，鲸鱼在云海中游动

**公式拆解**：
- 概念：不可能的组合——海洋生物在天空中
- 主体：巨鲸 + 云海
- 风格：超现实主义 + 写实质感
- 光线：金色夕阳从下方照亮云层
- 色彩：暖金+淡蓝
- 构图：大远景
- 氛围：梦幻震撼

**最终提示词**：
```
超现实主义画面，巨大的鲸鱼在金色的云海中缓缓游动，鲸尾拨开云雾泛起涟漪般的波纹，
夕阳从云层下方透出金色光芒，暖金与淡蓝的渐变天空，写实质感，梦幻而震撼
```

**为何这样写**：超现实的关键是"逻辑错位但质感写实"——鲸鱼的行为（游动）和质感（真实皮肤纹理）是写实的，但环境（天空云海）是错位的；云海涟漪是连接两个世界的细节桥梁

### Example: 情绪可视化——孤独

**用户需求**：把"孤独感"可视化为一幅画

**公式拆解**：
- 情绪概念：孤独
- 视觉隐喻：单人 + 巨大空旷空间
- 场景：空旷美术馆 + 一幅小画
- 光线：单一顶光 + 大面积阴影
- 色彩：冷灰+暗蓝
- 风格：极简摄影
- 构图：人物极小 vs 空间极大

**最终提示词**：
```
空旷的美术馆大厅中，一个孤独的身影站在巨大的白墙前，墙上只挂着一幅极小的画，
单一顶光在身影周围投下长长的影子，冷灰色调，极简构图，负空间占据画面的绝大部分，
孤独而宁静的氛围
```

**为何这样写**：用空间比例（小人+大空间）视觉化"孤独"；负空间（大量空白）制造空虚感就是孤独本身

### Example: 赛博朋克猫咪（跨类别创作）

**用户需求**：写一个赛博朋克风格的机械猫 —— 注意：这个场景不在任何单一示例文件中，需要用方法论组合

**公式拆解**：
- 主体：猫（动物类）+ 赛博朋克机械改造（科幻）
- 特征：金属外壳 + 霓虹发光纹路 + 一只眼为机械义眼
- 场景：雨夜霓虹街道
- 风格：赛博朋克 + 写实
- 光线：霓虹蓝紫 + 湿润地面反射
- 画质：超写实

**最终提示词**：
```
一只赛博朋克风格的机械猫蹲在雨夜的霓虹灯招牌下，金属外壳上有细密的刻线，
身体侧面的霓虹蓝色发光纹路，一只眼睛是橙色的机械义眼，雨滴落在金属外壳上形成水珠，
霓虹灯在湿润地面反射出彩色光斑，赛博朋克风格，超写实
```

**为何这样写**：动物（猫）+风格（赛博朋克）的跨类组合；保留了猫的"蹲"姿态（动物类）同时加入金属外壳和义眼（科幻）；雨夜湿润地面反射霓虹是赛博朋克的环境标配

## Category-Specific Tips

1. **概念先于画面** — 先想清楚"我要表达什么"，再找视觉隐喻。不要跳过概念直接堆砌视觉元素
2. **写实质感是超现实的基石** — 越是不可置信的场景，越需要写实的材质和光线来让它"可信"
3. **色彩即情绪** — 抽象画面中色彩体系就是内容本身，比具象画面更依赖色彩传递信息
4. **跨类别组合是最强创造力工具** — 动物+赛博朋克、国风+科幻、美食+建筑...大胆组合不同类别的元素
5. **负空间是强大的构图工具** — 抽象/概念类尤其需要留白来制造呼吸感和强调主体
