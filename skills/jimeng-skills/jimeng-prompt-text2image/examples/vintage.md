# Vintage/Retro — 怀旧/复古

## Instructions

怀旧类提示词的核心是**时间痕迹和年代美学**。胶片颗粒、褪色、暖黄调、暗角——这些视觉老化元素是复古感的基石。关键是明确年代锚点：民国、昭和、80年代、90年代、Y2K——每个年代有独特的视觉符号。

## Scenario Patterns

| 子场景 | 配方侧重 | 典型触发词 |
|--------|---------|-----------|
| 老照片/胶片 | 胶片颗粒+褪色+暗角+暖黄 | 老照片, 胶片, film, 怀旧 |
| 民国风情 | 旗袍+老上海+黄包车+Art Deco | 民国, 老上海, 花样年华 |
| 昭和复古 | 和服+木建筑+电风扇+暖光 | 昭和, 日本复古 |
| 80年代 | 霓虹色+Disco+爆炸头+卡带 | 80年代, disco, 复古 |
| 90年代/Y2K | 滑板+宽松牛仔+随身听+像素 | 90年代, Y2K, 千禧 |
| 复古色调 | 胶片预设+褪色处理+漏光 | 复古调色, 宝丽来 |

## Examples

### Example: 民国风情

**用户需求**：写一个老上海民国风情的画面

**公式拆解**：
- 主体：穿旗袍的女性 + 波浪烫发+珍珠耳环
- 场景：老上海弄堂/石库门 + 斑驳墙壁
- 光线：黄昏金色柔光
- 后期：胶片颗粒+褪色处理+暖黄调
- 氛围：怀旧优雅
- 风格：民国复古摄影

**最终提示词**：
```
一位穿墨绿色丝绒旗袍的女性站在老上海石库门弄堂里，波浪烫发上别着珍珠发夹，
斑驳的青砖墙上爬着爬山虎，黄昏金色柔光斜照，地面是湿润的青石板，
胶片颗粒感和褪色处理，暖黄色调，轻微暗角，民国复古摄影风格，花样年华般的怀旧氛围
```

### Example: 80年代Disco

**用户需求**：写一个80年代迪斯科舞厅的复古画面

**公式拆解**：
- 场景：迪斯科舞厅 + 镜面球+霓虹灯管
- 人物：穿喇叭裤+亮片上衣的舞者
- 光线：旋转镜面球的彩色光斑 + 霓虹粉紫
- 风格：80年代复古
- 后期：胶片颗粒+柔焦

**最终提示词**：
```
80年代的迪斯科舞厅，天花板上旋转的镜面球向四周投射彩色光斑，
粉色和蓝色的霓虹灯管勾勒出舞池轮廓，穿着喇叭牛仔裤和亮片上衣的年轻人在舞池中
扭动，空气中弥漫着微尘在灯光中飞舞，柔焦效果和胶片颗粒感，80年代复古纪实摄影
```

### Example: 90年代街头

**用户需求**：写一个90年代中国街头的怀旧画面

**公式拆解**：
- 场景：90年代中国街头 + 自行车+老式公交
- 细节：公用电话亭+老式招牌+搪瓷杯
- 光线：午后阳光
- 后期：轻微褪色+暖色调

**最终提示词**：
```
90年代中国城市街头，成排的二八大杠自行车停在灰色砖墙旁，一辆老式铰接公交车驶过，
路边公用电话亭里有人在打电话，商店招牌是褪色的红色楷体字，午后阳光透过法国梧桐
在地面投下斑驳光影，轻微褪色的暖黄色调，怀旧纪实摄影，时光倒流的年代感
```

## Category-Specific Tips

1. **年代锚点要精准** — 民国（旗袍+石库门+Art Deco）、80年代（霓虹+Disco+喇叭裤）、90年代（滑板+随身听+宽松牛仔）
2. **胶片老化效果** — "胶片颗粒""褪色""漏光""暗角""柔焦"是复古感的技术关键词
3. **年代道具是点睛** — 公用电话、搪瓷杯、老式公交车、卡带Walkman、黑白电视
4. **色彩偏离现代审美** — 略微偏黄、偏青或偏品红，不追求"正确"的白平衡
5. **昭和与民国的区别** — 昭和（日式木建筑+和服+电风扇）、民国（旗袍+石库门+Art Deco）
