# Festival & Holiday — 节日/节庆

## Instructions

节日类提示词的核心是**文化符号和庆典氛围**。每个节日有其标志性的视觉元素——春节的红灯笼和鞭炮、中秋的圆月和月饼、圣诞的树和雪、万圣的南瓜灯。关键是组合这些符号并营造对应的氛围：喜庆、温馨、神秘。

## Scenario Patterns

| 子场景 | 配方侧重 | 典型触发词 |
|--------|---------|-----------|
| 春节/新年 | 红灯笼+对联+烟花+年夜饭+团圆 | 春节, 过年, 除夕, 新年 |
| 中秋 | 圆月+月饼+桂花+团圆+赏月 | 中秋, 月亮, 月饼 |
| 元宵/灯会 | 花灯+灯谜+汤圆+舞龙 | 元宵, 灯会, 花灯 |
| 圣诞 | 圣诞树+雪+礼物+暖灯+红绿 | 圣诞, 圣诞节, Christmas |
| 万圣节 | 南瓜灯+糖果+装扮+暗调 | 万圣, 南瓜, Halloween |
| 端午 | 龙舟+粽子+艾草+江河 | 端午, 龙舟, 粽子 |
| 国庆 | 国旗+烟花+阅兵+庆典 | 国庆, 庆典 |

## Examples

### Example: 春节团圆

**用户需求**：写一个中国传统春节的家庭团圆场景

**公式拆解**：
- 场景：中式客厅 + 红灯笼+对联+窗花
- 人物：一家人围坐圆桌
- 细节：热气腾腾的年夜饭+红包
- 光线：暖红灯笼光+烛光
- 氛围：温馨团圆
- 风格：纪实摄影

**最终提示词**：
```
中国传统春节的家庭团圆场景，一家人围坐在铺着红桌布的圆桌前吃年夜饭，
热气腾腾的饺子和红烧鱼摆在桌中央，背景的窗户上贴着红色剪纸窗花，墙上挂着
红灯笼和春联，暖红的灯光映在每个人的笑脸上，长辈正在给孩子发红包，
温馨喜庆的春节纪实摄影风格
```

### Example: 中秋节

**用户需求**：写一个中秋赏月的唯美画面

**公式拆解**：
- 主体：圆月 + 最大最亮
- 前景：中式凉亭 + 桂花树
- 道具：月饼+茶具
- 光线：月光洒落
- 氛围：团圆思念
- 色彩：深蓝夜空+暖黄月光

**最终提示词**：
```
中秋之夜一轮巨大的金色满月悬挂在深蓝的夜空中，前景的中式凉亭飞檐下摆放着茶桌，
桌上有切开露出蛋黄的月饼和冒着热气的桂花茶，一树桂花在月光下开放，月光洒在
石板地面上如水银泻地，中秋赏月摄影，唯美圆满的氛围
```

### Example: 圣诞节

**用户需求**：写一个温馨的北欧圣诞夜

**公式拆解**：
- 场景：北欧风格客厅 + 壁炉+圣诞树
- 细节：礼物+蜡烛+窗外飘雪
- 光线：壁炉火光+圣诞树灯光
- 氛围：温馨hygge
- 色彩：红绿+暖金

**最终提示词**：
```
北欧风格客厅的圣诞夜，装饰着金色灯串和红色装饰球的圣诞树在角落发光，
壁炉中燃烧的柴火发出噼啪声和温暖光芒，三只装着礼物的红袜子挂在壁炉上，
窗外飘着雪花落在窗台上，沙发上有厚实的针织毛毯，温馨的hygge圣诞氛围，室内摄影风格
```

### Example: 万圣节

**用户需求**：写一个万圣节夜晚的奇幻场景

**公式拆解**：
- 场景：万圣夜街道 + 南瓜灯+蜘蛛网+枯树
- 人物：装扮成巫师的小孩
- 光线：南瓜灯暖黄光 + 暗紫夜空
- 氛围：诡异但有趣

**最终提示词**：
```
万圣节的夜晚，街道两旁摆放着雕刻成鬼脸的南瓜灯发出暖黄色的光，枯树上的树枝
如手指般伸向暗紫色的夜空，装扮成小巫师的孩子提着南瓜糖果篮挨家按门铃，
蜘蛛网挂在门廊栏杆上，远处有烟花在夜空中绽放，诡异而有趣的万圣节氛围
```

## Category-Specific Tips

1. **每个节日有固定色板** — 春节（红+金）、中秋（深蓝+暖金）、圣诞（红+绿+白）、万圣（橙+黑+紫）
2. **节日符号是触发关键** — 灯会=花灯+灯谜，端午=龙舟+粽子，元宵=汤圆+猜灯谜
3. **食物是节日文化的核心** — 年夜饭、月饼、粽子、汤圆、圣诞布丁、万圣糖果
4. **人物互动制造情感** — 团圆的、发红包的、赏月的、互送礼物的
5. **夜景节日要让光源丰富** — 灯笼+烛光+烟花+灯串，多光源的夜景节庆更璀璨
