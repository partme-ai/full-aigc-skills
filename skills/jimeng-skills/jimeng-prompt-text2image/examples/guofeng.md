# Guofeng — 国风/传统文化

## Instructions

国风类提示词的核心是**传统美学体系的选择**。不同朝代、不同艺术形式有截然不同的视觉语言。关键是先确定"哪一种中国风"——水墨写意、工笔重彩、敦煌壁画、宋代青绿、还是现代国潮，然后再选对应的元素和色彩体系。

## Scenario Patterns

| 子场景 | 配方侧重 | 典型触发词 |
|--------|---------|-----------|
| 汉服人像 | 服装朝代+发型+古典环境+逆光 | 汉服, 唐制, 明制, 宋制, 古装 |
| 水墨山水 | 水墨+留白+山峦+云雾+意境 | 水墨, 山水, 留白, 写意 |
| 工笔花鸟 | 工笔+精细+花鸟+绢本质感 | 工笔, 花鸟, 精细, 绢本 |
| 敦煌壁画 | 飞天+飘带+土红石绿+斑驳 | 敦煌, 飞天, 壁画, 西域 |
| 青绿山水 | 石青石绿+层峦+金色+长卷 | 青绿, 千里江山, 宋代 |
| 武侠/仙侠 | 侠客+竹林/山巅+剑+衣袂 | 武侠, 仙侠, 侠客, 江湖 |
| 禅意/茶道 | 茶室+极简+枯山水+侘寂 | 禅, 茶道, 和敬清寂, 枯山水 |
| 戏曲/非遗 | 脸谱+戏服+舞台+动态 | 京剧, 昆曲, 皮影, 剪纸 |

## Examples

### Example: 水墨山水

**用户需求**：写一个宋代风格的水墨山水画提示词

**公式拆解**：
- 风格：宋代水墨山水（核心！前置到构思首位）
- 主体/场景：层峦叠嶂 + 云雾 + 瀑布流泉 + 松树
- 点缀：扁舟 + 亭台楼阁
- 色彩：水墨黑白灰为主 + 淡赭石点缀
- 质感：宣纸质感 + 墨韵渲染
- 构图：长卷式留白

**最终提示词**：
```
宋代水墨山水画风格，层峦叠嶂的远山，山间云雾缭绕，近处有瀑布流泉和古松，
一叶扁舟在江上，亭台楼阁隐现山间，水墨黑白灰为主，淡赭石点缀，宣纸质感，
大面积留白，墨韵自然渲染
```

**为何这样写**：水墨画的核心在"留白"和"墨韵"，必须明确写出；扁舟和亭台是山水画经典的"画眼"，提供视觉焦点

### Example: 敦煌飞天

**用户需求**：想要敦煌壁画风格的飞天仙女

**公式拆解**：
- 主体：飞天仙女 + 飘逸彩带+天衣 + 琵琶/莲花
- 风格：敦煌壁画风格
- 色彩：土红+石绿+金箔色 + 暖色调
- 质感：斑驳壁画质感
- 动作：在流云中飞舞
- 表情：面容丰润慈悲

**最终提示词**：
```
敦煌壁画风格的飞天仙女，飘逸的彩带和天衣，在流云和天花间飞舞，手捧琵琶，
面容丰润慈悲，土红、石绿、金箔色为主的暖色调，斑驳的壁画质感
```

**为何这样写**：敦煌的核心是色彩体系（土红+石绿+金箔）和"斑驳质感"；彩带飘动是飞天区别于一般仙女的特征

### Example: 武侠江湖

**用户需求**：写一个白衣侠客在竹海之巅的画面

**公式拆解**：
- 主体：白衣侠客 + 长发束起 + 长剑
- 场景：竹海之巅
- 动作：风吹衣袍和竹林
- 风格：水墨与写实结合
- 色彩：淡雅青绿调
- 构图：留白式
- 氛围：雾气缭绕

**最终提示词**：
```
白衣侠客站在竹海之巅，长发束起，手持长剑，风吹动衣袍和竹林，水墨与写实结合的风格，
留白构图，雾气缭绕，淡雅青绿调
```

**为何这样写**：古风侠客的魂在"风"——衣袍和竹林的动态制造气势；水墨与写实结合比纯水墨更适合即梦

### Example: 禅意茶室

**用户需求**：写一个日式茶室的静谧画面

**公式拆解**：
- 场景：日式茶室 + 庭院绿意
- 主体/道具：抹茶碗 + 茶筅 + 和果子
- 光线：侧窗自然柔光
- 氛围：和敬清寂
- 风格：极简美学摄影
- 色彩：木色+绿+抹茶绿

**最终提示词**：
```
日式茶室的静谧一隅，窗外的庭院绿意盎然，抹茶碗中的泡沫细腻碧绿，手前有和果子，
柔和的自然光从侧窗洒入，和敬清寂的氛围，极简美学摄影
```

## Category-Specific Tips

1. **朝代决定了视觉体系** — 唐（丰腴华丽）、宋（清雅简约）、明（端庄大气），混搭会不协调
2. **国风色彩有固定搭配** — 青绿山水的石青+石绿，敦煌的土红+石绿+金，水墨的黑白+淡赭石
3. **"留白"是国风的核心** — 构图词中务必加入"留白"，避免画面过满
4. **质感词区分媒介** — "宣纸质感""绢本质感""壁画质感""漆器光泽"，不同载体完全不同
5. **现代国潮要加"现代"锚点** — 纯传统可能太复古，加"现代国潮设计""时尚与传统的碰撞"可拉回当代审美
