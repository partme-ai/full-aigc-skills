# Anime — 二次元/动漫

## Instructions

二次元类提示词的关键是**指定动画风格体系**和**保持视觉语言一致**。不同动画风格（赛璐珞、新海诚、吉卜力、90年代复古）有截然不同的上色方式、光影处理和线条风格。先选风格体系，再填充角色和场景。

## Scenario Patterns

| 子场景 | 配方侧重 | 典型触发词 |
|--------|---------|-----------|
| 日式动漫 | 角色+服装+赛璐珞上色+风格指定 | 动漫, 日漫, 二次元, anime |
| 新海诚风 | 光影通透+天空+城市/自然+青春 | 新海诚, 通透, 天空, 唯美 |
| 吉卜力风 | 田园+治愈+手绘质感+温暖色调 | 吉卜力, 宫崎骏, 治愈, 田园 |
| 赛博朋克动漫 | 未来都市+霓虹+机械+赛璐珞 | 赛博朋克, 科幻动漫 |
| 萌系/日常 | 可爱角色+校园/日常+明亮色调 | 萌, 可爱, 日常, 校园 |
| 战斗/热血 | 动态姿势+特效+机甲/武器+张力 | 战斗, 机甲, 热血, 动作 |

## Examples

### Example: 奇幻魔法少女

**用户需求**：写一个日式动漫风格的银发魔法少女

**公式拆解**：
- 主体：银发少女 + 紫色眼瞳 + 白色魔法袍
- 场景：漂浮的魔法阵中心 + 星空背景
- 道具/特效：环绕发光符文 + 背后光翼展开
- 风格：日式动漫 + 新海诚风格光影 + 赛璐珞上色
- 光线：魔法阵发光照亮的通透感
- 画质：动画电影画质

**最终提示词**：
```
日式动漫风格银发少女，紫色眼瞳，穿着白色魔法袍，站在漂浮的魔法阵中心，
周围环绕发光符文，背后有光翼展开，星空背景，新海诚风格的光影通透感，
赛璐珞上色，动画电影画质
```

**为何这样写**：风格混合（赛璐珞+新海诚光影）是现在流行的动漫风格；发光符文和光翼增加画面的华丽度

### Example: 吉卜力田园

**用户需求**：想要宫崎骏风格的田园风景，宁静治愈

**公式拆解**：
- 风格：吉卜力风格（核心定位）
- 场景：绿色山坡 + 大树 + 小花 + 农田 + 白色小屋
- 天气：蓝天白云
- 光线：柔和温暖光影
- 氛围：宁静治愈
- 画质：手绘质感

**最终提示词**：
```
吉卜力风格的田园风景画，绿色的山坡上有一棵巨大的树，白色的小花点缀草丛，
蓝天白云之下农田整齐有序，远处有白色小屋，光影柔和温暖，宁静治愈的画风
```

**为何这样写**：吉卜力的核心要素是"田园+温暖光影+手绘质感"；元素选择（大树、小花、小屋、农田）都是吉卜力标志性符号

### Example: 机甲战斗

**用户需求**：写一个巨型机甲在城市中战斗的场景，要有大片感

**公式拆解**：
- 主体：巨型机甲 + 橙红涂装 + 金属战损细节
- 场景：城市 + 楼宇
- 动作/特效：粒子炮光束 + 爆炸碎片+烟尘
- 风格：动画电影风格
- 构图：动态战斗构图
- 画质：动画电影画质

**最终提示词**：
```
巨型机甲在城市中战斗，明亮的橙红色涂装，金属装甲上的划痕和战损细节，
粒子炮发射的光束穿过楼宇，爆炸产生的碎片和烟尘，动态感极强的战斗场景，
动画电影画质
```

### Example: 萌系猫娘

**用户需求**：写一个可爱的猫娘在日常场景中

**公式拆解**：
- 主体：猫娘少女 + 橙色猫耳+长尾 + 学生制服
- 场景：夕阳下的学校天台
- 动作：手中拿铜锣烧 + 活泼开心表情
- 风格：萌系动漫
- 光线：明亮暖色调夕阳
- 质感：透明感上色 + 毛发蓬松

**最终提示词**：
```
萌系风格的猫娘少女，橙色的猫耳和长尾，穿着学生制服，手中拿着铜锣烧，
在夕阳下的学校天台上，活泼开心的表情，明亮的暖色调，透明感上色，毛发蓬松的质感
```

## Category-Specific Tips

1. **风格词要明确** — "赛璐珞上色"（cel shading）、"透明感上色"、"厚涂"（厚塗り）是上色技法的关键区别
2. **热门风格可混合但要克制** — 最多混合2种风格（如"新海诚光影+赛璐珞上色"），超过2种即梦会混乱
3. **特效词大胆用** — 二次元擅长表现发光、粒子、能量场等特效，这恰好是即梦的强项
4. **比例和画幅** — 动漫角色多用竖构图（3:4或9:16），场景多用横构图（16:9）
5. **眼睛描述是二次元灵魂** — "紫色眼瞳""星空般的眼睛""猫眼竖瞳"等细节即梦能较好呈现
