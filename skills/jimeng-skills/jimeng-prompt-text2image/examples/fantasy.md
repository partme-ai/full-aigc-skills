# Fantasy — 奇幻/魔幻

## Instructions

奇幻类提示词的核心是**建立可信的架空世界**。龙、精灵、魔法、城堡——这些元素需要写实质感来让不可能的变成可信的。关键是细节具体化：龙的鳞片是什么颜色和质感？城堡是哪个时代的风格？魔法是什么颜色和形态？

## Scenario Patterns

| 子场景 | 配方侧重 | 典型触发词 |
|--------|---------|-----------|
| 龙与巨兽 | 物种+鳞片/毛皮+姿态+环境+规模 | 龙, 巨兽, 火龙, 冰龙 |
| 魔法/法师 | 魔法师+法杖+魔法特效+袍子 | 魔法师, 巫师, 魔法, 咒语 |
| 精灵/种族 | 精灵/矮人/兽人+特征+服饰+森林 | 精灵, 矮人, 半兽人, 奇幻种族 |
| 奇幻城堡 | 城堡+风格+环境+云雾+规模 | 城堡, 王城, 要塞, 空中城 |
| 神话生物 | 独角兽/凤凰/狮鹫+特征+神圣感 | 独角兽, 凤凰, 狮鹫, 神话 |
| 剑与魔法战斗 | 战士+武器+魔法对轰+动态 | 战斗, 屠龙, 魔法对决 |

## Examples

### Example: 火焰巨龙

**用户需求**：写一头盘踞在金币堆上的红龙

**公式拆解**：
- 主体：红龙 + 深红/暗橙鳞片+翼展 + 竖瞳
- 场景：山洞深处的金币宝库
- 光线：金币反射的暖光 + 龙喉中酝酿的火焰光
- 氛围：威严压迫
- 风格：奇幻概念艺术
- 画质：极其精细

**最终提示词**：
```
一头巨大的红龙盘踞在山洞深处的金币堆上，深红色的鳞片在金币反射的暖光下闪耀，
收拢的翼膜透出暗橙色的光，竖瞳凝视着前方，喉咙深处酝酿着橘色火焰发出微光，
金币和珠宝堆积如山散落四周，奇幻概念艺术风格，极其精细，威严而压迫的氛围
```

### Example: 精灵森林

**用户需求**：写一个高等精灵居住的魔法森林

**公式拆解**：
- 场景：银白色魔法森林 + 发光植物+水晶树
- 建筑：精灵塔楼 + 优雅曲线
- 光线：月光+生物荧光
- 色彩：银白+蓝+紫
- 氛围：神圣静谧
- 风格：奇幻场景概念艺术

**最终提示词**：
```
高等精灵居住的魔法森林，银白色树干的水晶树在月光下发出柔和的蓝白色荧光，
林间漂浮着发光的孢子，优雅的精灵塔楼以流畅的曲线融入巨树之中，
发光的紫色藤蔓从建筑上垂落，一条银色的小溪在林间蜿蜒，奇幻场景概念艺术风格，
神圣而静谧的氛围
```

### Example: 独角兽

**用户需求**：写一只在林间溪边的神圣独角兽

**公式拆解**：
- 主体：白色独角兽 + 螺旋金角+银色鬃毛+发光
- 场景：林间溪流 + 晨雾+野花
- 光线：晨光丁达尔效应 + 自身圣光
- 氛围：神圣纯洁
- 风格：奇幻插画

**最终提示词**：
```
一只纯白色的独角兽站在林间清澈的溪流边，螺旋状的金色独角发出柔和的光芒，
银色的鬃毛和尾巴在微风中飘动，晨光穿透树林形成丁达尔光柱，周围开满蓝色和
白色的野花，溪水在蹄边流过泛起涟漪，奇幻插画风格，神圣纯洁的仙境氛围
```

## Category-Specific Tips

1. **奇幻需要写实质感** — 越是不可信的生物，越需要具体的质感描述（"粗粝的鳞片""光滑的独角""羽毛上的虹彩"）
2. **魔法要有颜色和形态** — 不写"魔法"，写"蓝色的奥术能量在手掌间缠绕""金色的符文光环旋转"
3. **规模感靠对比** — 巨龙旁放一个"渺小的骑士剪影"，巨大的城堡下"蚂蚁般的马车"
4. **种族特征要精准** — 精灵（尖耳+优雅）、矮人（矮壮+大胡子+战斧）、兽人（绿皮+獠牙+肌肉）
5. **光源决定奇幻氛围** — 魔法本身是光源（"法杖顶端的蓝色光球照亮周围"），处理好魔法光和环境光的互动
