# Children — 儿童/亲子

## Instructions

儿童类提示词的核心是**纯真感和自然表情**。避免摆拍式的僵硬——抓拍的笑、好奇的眼神、专注的玩耍瞬间比pose更有感染力。光线多用柔和的自然光，色彩偏明亮温暖。

## Scenario Patterns

| 子场景 | 配方侧重 | 典型触发词 |
|--------|---------|-----------|
| 新生儿 | 包裹+柔软毯子+柔光+浅景深 | 新生儿, 满月, 婴儿 |
| 儿童写真 | 服装+道具+户外/影棚+自然表情 | 儿童, 写真, 百天, 周岁 |
| 亲子互动 | 父母与孩子+拥抱+自然光+温馨 | 亲子, 母子, 父子, 家庭 |
| 孕妇照 | 孕肚+柔光+温馨+亲密 | 孕妇, 孕照, 孕期 |
| 户外玩耍 | 草地+阳光+动态+笑脸 | 玩耍, 户外, 公园, 游乐场 |
| 校园/学习 | 教室+书包+读书+伙伴 | 上学, 读书, 幼儿园 |

## Examples

### Example: 新生儿

**用户需求**：写一个新出生婴儿的温馨照片

**公式拆解**：
- 主体：新生儿 + 粉色柔软包巾 + 紧闭双眼+小手蜷缩
- 场景：柔软毛毯上
- 光线：窗边柔和自然光
- 构图：俯拍 + 浅景深
- 风格：新生儿摄影
- 氛围：温暖柔软

**最终提示词**：
```
新生儿被粉色柔软针织包巾包裹，紧闭双眼，小手蜷缩在脸颊旁，躺在米色羊绒毯上，
窗边柔和的自然光从侧面洒入，照亮婴儿的绒毛和细嫩皮肤，俯拍构图，浅景深，
新生儿摄影风格，温暖柔软的氛围，极致细节
```

### Example: 亲子互动

**用户需求**：写一个妈妈和孩子在草地上的温馨画面

**公式拆解**：
- 主体：年轻妈妈 + 2岁小女孩
- 动作：妈妈将孩子举高高 + 两人大笑
- 场景：春日草地 + 远处野花
- 光线：午后暖阳 + 逆光
- 风格：家庭纪实摄影
- 氛围：温馨快乐

**最终提示词**：
```
年轻妈妈将2岁小女孩举高高，两人仰头开心大笑，春日翠绿草地上远处有黄色野花，
午后暖阳逆光在两人发丝上形成金色光晕，穿着浅色系的棉麻亲子装，
家庭纪实摄影风格，温馨快乐，充满爱
```

### Example: 孕妇照

**用户需求**：写一个唯美孕妇照

**公式拆解**：
- 主体：孕妇 + 孕肚 + 白色蕾丝长裙
- 姿态：双手轻抚孕肚 + 低头温柔注视
- 场景：窗边+白纱窗帘飘动
- 光线：金色夕阳侧光
- 风格：唯美孕期摄影

**最终提示词**：
```
孕期女性站在大落地窗前，白色蕾丝长裙衬托隆起的孕肚，双手轻抚腹部低头温柔注视，
白纱窗帘在微风中轻轻飘动，金色夕阳侧光照亮侧脸和孕肚轮廓，唯美孕期摄影风格，
柔和的暖色调，温馨圣洁
```

## Category-Specific Tips

1. **儿童表情靠场景激发** — 不写"开心的孩子"，写"追泡泡的孩子"、"吃到冰淇淋笑了"
2. **安全第一** — 描述中的动作要安全自然，避免危险姿势
3. **服装柔软舒适** — 棉麻、针织、纯棉，避开过于成人化的服装描述
4. **光线一定要柔** — 儿童皮肤需要柔和的散射光，避免硬光造成不自然阴影
5. **眼神是灵魂** — "好奇的大眼睛""专注地盯着蝴蝶""亮晶晶的眼眸"比任何pose都重要
