# Game — 游戏/电竞

## Instructions

游戏类提示词覆盖**游戏原画、角色设计、游戏截图级画面、像素艺术和电竞场景**。关键是明确游戏类型（RPG、FPS、像素等）和视觉风格（写实、风格化、low-poly）。对于角色设计，需要描述职业、装备、种族等游戏特有属性。

## Scenario Patterns

| 子场景 | 配方侧重 | 典型触发词 |
|--------|---------|-----------|
| 游戏角色 | 职业+种族+装备+pose+原画风 | 角色, 英雄, NPC, 职业, 种族 |
| 游戏场景 | 关卡+地貌+建筑+氛围+引擎 | 关卡, 地图, 场景, 世界 |
| 像素艺术 | 像素+8bit/16bit+等距+颜色限制 | 像素, pixel, 8bit, 复古游戏 |
| 电竞现场 | 场馆+观众+大屏幕+灯光 | 电竞, 比赛, 总决赛, arena |
| 游戏截图 | 引擎名+画质+HUD+视角 | 游戏截图, in-game, 实机 |
| 装备/道具 | 武器+魔法效果+材质+特写 | 武器, 装备, 道具, 神器 |

## Examples

### Example: RPG角色

**用户需求**：写一个暗黑奇幻风格的游侠角色设计

**公式拆解**：
- 主体：女性暗夜精灵游侠 + 暗紫色皮肤+银色长发+精灵长耳
- 装备：黑色皮甲+披风 + 复合弓+淬毒箭矢
- 姿态：半蹲瞄准 + 拉满弓
- 场景：月光下的密林
- 风格：暗黑奇幻游戏原画
- 画质：3A游戏概念艺术

**最终提示词**：
```
暗黑奇幻风格的女性暗夜精灵游侠，暗紫色皮肤，银色长发束起，精灵长耳，
穿着黑色皮甲和深绿斗篷，手持复合弓拉满瞄准，箭矢尖淬毒发出绿色微光，
半蹲在月光透入的密林中，暗黑奇幻游戏原画风格，3A概念艺术品质，极其精细
```

### Example: 像素艺术

**用户需求**：写一个等距视角像素风格的小镇

**公式拆解**：
- 风格：像素艺术 + 等距视角(isometric) + 16-bit
- 主体：像素小镇 + 小房子+树+小河
- 色彩：明亮饱和调色板
- 细节：像素级纹理
- 氛围：复古游戏感

**最终提示词**：
```
等距视角像素艺术，16-bit复古游戏风格的小镇场景，红色屋顶的小房子错落有致，
绿色的像素树木点缀其间，蓝色的小河穿城而过上面有石桥，街道上有像素小人走动，
明亮的饱和调色板，像素级精细纹理，复古RPG游戏场景，怀旧游戏美学
```

### Example: 电竞现场

**用户需求**：写一个电竞总决赛的现场盛况

**公式拆解**：
- 场景：电竞场馆 + 圆形舞台+巨大LED屏幕
- 灯光：聚光灯+灯带+观众灯海
- 观众：满座+应援灯牌+欢呼
- 氛围：激动人心
- 构图：广角全景
- 风格：赛事纪实摄影

**最终提示词**：
```
电竞总决赛的巨型场馆全景，中央圆形舞台上两支战队分坐两侧，头顶悬浮四面巨大LED屏幕
显示游戏实况，蓝红两色聚光灯交汇照亮舞台中央的奖杯，满座数万观众挥舞着荧光棒和灯牌，
绚丽的灯带环绕整个场馆，激动人心的电竞盛况，广角全景纪实摄影
```

## Category-Specific Tips

1. **游戏引擎可作风格锚点** — "虚幻引擎5""Unity实时渲染""Octane渲染"即梦能理解其画面风格
2. **像素艺术要写位数和视角** — "16-bit像素""等距视角""横向卷轴"，这些定义了像素游戏的外观
3. **角色设计要写职业+种族** — 游戏角色核心三要素：种族（精灵/人类/兽人）+职业（战士/法师/游侠）+装备
4. **3A风格必须加高画质词** — "3A概念艺术""极其精细""次世代画质"，否则即梦可能产出过于简单的画面
5. **游戏UI/HUD可描述** — 如果要游戏截图感，写"画面角落有迷你地图和血条UI"增加真实感
