# Pet — 宠物

## Instructions

宠物类提示词的要点是**品种特征精准**和**萌态捕捉**。准确写出品种名（英短蓝猫、柴犬、布偶猫、柯基）远比"可爱的猫"效果好。即梦能识别常见品种的视觉特征。萌态来自具体动作描述：歪头、打滚、伸懒腰、打哈欠。

## Scenario Patterns

| 子场景 | 配方侧重 | 典型触发词 |
|--------|---------|-----------|
| 猫咪日常 | 品种+眼睛+毛色+慵懒/好奇 | 猫, 猫咪, 英短, 布偶, 橘猫 |
| 狗狗日常 | 品种+耳朵+舌头+奔跑/摇尾 | 狗, 柴犬, 金毛, 柯基, 哈士奇 |
| 宠物肖像 | 品种+正面+棚拍光+眼神 | 宠物写真, 宠物肖像 |
| 宠物搞笑 | 品种+夸张表情/姿势+道具 | 搞笑, 可爱, 萌, 表情包 |
| 异宠 | 兔/仓鼠/鹦鹉/蜥蜴+特征+微距 | 兔子, 仓鼠, 鹦鹉, 爬宠 |
| 人宠互动 | 主人+宠物+互动+生活场景 | 遛狗, 吸猫, 陪伴 |

## Examples

### Example: 布偶猫

**用户需求**：写一张布偶猫的萌照，蓝眼睛要有神

**公式拆解**：
- 主体：布偶猫 + 蓝色大眼睛 + 长毛重点色 + 粉鼻
- 姿态：歪头好奇看向镜头
- 场景：窗边猫爬架 + 阳光
- 光线：自然柔光
- 构图：特写+浅景深
- 画质：毛发蓬松质感

**最终提示词**：
```
一只布偶猫蹲在窗边的猫爬架上，蓝宝石般的大眼睛好奇地歪头看向镜头，重点色长毛
在阳光下蓬松柔软，粉色的鼻子和肉垫，阳光在毛发边缘形成温暖光晕，萌宠摄影，
浅景深，毛发纤毫毕现
```

### Example: 柴犬

**用户需求**：写一只在草地上奔跑的柴犬

**公式拆解**：
- 主体：赤色柴犬 + 立耳+卷尾 + 吐舌头
- 动作：草地上奔跑 + 回望镜头
- 场景：公园草地 + 蓝天
- 光线：午后阳光
- 风格：宠物动态摄影

**最终提示词**：
```
一只赤色柴犬在翠绿草地上欢快奔跑，立耳向后飞起，卷尾高高翘起，吐着舌头回望镜头，
午后阳光照亮金红色的毛发，蓝天白云背景，动态宠物摄影，快乐自由，毛发飘逸感
```

### Example: 柯基

**用户需求**：写一只柯基的可爱写真，突出短腿和蜜桃臀

**公式拆解**：
- 主体：彭布罗克威尔士柯基 + 大耳朵+短腿+圆臀
- 姿态：趴在地上 + 前爪伸直 + 回头看
- 场景：木地板 + 地毯
- 光线：室内暖光
- 构图：低角度（与柯基同高）

**最终提示词**：
```
一只彭布罗克威尔士柯基趴在家中的木地板上，前爪向前伸直，圆滚滚的蜜桃臀朝镜头，
回头露出憨憨的笑容，大耳朵竖起，浅黄白色被毛在室内暖光下柔软有光泽，
低角度拍摄与柯基视线同高，萌宠摄影，温馨可爱
```

## Category-Specific Tips

1. **品种名必须准确** — "英短蓝猫""柴犬""布偶猫"比"猫""狗"效果好10倍
2. **毛色词紧随品种** — "赤色柴犬""蓝重点色布偶""三花英短"，品种+毛色一起写
3. **萌态来自动作细节** — 歪头、伸懒腰、打滚、踩奶、飞机耳，具体动作远比"可爱"有效
4. **低角度拍摄** — 与宠物视线平齐的角度最自然，避免俯拍
5. **异宠要强调独特特征** — 兔子的长耳、仓鼠的颊囊、鹦鹉的羽毛色彩
