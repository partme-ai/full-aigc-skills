# Commercial — 商业/电商/营销

## Instructions

商业类提示词的核心是**清晰传达产品价值和品牌调性**。不同于艺术创作，商业视觉有明确的目标——卖产品、传达信息、建立品牌认知。关键是产品突出 + 背景干净 + 光线专业 + 符合平台规范。

## Scenario Patterns

| 子场景 | 配方侧重 | 典型触发词 |
|--------|---------|-----------|
| 电商白底图 | 产品+纯白背景+均匀布光+细节 | 电商, 白底, 商品图, 主图 |
| 广告大片 | 产品+场景+氛围光+品牌调性 | 广告, 大片, campaign, 品牌 |
| 海报设计 | 主题+排版空间+视觉冲击+风格 | 海报, 宣传, 活动, promotion |
| 社交媒体 | 场景+生活方式+自然光+真实感 | 小红书, 朋友圈, 种草, 分享 |
| Banner/头图 | 主题+文字预留区+色彩+构图 | banner, 头图, 封面, 首屏 |
| 包装展示 | 产品+包装+场景+光影 | 包装, 礼盒, 开箱, unboxing |

## Examples

### Example: 电商白底图

**用户需求**：写一个运动鞋的电商白底产品图

**公式拆解**：
- 主体：运动鞋 + 材质细节（网面+气垫+橡胶底）
- 背景：纯白无缝背景
- 光线：均匀柔光 + 无明显阴影
- 构图：多角度之中的主图角度（45度侧视）
- 风格：电商产品摄影
- 画质：超清细节

**最终提示词**：
```
一双白色运动鞋，纯白无缝背景，网面材质纹理清晰可见，气垫中底透明有光泽，
鞋底橡胶纹路细节，均匀的专业影棚柔光，无阴影，45度侧视展示侧面和鞋底，
电商产品摄影，超清细节
```

**为何这样写**：白底+无影是电商平台的基本要求；45度角展示最多的产品信息；材质词（网面+气垫+橡胶）传达品质

### Example: 香水广告大片

**用户需求**：写一个高端香水广告大片的提示词

**公式拆解**：
- 主体：琥珀色香水瓶
- 场景/道具：粗糙灰色岩石（材质对比）
- 光线：斜阳从右上 + 金色高光 + 温暖投影
- 构图：瓶身棱角线条分明
- 背景：简洁抽象阴影
- 风格：高级香氛广告

**最终提示词**：
```
一瓶琥珀色香水瓶置于粗糙的灰色岩石上，斜阳从右上角打来，在玻璃瓶上形成金色高光
和温暖的投影，瓶身的棱角线条分明，背景是简洁的抽象阴影，高级香氛广告大片
```

**为何这样写**：粗糙岩石 vs 光滑玻璃 = 材质对比制造高级感；单光源+金色高光 = 奢侈品的经典布光方案

### Example: 生活方式（小红书风格）

**用户需求**：写一个适合小红书的夏日饮品分享场景

**公式拆解**：
- 主体：冰拿铁 + 玻璃杯水珠
- 场景：阳台小桌 + 绿植背景
- 光线：自然晨光
- 风格：生活美学摄影
- 氛围：清新自然
- 构图：俯拍

**最终提示词**：
```
一杯冰拿铁放在阳台的木质小桌上，玻璃杯外壁凝结着水珠，咖啡与牛奶的分层清晰可见，
旁边放着一本翻开的杂志和一副墨镜，背景是模糊的绿植和城市天际线，早晨柔和的自然光，
生活美学摄影风格，清新自然，俯拍构图
```

**为何这样写**：生活方式类需要"场景感"——不只是产品，而是产品所处的完整生活场景；水珠、杂志、墨镜都是生活感的道具

### Example: 促销海报

**用户需求**：写一个618电商促销海报的视觉画面

**公式拆解**：
- 主体/主题：购物狂欢氛围
- 色彩：红色+金色主调
- 元素：礼盒堆叠 + 彩带 + 光影
- 光线：聚光灯效果
- 构图：居中 + 上方留白（预留给文字）
- 风格：电商海报设计

**最终提示词**：
```
红色和金色主调的促销场景，堆叠的精致礼盒上系着金色丝带，聚光灯从上方打下形成舞台效果，
金色碎屑在空中飘落，背景为深红渐变，画面上方保留大面积纯色空间，电商促销海报设计风格
```

**为何这样写**：海报类必须预留文字空间；红+金是电商促销的经典配色；飘落的金色碎屑制造节日氛围

## Category-Specific Tips

1. **白底图要"无影"或"极淡影"** — 电商平台通常要求纯白背景，明确写"无阴影"或"柔和淡影"
2. **预留文字空间** — 海报和Banner类务必写"上方/左侧留白"或"大面积纯色区域"，方便后期加文案
3. **场景化 > 产品堆砌** — 生活方式类的核心是展示产品在被使用的状态，而非产品本身
4. **平台风格匹配** — 小红书（清新自然）、天猫（精致高级）、拼多多（实惠接地气），词库选择要匹配平台调性
5. **Logo/文字不要依赖AI** — 即梦无法稳定生成文字，品牌Logo和产品名应在后期添加
