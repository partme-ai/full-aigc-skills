# Wedding — 婚礼/庆典

## Instructions

婚礼类提示词的核心是**浪漫感和仪式感**。白色/粉色/金色是主色调，柔光、逆光、浅景深是标准配置。关键是捕捉情感瞬间——对视、牵手、拥抱、誓言——以及场景的精致布置（鲜花、拱门、烛光）。

## Scenario Patterns

| 子场景 | 配方侧重 | 典型触发词 |
|--------|---------|-----------|
| 婚纱肖像 | 婚纱材质+柔光+浅景深+仙女感 | 婚纱, 新娘, 婚纱照 |
| 婚礼仪式 | 仪式场景+鲜花布置+宾客+氛围光 | 婚礼, 仪式, 教堂, 户外婚礼 |
| 情侣互动 | 对视+牵手+自然笑容+逆光 | 情侣, 求婚, 蜜月, 情侣照 |
| 婚礼细节 | 戒指+捧花+请柬+蛋糕+桌面布置 | 婚戒, 捧花, 婚礼布置 |
| 中式婚礼 | 龙凤褂+红盖头+花轿+红色调 | 中式婚礼, 凤冠霞帔, 传统婚礼 |
| 求婚瞬间 | 单膝跪地+戒指盒+惊喜表情+夜景 | 求婚, 表白, 惊喜 |

## Examples

### Example: 婚纱肖像

**用户需求**：写一个新娘婚纱照的提示词，要有仙女感

**公式拆解**：
- 主体：新娘 + 蕾丝抹胸婚纱 + 拖尾长纱
- 姿态：低头微笑 + 手捧捧花
- 场景：花园草坪 + 白色玫瑰拱门
- 光线：金色夕阳逆光 + 头纱透光
- 风格：浪漫婚纱摄影
- 构图：中景 + 浅景深

**最终提示词**：
```
新娘穿着蕾丝抹胸婚纱，长拖尾薄纱在草地上展开，手捧白玫瑰与尤加利叶捧花，
低头温柔微笑，站在白色玫瑰拱门前，金色夕阳逆光穿透头纱形成柔美光晕，
浪漫婚纱摄影风格，浅景深，梦幻唯美
```

### Example: 中式婚礼

**用户需求**：写一个传统中式婚礼的场景

**公式拆解**：
- 主体：新郎新娘 + 龙凤褂+凤冠霞帔
- 场景：红色喜堂 + 双喜+灯笼+红绸
- 动作：夫妻对拜或掀盖头
- 光线：暖红烛光 + 灯笼柔光
- 色彩：红色+金色主调
- 风格：中式传统婚礼纪实

**最终提示词**：
```
传统中式婚礼喜堂，新娘穿龙凤褂戴凤冠霞帔端坐红木椅上，红色丝绸盖头被新郎用喜秤轻掀，
背景大红双喜字高挂，红灯笼和红绸装饰满堂，暖红烛光映照脸庞，红色与金色交织，
中式传统婚礼纪实摄影风格，庄重而喜庆
```

### Example: 求婚瞬间

**用户需求**：写一个夜景求婚的感人瞬间

**公式拆解**：
- 主体：情侣 + 男士单膝跪地 + 戒指盒
- 表情：女方惊喜捂嘴
- 场景：城市观景台 + 城市灯光夜景
- 光线：远处城市暖光 + 前景暖色小灯串
- 氛围：浪漫感动

**最终提示词**：
```
夜景城市观景台上求婚瞬间，男士单膝跪地手举打开的戒指盒，女方双手捂嘴惊喜落泪，
背景是城市璀璨灯火和夜空，栏杆上缠绕暖色小灯串，浪漫感动的氛围，纪实婚礼摄影风格
```

## Category-Specific Tips

1. **柔光是婚纱的灵魂** — 逆光+薄纱透光是婚纱照的经典配方，营造柔和梦幻
2. **捧花/戒指/蛋糕要写具体花材** — "白玫瑰+尤加利叶"比"好看的捧花"强
3. **中式婚礼色彩要浓** — 大红+金+龙凤刺绣+双喜，色彩饱和度要够
4. **情感大于摆拍** — 抓拍感动瞬间（落泪、对视、大笑）比标准pose更有感染力
5. **场地定调** — 海滩婚礼（开阔蔚蓝）、教堂婚礼（庄严神圣）、户外花园（自然田园）
