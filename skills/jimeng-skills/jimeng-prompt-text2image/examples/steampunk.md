# Steampunk — 蒸汽朋克

## Instructions

蒸汽朋克的核心是**维多利亚时代美学 + 蒸汽机械科技**。黄铜、齿轮、蒸汽管道、皮革、木质——这些材质是蒸汽朋克的视觉基础。色彩以铜色、黄铜金、深棕、墨绿为主。关键是把19世纪的服装和建筑与超前的机械装置融合。

## Examples

### Example: 蒸汽飞艇

**用户需求**：写一艘在云层中航行的蒸汽朋克飞艇

**公式拆解**：
- 主体：巨型蒸汽飞艇 + 铜质船体+木甲板+帆布气囊
- 细节：螺旋桨+蒸汽管道+齿轮机构
- 场景：金色云层之上 + 日落
- 光线：金色夕阳照在铜质船体上
- 风格：蒸汽朋克概念艺术

**最终提示词**：
```
巨型蒸汽朋克飞艇在金色云层之上航行，黄铜和木质拼接的船体在夕阳下泛着温暖的光泽，
巨大的帆布气囊上缝有补丁，铜质螺旋桨在船尾旋转，甲板上交错的蒸汽管道喷出白色蒸汽，
精密的齿轮机构在船侧隐约可见，金色夕阳穿透云层，蒸汽朋克概念艺术风格，壮丽冒险
```

## Category-Specific Tips

1. **黄铜+皮革+木头是蒸汽朋克三材质** — 所有描述围绕这三种材质展开
2. **色彩板限定** — 铜金、深棕、墨绿、暗红、象牙，使用这个限定的复古色板
3. **蒸汽和齿轮是必选项** — 几乎每个蒸汽朋克画面都需要"齿轮"和"蒸汽"两个元素
4. **维多利亚服装定时代** — 高礼帽、燕尾服、束腰裙、护目镜，服装锚定19世纪
5. **科技要有"手工感"** — 蒸汽朋克中的科技不是光滑的未来感，而是铆钉、焊缝、黄铜氧化痕迹
