# Landscape — 风景/场景

## Instructions

风景类提示词的核心是**空间感和氛围**。场景（地貌+时间+天气）占主导，需要营造"身临其境"的感觉。关键是建立一个清晰的视觉层次：前景（细节/引导）→ 中景（主体）→ 远景（氛围）。

## Scenario Patterns

| 子场景 | 配方侧重 | 典型触发词 |
|--------|---------|-----------|
| 自然风光 | 地貌+时间+光线+大气效果 | 山水, 日出, 云海, 草原, 沙漠 |
| 城市景观 | 建筑群+时间+光线+人流 | 天际线, 街景, 夜景, 城市 |
| 水景 | 水面+反射+光线+天气 | 湖, 海, 河流, 倒影, 瀑布 |
| 季节限定 | 季节元素+色彩+光线 | 樱花, 红叶, 雪景, 花海 |
| 星空/天象 | 夜空+银河+前景+长曝光 | 星空, 银河, 极光, 流星 |
| 航拍/鸟瞰 | 鸟瞰视角+地貌纹理+光影 | 航拍, 鸟瞰, 高空, 俯拍 |

## Examples

### Example: 秋日森林

**用户需求**：写一个秋天森林的风景提示词，要有光感

**公式拆解**：
- 场景：挪威森林 + 金色红色秋叶 + 小溪
- 时间/天气：阳光透过密林
- 光线：丁达尔光柱（体积光）
- 风格：油画质感
- 色彩：温暖色调
- 构图：纵深构图（树木排列 + 小溪引导线）
- 画质：立体感强

**最终提示词**：
```
秋日挪威森林，金色和红色的树叶铺满地面，阳光透过密林形成丁达尔光柱，
小溪在落叶间流过，空气中漂浮着雾气和微尘，油画质感，温暖色调，立体感强
```

**为何这样写**：丁达尔光柱是风景摄影的"作弊码"，瞬间提升层次；铺满地面的落叶+漂浮的微尘制造空间深度；小溪作为引导线

### Example: 城市夜景

**用户需求**：想要一个未来感赛博朋克城市夜景

**公式拆解**：
- 场景：未来城市 + 高楼 + 全息广告
- 时间：深夜
- 天气：雨后湿润
- 风格：赛博朋克
- 光线：霓虹灯蓝紫调 + 地面反射
- 构图：仰拍 + 广角
- 画质：超写实

**最终提示词**：
```
雨后的未来城市深夜，高耸的摩天大楼之间悬浮着全息投影广告，霓虹灯光在湿漉漉
的街道上形成彩色倒影，蓝紫色调为主，穿插品红色霓虹灯，赛博朋克风格，仰拍广角
构图，超写实
```

**为何这样写**：雨后湿润地面制造霓虹反射是赛博朋克的标志性元素；蓝紫+品红的限定色板增强风格一致性

### Example: 草原星空

**用户需求**：写一个星空下的草原，要有地面暖光对比

**公式拆解**：
- 场景：蒙古草原 + 蒙古包 + 萤火虫
- 时间：深夜
- 天象：银河横贯天际
- 光线：冷蓝星空 vs 暖黄蒙古包灯光
- 风格：长曝光效果
- 氛围：壮丽震撼
- 画质：细节丰富

**最终提示词**：
```
蒙古草原的夜晚，银河横贯天际，繁星如钻石般清晰，前景的蒙古包透出暖黄灯光，
地面有萤火虫飞舞，长曝光效果，壮丽震撼，冷蓝色调与暖光对比
```

**为何这样写**：冷暖对比是这张图的灵魂——冷蓝星空与暖黄灯光的色彩对立让画面更有张力；萤火虫增加前景趣味性

### Example: 江南水乡

**用户需求**：烟雨江南古镇，要有水墨意境

**公式拆解**：
- 场景：江南古镇 + 青石板路 + 小桥流水 + 白墙黛瓦
- 天气：烟雨朦胧
- 人物点缀：撑伞行人
- 风格：水墨画意境
- 色彩：淡雅青灰调
- 细节：雨滴落水涟漪

**最终提示词**：
```
烟雨朦胧的江南古镇，青石板路和小桥流水，白墙黛瓦的建筑倒映在水面，
撑伞的行人，水墨画般的意境，淡雅的青灰色调，雨滴滴落水面泛起涟漪
```

## Category-Specific Tips

1. **善用大气效果** — 雾、丁达尔光、雨、雪、尘埃可大幅增加画面深度
2. **前景不可少** — 纯远景容易显得空，加入前景元素（树枝、岩石、花草、人物剪影）增加层次
3. **天气即情绪** — 同样的场景，晴/雨/雾/雪给出完全不同的情绪，这是最重要的变量之一
4. **色彩主调决定基调** — 暖色调（温馨/怀旧）、冷色调（宁静/孤独）、去饱和（苍凉/极简）
5. **水的反射是双倍内容** — 水面倒影等于给画面增加了一个维度，夜景和黄昏尤其有效
