# Minimalism — 极简主义

## Instructions

极简类提示词的核心是**做减法**。负空间（大量留白）、单一主体、限定色板、几何构图——用最少元素传达最大视觉冲击力。关键是克制：宁可少写一个元素，也不要把画面填满。

## Scenario Patterns

| 子场景 | 配方侧重 | 典型触发词 |
|--------|---------|-----------|
| 几何抽象 | 几何形状+纯色背景+色彩对比 | 几何, 抽象, 色块, 构成 |
| 建筑极简 | 白墙+光影+几何+材质 | 极简建筑, 白墙, 线条 |
| 静物极简 | 单一物品+纯色背景+长阴影 | 极简静物, 一花一瓶 |
| 人像极简 | 单人+大负空间+纯色+剪影 | 极简人像, 剪影, 孤独 |
| 日式侘寂 | 枯枝+陶器+留白+自然光 | 侘寂, 禅意, 空 |
| 产品极简 | 单一产品+纯白/黑背景+干净光 | 极简产品, 白底 |

## Examples

### Example: 极简静物

**用户需求**：写一个极简风格的一枝花

**公式拆解**：
- 主体：单一枝白色马蹄莲（减少到只有一个元素）
- 花器：粗陶花瓶
- 背景：大面积纯色白墙
- 光线：单侧柔光+长阴影
- 构图：大面积负空间
- 风格：极简摄影

**最终提示词**：
```
一支白色马蹄莲插在粗陶花瓶中，花瓶置于纯白空间的中央，大面积的白色负空间占据画面
绝大部分，单侧柔光投下一条修长的阴影在白墙上，极简摄影风格，克制的色彩，
空灵而优雅的安静感
```

### Example: 极简建筑

**用户需求**：写一个安藤忠雄风格的光之教堂极简空间

**公式拆解**：
- 场景：清水混凝土空间 + 十字形光缝
- 光线：十字光束射入暗空间（唯一光源）
- 材质：光滑清水混凝土
- 色彩：黑白灰
- 氛围：神圣极简
- 构图：对称

**最终提示词**：
```
安藤忠雄风格的极简混凝土空间，昏暗的室内只有一面墙上开有十字形缝隙，阳光从十字缝
透入形成十字光束，照亮光滑的清水混凝土墙面和地面，黑白灰的纯粹色调，对称构图，
极简建筑摄影风格，神圣而静谧的空间诗学
```

### Example: 极简人像

**用户需求**：写一个极简的剪影人像，孤独感

**公式拆解**：
- 主体：一个人的剪影
- 场景：大面积纯色渐变天空
- 地面：水平线极低+纯黑地面
- 光线：逆光落日
- 构图：极小人物+巨量天空
- 氛围：孤独

**最终提示词**：
```
一个人的黑色剪影站在极低的地平线上，大面积渐变的淡橙色天空占据画面95%的空间，
落日如一个微小的金色圆点，水平线近乎抽象的一条黑线，极简构图，孤独而辽阔，
极简艺术摄影
```

## Category-Specific Tips

1. **负空间是极简的核心** — "大面积留白""纯色背景占据画面X%"明确写出空间比例
2. **主体只有一个** — 极简静物中，一个物品比一组物品更有力
3. **色彩限定2-3色** — 单色调或双色对比，色彩越少越极简
4. **光线制造层次** — 极简画面中，光影对比是唯一的结构手段
5. **日式侘寂=不规则+自然材质+留白** — 不完美的手工感、粗陶、枯木、素色
