# Fashion — 时尚/服装/配饰

## Instructions

时尚类提示词的核心是**服装质感和态度**。重点是服装材质（丝绸、皮革、棉麻、针织）、廓形（oversized、修身、A字）以及模特姿态传达的气场。光线多用柔光或棚拍灯光，背景简洁或场景化。

## Scenario Patterns

| 子场景 | 配方侧重 | 典型触发词 |
|--------|---------|-----------|
| 时装大片 | 服装廓形+超模姿态+棚拍光 | 时尚, 大片, editorial, vogue |
| 街头穿搭 | 街景+自然光+休闲混搭 | 街拍, 穿搭, 日常, OOTD |
| 珠宝配饰 | 微距+金属光泽+宝石火彩 | 珠宝, 钻石, 项链, 戒指 |
| 鞋包特写 | 材质纹理+静物布光+品牌感 | 包包, 鞋, 皮具, 奢侈品 |
| T台走秀 | 动态+追光+夸张设计+舞台 | 走秀, runway, 时装周 |
| 美妆特写 | 妆容+唇色+眼妆+皮肤质感 | 美妆, 口红, 眼影, 护肤 |

## Examples

### Example: 时装大片

**用户需求**：写一个高定时装大片的提示词

**公式拆解**：
- 主体：超模 + 黑色丝绒曳地长裙 + 深V领+立体剪裁
- 姿态：侧身回眸 + 手轻提裙摆
- 场景：纯黑背景
- 光线：单束顶光 + 裙摆边缘反光
- 风格：Vogue editorial摄影
- 构图：全身 + 低角度仰拍

**最终提示词**：
```
超模身着黑色丝绒曳地长裙，深V领设计搭配立体剪裁廓形，侧身回眸姿态，手轻提裙摆，
纯黑背景，单束顶光照亮面部和肩线，裙摆丝绒质感在光线下泛出微妙光泽，
Vogue editorial摄影风格，低角度仰拍，高级时装感
```

### Example: 珠宝微距

**用户需求**：写一个钻石项链的珠宝摄影

**公式拆解**：
- 主体：铂金钻石项链 + 梨形主钻 + 密镶碎钻
- 场景：深蓝丝绒底座
- 光线：多点柔光 + 钻石火彩
- 构图：微距特写
- 风格：奢侈珠宝摄影

**最终提示词**：
```
铂金钻石项链微距特写，梨形主钻周围密镶碎钻，深蓝色丝绒底座，多点柔光打亮每一颗钻石，
折射出七彩火彩，铂金链节的精密纹理，奢侈珠宝摄影风格，极浅景深，极致细节
```

### Example: 街头穿搭

**用户需求**：写一个秋日街头穿搭的街拍

**公式拆解**：
- 主体：年轻女性 + 驼色oversized羊毛大衣 + 白色高领+直筒牛仔裤+踝靴
- 场景：秋日街道 + 落叶 + 咖啡馆橱窗
- 姿态：行走中抓拍 + 手拿咖啡杯
- 光线：午后暖阳 + 逆光
- 风格：时尚街拍

**最终提示词**：
```
年轻女性走在秋日街道上，驼色oversized羊毛大衣内搭白色高领针织，直筒牛仔裤配棕色踝靴，
行走中抓拍的动态瞬间，一手拿咖啡杯，午后暖阳逆光勾勒出发丝光晕，街道两侧法国梧桐落叶，
咖啡馆橱窗暖光作为背景，时尚街拍风格，自然随性
```

## Category-Specific Tips

1. **材质词是时尚的灵魂** — "丝绒""缎面""羊绒""小羊皮""雪纺"比"好看的衣服"有效百倍
2. **廓形用英文更精准** — oversized, slim-fit, A-line, boyfriend, cropped 等词即梦能理解
3. **珠宝类必须写火彩** — 钻石"七彩火彩"、珍珠"温润珠光"、黄金"哑光拉丝"，光感决定质感
4. **品牌风格可引用** — "爱马仕风格""Celine极简""Gucci复古奢华"即梦能理解其视觉调性
5. **妆容要细化到部位** — 不写"好看的妆"，写"哑光红唇+微闪金棕眼影+自然浓眉"
