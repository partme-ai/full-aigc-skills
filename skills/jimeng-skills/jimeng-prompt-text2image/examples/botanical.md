# Botanical — 植物/花卉/园艺

## Instructions

植物类提示词的核心是**生命力和质感**。花瓣的柔嫩、叶片的脉络、茎秆的绒毛、苔藓的湿润——质感描述是植物摄影的灵魂。光线多用柔和的自然光或逆光来表现通透感。露珠、晨雾、雨后是加分项。

## Scenario Patterns

| 子场景 | 配方侧重 | 典型触发词 |
|--------|---------|-----------|
| 花卉特写 | 花瓣层次+露珠+逆光+浅景深 | 花卉, 花朵, 鲜花, 花瓣 |
| 绿植/多肉 | 叶片质感+形态+盆器+光影 | 绿植, 多肉, 盆栽, 室内植物 |
| 盆景/盆栽 | 树形+盆器+留白+东方美学 | 盆景, 盆栽, 禅意 |
| 花园/庭院 | 整体+花境+路径+季节 | 花园, 庭院, 花境, 园林 |
| 森林/树林 | 树木+阳光穿透+苔藓+步道 | 森林, 树林, 原始林, 竹林 |
| 微观植物 | 苔藓/菌类/蕨类+微距+湿润 | 苔藓, 蘑菇, 菌类, 微观 |

## Examples

### Example: 玫瑰特写

**用户需求**：写一朵晨露中的玫瑰花特写

**公式拆解**：
- 主体：深红色玫瑰 + 层叠花瓣 + 天鹅绒质感
- 细节：露珠在花瓣边缘 + 水珠折射
- 光线：清晨散射柔光 + 逆光透出花瓣纹理
- 构图：极浅景深微距 + 花心特写
- 风格：花卉摄影

**最终提示词**：
```
深红色玫瑰花的微距特写，多层花瓣从花心向外螺旋展开，天鹅绒般的花瓣质感上挂着
晶莹的晨露，清晨柔和的逆光穿透花瓣显露出细腻的纹理，极浅景深聚焦花心，
水珠折射微光，花卉微距摄影风格
```

### Example: 多肉盆栽

**用户需求**：写一组多肉植物的室内盆栽

**公式拆解**：
- 主体：多肉组合 + 莲花掌+虹之玉+熊童子
- 盆器：水泥质感手工花盆
- 场景：窗台 + 散射光
- 风格：植物生活美学
- 构图：平视 + 多盆错落排列

**最终提示词**：
```
一组多肉植物盆栽排列在窗台上，莲花掌的蓝绿色莲座、虹之玉的红色叶片和熊童子的
毛茸茸肥厚叶片形成质感对比，手工水泥花盆的粗糙质感，窗边柔和的散射光，
植物生活美学摄影风格，干净温暖，治愈感
```

### Example: 竹林

**用户需求**：写一个幽静竹林的画面

**公式拆解**：
- 主体：翠竹 + 竹节+竹叶
- 场景：竹林深处 + 石板小径
- 光线：阳光从竹叶间隙穿透 + 光斑洒落
- 氛围：幽静禅意
- 构图：仰拍 + 引导线（竹竿向上）
- 风格：自然摄影

**最终提示词**：
```
幽静的竹林深处，修长的翠竹高耸入云，竹节分明，阳光从竹叶间隙穿透洒下斑驳光斑，
一条石板小径蜿蜒通向深处，地面覆盖枯黄竹叶，青翠与淡金的色调，仰拍构图突出
竹竿的垂直节奏感，自然摄影风格，静谧禅意
```

## Category-Specific Tips

1. **花卉要写花名** — "大卫奥斯汀玫瑰""无尽夏绣球""染井吉野樱"比"好看的花"精准
2. **露珠是花卉的魔法词** — 加上"晨露挂花瓣""雨后的水珠"瞬间提升通透感和生命力
3. **逆光穿透花瓣最美** — 花瓣在逆光下呈现半透明纹理，这是花卉摄影的秘诀
4. **绿植用叶形叶色区分** — "龟背竹的裂叶""琴叶榕的大叶""蕨类的羽状复叶"
5. **森林要有"层次"** — 前景蕨类/苔藓 → 中景树干 → 远景雾气/光柱，三层结构
