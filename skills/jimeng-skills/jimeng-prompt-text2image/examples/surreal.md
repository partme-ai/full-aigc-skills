# Surrealism — 超现实主义

## Instructions

超现实类提示词的核心是**逻辑错位但质感写实**——把现实中不可能共存的元素放在同一个画面中，但每一个元素本身都以写实的方式呈现。达利的融化的钟表、马格利特的漂浮礼帽——超现实是梦境逻辑在视觉中的体现。

## Scenario Patterns

| 子场景 | 配方侧重 | 典型触发词 |
|--------|---------|-----------|
| 梦境/潜意识 | 不合逻辑的组合+写实质感+飘浮 | 梦境, 潜意识, dream, 漂浮 |
| 自然法则颠倒 | 重力反转+比例错位+材料置换 | 颠倒, 倒置, 失重 |
| 身体变形 | 人体+自然元素+机械+变形 | 变形, 异化, 人面花 |
| 空间悖论 | 不可解空间+无限循环+拓扑异常 | 悖论, 无尽, 不可能空间 |
| 时间意象 | 融化的钟+沙漏+时空扭曲 | 时间, 永恒, 瞬间 |
| 哲学隐喻 | 抽象概念+象征物+荒诞 | 隐喻, 象征, 存在 |

## Examples

### Example: 梦境漂浮

**用户需求**：写一个鲸鱼在云海中游动的超现实画面

**公式拆解**：
- 概念：海洋与天空互换——鲸鱼在云海中
- 主体：座头鲸 + 真实皮肤纹理
- 场景：金色云海 + 夕阳
- 细节：云海如海水般有波浪和涟漪
- 风格：超现实主义+写实质感
- 氛围：梦幻震撼

**最终提示词**：
```
超现实主义画面，一头巨大的座头鲸在金色的云海中缓缓游动，鲸鱼的皮肤纹理和藤壶
清晰真实，巨大的胸鳍拨开云雾泛起涟漪般的气浪，从云层中跃出时带起云雾如浪花飞溅，
夕阳从云层下方透出金色光，超现实主义摄影风格，写实质感，梦幻而震撼
```

### Example: 融化的时间

**用户需求**：写一个达利风格的钟表融化画面

**公式拆解**：
- 主体：怀表/挂钟 + 如软糖般融化垂坠
- 场景：荒凉的超现实沙漠 + 枯树+远处海面
- 光线：戏剧性低角度光
- 风格：达利超现实主义
- 氛围：荒诞神秘

**最终提示词**：
```
超现实主义荒原上，一只金色的怀表如软糖般融化垂挂在枯树枝上，表盘变形拉长但罗马数字
依然清晰，另一只挂钟像奶酪一样从方形台座上滑落，远处是平静如镜的海面和淡紫色的天空，
达利超现实主义风格，荒诞而神秘，极其精细的油画质感
```

### Example: 人面花

**用户需求**：写一个花瓣变成人脸的超现实植物

**公式拆解**：
- 概念：人脸与花瓣融合
- 主体：玫瑰花 + 花瓣上有闭眼的女性面庞
- 细节：花瓣的纹理和人皮肤的过渡
- 光线：柔和侧光
- 风格：超现实摄影
- 氛围：神秘美丽

**最终提示词**：
```
超现实主义画面，一朵巨大的白玫瑰的花瓣上浮现出一张闭眼的女性面庞，花瓣的纹理
在脸颊处逐渐过渡为人类皮肤，花蕊中的露珠如同泪水，茎秆上的尖刺是金属质感的银色，
纯黑背景，柔和的侧光照亮花瓣的透明纹理和面庞的轮廓，超现实摄影风格
```

## Category-Specific Tips

1. **写实质感是超现实的基石** — 越是不可能的事，越需要具体的材料和质感让它"可信"
2. **错位要明确** — "云海如海水""花瓣如皮肤"，明确写出A像B一样行为/质感
3. **空间/物理法则反常** — 重力反转、尺度错乱（巨大小物件/微小大物体）、材质置换
4. **梦境逻辑** — 场景可能不合常理地切换：门里是星空，窗户中是海底
5. **引用超现实大师** — "达利风格""马格利特风格""基里科风格"即梦能理解其视觉语言
