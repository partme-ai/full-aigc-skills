# Architecture — 建筑/室内

## Instructions

建筑/室内类提示词的核心是**空间感和材质**。关键是明确建筑风格（哥特、现代、侘寂、Art Deco 等）+ 空间层次（前景→中景→远景）+ 光线如何定义空间。建筑摄影不同于风景——需要强调结构、线条、比例和材质对比。

## Scenario Patterns

| 子场景 | 配方侧重 | 典型触发词 |
|--------|---------|-----------|
| 宗教建筑 | 拱顶+彩窗+烛光+庄严 | 教堂, 寺庙, 穹顶, 彩窗 |
| 现代住宅 | 材质+落地窗+景观+简约 | 住宅, 极简, 别墅, 客厅 |
| 历史建筑 | 石质+拱门+历史感+暖光 | 城堡, 宫殿, 古迹, 图书馆 |
| 未来建筑 | 曲线+玻璃+金属+参数化 | 未来, 流线型, 扎哈, 科技 |
| 室内设计 | 家具+配色+光线+风格 | 室内, 装修, 侘寂, 北欧 |
| 商业空间 | 灯光+材质+氛围+人流 | 咖啡馆, 餐厅, 酒店, 商店 |

## Examples

### Example: 哥特教堂

**用户需求**：写一个巴黎圣母院内部的庄严场景

**公式拆解**：
- 建筑特征：高耸肋状拱顶 + 彩色玻璃窗
- 光线：彩窗透入斑斓光柱
- 细节：烛光摇曳 + 石柱纹理
- 氛围：庄严神圣
- 构图：广角表现空间纵深
- 风格：建筑摄影

**最终提示词**：
```
巴黎圣母院内部，高耸的肋状拱顶，彩色玻璃窗透入斑斓的光线，形成彩色的光柱，
烛光摇曳，石柱的纹理质感，庄严神圣的氛围，广角镜头表现空间纵深感，建筑摄影风格
```

**为何这样写**：彩色光柱是哥特教堂的视觉标志；广角镜头突出"高耸"的空间感；烛光增加暖色点缀

### Example: 侘寂住宅

**用户需求**：想要一个侘寂风格的极简客厅

**公式拆解**：
- 风格：侘寂（wabi-sabi）
- 材质：艺术涂料墙面 + 木质家具
- 空间关系：落地窗引入庭院绿色
- 点缀：破碎花瓶+枯枝
- 光线：柔和自然光
- 氛围：静谧空灵
- 构图：室内设计摄影

**最终提示词**：
```
侘寂风格的客厅，浅米色的艺术涂料墙面，柳宗理风格的木质家具，大落地窗引入庭院的绿色，
破碎的花瓶中插着枯枝，柔和的自然光，静谧空灵的氛围，室内设计摄影
```

**为何这样写**：侘寂的核心是"不完美之美"——破碎花瓶和枯枝是侘寂的符号；落地窗引入自然模糊室内外边界

### Example: 未来建筑

**用户需求**：扎哈·哈迪德风格的参数化建筑外观

**公式拆解**：
- 建筑特征：流畅白色曲线 + 玻璃幕墙
- 光线：阳光下优雅阴影
- 环境：蓝天+绿色景观
- 风格：未来主义建筑效果图
- 材质对比：清水混凝土 vs 玻璃

**最终提示词**：
```
扎哈·哈迪德风格的参数化建筑设计，流畅的白色曲线在阳光下形成优雅的阴影，
玻璃幕墙反射蓝天，周围有绿色景观，未来主义建筑效果图，清水混凝土与玻璃的对比
```

### Example: 北欧咖啡馆

**用户需求**：冬日北欧咖啡馆的内部，温暖慵懒

**公式拆解**：
- 场景：北欧风格咖啡馆 + 落地窗+窗外白雪
- 室内元素：壁炉 + 毛毯扶手椅 + 现磨咖啡
- 色彩：原木色+白色极简
- 氛围：hygge温暖慵懒
- 光线：室内暖光 vs 窗外冷白

**最终提示词**：
```
北欧风格的咖啡馆内部，落地窗外白雪皑皑，室内有温暖的壁炉，毛毯覆盖的扶手椅，
吧台上现磨咖啡的蒸汽，原木色和白色的极简配色，hygge氛围，温暖慵懒
```

## Category-Specific Tips

1. **建筑风格名称要准确** — "哥特式""包豪斯""侘寂""Art Deco" 即梦都能理解，比模糊的"好看的建筑"强百倍
2. **材质对比制造张力** — 玻璃vs混凝土、木材vs金属、粗糙vs光滑
3. **光线是建筑的第四维度** — 同样的空间，晨光vs正午vs黄昏完全不同，务必指定时间和光线方向
4. **室内类加"窗外景观"** — 窗外的景色（庭院、城市、海景）定义空间的呼吸感
5. **广角/移轴的选择** — 室内用广角表现空间感，建筑外观用移轴防止透视变形
