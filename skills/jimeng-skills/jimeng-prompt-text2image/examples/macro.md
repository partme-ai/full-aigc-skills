# Macro & Close-up — 微距/特写摄影

## Instructions

微距类提示词的核心是**揭示肉眼不可见的细节世界**。昆虫复眼、水珠表面张力、花粉颗粒、雪花晶体——微距摄影让微小之物成为主角。关键是极浅景深和极高的细节要求。

## Scenario Patterns

| 子场景 | 配方侧重 | 典型触发词 |
|--------|---------|-----------|
| 昆虫微距 | 复眼+翅膀纹理+毛刺+颜色 | 昆虫, 蝴蝶, 蜜蜂, 甲虫 |
| 水珠/露珠 | 水珠+折射+倒影+表面张力 | 水珠, 露珠, 水滴, 微距 |
| 雪花/冰晶 | 六角晶体+冰晶结构+微观 | 雪花, 冰晶, 霜花 |
| 花粉/种子 | 花粉粒+种子结构+微观 | 花粉, 种子, 孢子 |
| 织物/材质 | 纤维+编织纹理+磨损 | 纤维, 丝绸, 牛仔, 纸 |
| 眼睛虹膜 | 虹膜纹理+瞳孔+色彩层次 | 眼睛, 虹膜, 瞳孔 |

## Examples

### Example: 蝴蝶翅膀

**用户需求**：写一个蝴蝶翅膀的微距特写

**公式拆解**：
- 主体：蓝色闪蝶翅膀 + 鳞片结构
- 细节：鳞片排列如屋顶瓦片 + 蓝绿色结构色
- 光线：柔和的漫射光
- 构图：极微距+纯色虚化背景
- 画质：显微级细节

**最终提示词**：
```
蓝色闪蝶翅膀的极微距特写，翅膀上的鳞片如微型屋瓦般精密排列，每枚鳞片呈现
从蓝色到绿色的结构色渐变，鳞片边缘有细微的锯齿，极浅的景深让背景完全虚化
为蓝绿色的柔光，微距昆虫摄影，显微级细节，大自然的设计美学
```

### Example: 蛛网露珠

**用户需求**：写一个清晨蛛网上挂满露珠的微距

**公式拆解**：
- 主体：蜘蛛网 + 露珠串
- 细节：水珠折射周围景色+表面张力形成完美球体
- 光线：清晨逆光
- 背景：虚化的绿色+金色光斑
- 风格：微距自然摄影

**最终提示词**：
```
清晨的蜘蛛网微距特写，纤细的蛛丝上串着数十颗晶莹的露珠，每颗水珠因表面张力
形成完美球体并折射出倒置的周围景色，逆光穿透露珠让它们如钻石般闪烁，
背景虚化为柔和的绿色和金色光斑，微距自然摄影风格，清晨的微观仙境
```

### Example: 人类虹膜

**用户需求**：写一个人类眼睛虹膜的极致特写

**公式拆解**：
- 主体：人眼虹膜 + 放射状纹理+色彩层次
- 细节：瞳孔边缘+巩膜血丝
- 光线：环形微距闪光
- 构图：正面+眼球占满画面
- 风格：医学/艺术微距摄影

**最终提示词**：
```
人类眼睛虹膜的极致微距特写，琥珀色的虹膜中放射状的纤维纹理如日冕般从瞳孔向外
扩散，金色、棕色和灰色的色彩层次交织，瞳孔边缘清晰锐利，环形微距灯在眼球上
形成均匀的柔光，睫毛的根部清晰可见，微距摄影艺术，探索人体的微观之美
```

## Category-Specific Tips

1. **微距必写"极浅景深"** — 微距摄影的必然特征，背景彻底虚化
2. **昆虫写复眼和体毛** — 这两个细节最能体现微距的震撼力
3. **水珠写折射** — 水珠中倒映的微缩世界是微距摄影的经典
4. **光线要写环形或柔光** — 微距需要均匀无阴影的光源
5. **材质微距强调纹理** — "纸张纤维""丝绸经纬线""牛仔布的斜纹编织"
