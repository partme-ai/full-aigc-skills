# Medical — 医学/健康

## Instructions

医学类提示词覆盖**医学插图、解剖图示、手术场景和健康生活方式**。医学可视化需要精确——解剖结构、器官比例、色彩编码（动脉红/静脉蓝）都有标准。手术场景强调无菌感、蓝绿色手术服、无影灯。

## Scenario Patterns

| 子场景 | 配方侧重 | 典型触发词 |
|--------|---------|-----------|
| 解剖图示 | 器官+标注线+教科书风格+色彩编码 | 解剖, 心脏, 大脑, 骨骼 |
| 手术场景 | 外科医生+无影灯+器械+手术台 | 手术, 外科, 无影灯, 手术室 |
| 中药/草药 | 药材+药罐+古籍+东方医药 | 中药, 草药, 中医, 本草 |
| 医疗设备 | CT/MRI机器+无菌环境+科技感 | CT, MRI, 医疗设备 |
| 细胞/微观 | 细胞+显微镜+荧光染色+生物 | 细胞, 病毒, DNA, 微观 |
| 心理健康 | 大脑+光影+隐喻+情绪 | 心理健康, 冥想, 大脑 |

## Examples

### Example: 人体心脏解剖

**用户需求**：写一个医学教科书风格的心脏解剖图

**公式拆解**：
- 主体：人体心脏 + 四腔室+主动脉+冠状动脉
- 风格：医学教科书插图 + 色彩编码
- 细节：动脉（红）+静脉（蓝）+标注线
- 背景：干净白色/浅灰
- 画质：极其清晰

**最终提示词**：
```
人体心脏的医学解剖插图，四腔室结构清晰展示——左右心房和左右心室剖面可见，
主动脉弓从心脏顶部延伸而出，冠状动脉如树枝般分布在心肌表面，红色标注动脉系统，
蓝色标注静脉系统，标注线指示各解剖结构名称位置，浅灰色背景，医学教科书插图风格，
极其清晰精准
```

### Example: 中药药材

**用户需求**：写一个传统中药铺的场景

**公式拆解**：
- 场景：传统中药铺 + 百子柜（中药柜）+铜秤
- 细节：散落的干药材+药碾子+古籍医书
- 光线：暖黄纸窗透光
- 氛围：古朴医道
- 风格：纪实摄影

**最终提示词**：
```
传统中药铺的百子柜前，木质抽屉上贴着褪色的药材名标签，黄铜戥子秤放在柜台上
旁边有一本翻开的线装古籍医书，散落着几片黄芪和当归，清晨阳光从纸窗透入
形成柔和的暖黄色光束，空气中漂浮着微尘，古朴医道的氛围，纪实摄影风格
```

### Example: 神经元

**用户需求**：写一个荧光染色的大脑神经元显微画面

**公式拆解**：
- 主体：神经元细胞 + 树突+轴突+突触
- 色彩：荧光绿/蓝/红多色染色
- 背景：深黑 + 散焦光点
- 风格：荧光显微摄影
- 氛围：科学之美

**最终提示词**：
```
荧光染色的大脑神经元显微摄影，神经元的树突如树枝般向外延伸，轴突细长连接远处的
突触末梢，荧光绿色标记细胞体，红色标记神经纤维，蓝色标记胶质细胞，
在深黑色背景上形成璀璨的星空般的神经网络，荧光显微摄影，生命科学之美
```

## Category-Specific Tips

1. **解剖要写标准术语** — 左心房、冠状动脉、海马体、肺泡，专业术语让画面更精准
2. **色彩编码符合医学惯例** — 动脉红/静脉蓝/神经黄/淋巴绿
3. **手术室=蓝绿+无影灯+无菌** — 蓝绿色手术服、不锈钢器械、圆形无影灯是手术场景标配
4. **中药场景要有古旧感** — 木纹、黄铜、线装书、纸窗光，温度偏暖
5. **微观摄影用荧光色** — 现代生物学显微图大量使用荧光蛋白标记，颜色可以是任何鲜艳色调
