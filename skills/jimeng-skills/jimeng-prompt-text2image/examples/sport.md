# Sport & Fitness — 运动/健身

## Instructions

运动类提示词的核心是**动态张力和身体力量**。关键是冻结运动的高潮瞬间——起跳的最高点、冲刺的步伐、击球的瞬间。汗水、肌肉线条、运动装备的细节增强真实感和力量感。

## Scenario Patterns

| 子场景 | 配方侧重 | 典型触发词 |
|--------|---------|-----------|
| 篮球/足球 | 运动员+动作+球场+球+对抗 | 篮球, 足球, NBA, 世界杯 |
| 跑步/田径 | 跑者+跑道/公路+步伐+汗水 | 跑步, 马拉松, 冲刺 |
| 健身/举重 | 肌肉线条+器械+汗水+力量 | 健身, 举重, 健美, 哑铃 |
| 瑜伽/普拉提 | 体式+柔韧+宁静空间+晨光 | 瑜伽, 冥想, 柔韧 |
| 极限运动 | 滑板/冲浪/攀岩+动态+刺激 | 滑板, 冲浪, 攀岩, 跳伞 |
| 游泳/水上 | 泳者+水花+泳池/海+水下 | 游泳, 跳水, 水球 |

## Examples

### Example: 篮球扣篮

**用户需求**：写一个篮球运动员扣篮的爆发瞬间

**公式拆解**：
- 主体：篮球运动员 + 腾空+单手抓球扣篮
- 场景：篮球场 + 篮筐
- 光线：体育馆顶光 + 地板反射
- 动态：汗水飞溅 + 肌肉紧绷
- 构图：低角度仰拍
- 画质：高速快门冻结

**最终提示词**：
```
篮球运动员飞身扣篮的爆发瞬间，身体腾空单手抓球高举即将砸向篮筐，
手臂和腿部的肌肉线条紧绷，汗水在空中飞溅，低角度仰拍突出跳跃的高度和力量感，
体育馆顶光照亮运动员的身体轮廓，高速快门冻结动态，运动摄影风格，极致力量美学
```

### Example: 瑜伽

**用户需求**：写一个海边日出的瑜伽冥想画面

**公式拆解**：
- 主体：女性瑜伽者 + 树式（Vrksasana）+ 白色瑜伽服
- 场景：海边岩石/沙滩 + 日出
- 光线：金色晨光 + 逆光轮廓
- 氛围：宁静平和
- 风格：生活方式摄影

**最终提示词**：
```
女性瑜伽者在海边岩石上做出树式体式，一只脚稳稳扎根另一只脚抵在大腿内侧，
双手合十举过头顶，白色瑜伽服在海风中轻飘，日出金色晨光逆光勾勒出身体轮廓，
远处海面波光粼粼，宁静平和的氛围，瑜伽生活方式摄影
```

### Example: 冲浪

**用户需求**：写一个冲浪者在巨浪中的极限瞬间

**公式拆解**：
- 主体：冲浪者 + 站立冲浪板+身体压低
- 场景：巨大卷曲海浪 + 水花飞溅
- 光线：阳光穿透海浪的翡翠绿色
- 氛围：刺激极限
- 构图：侧面+海浪包围

**最终提示词**：
```
冲浪者在巨大的卷曲海浪中穿行，身体压低在冲浪板上用手轻触浪壁保持平衡，
翡翠绿色的海浪卷成管状包围着冲浪者，阳光穿透海水呈现出半透明的绿色层次，
白色水花在浪尖飞溅，极限运动摄影风格，刺激而壮美的自然之力
```

## Category-Specific Tips

1. **冻结高潮瞬间** — 扣篮的腾空点、冲浪穿管的瞬间、起跑的第一步，选择动作的最高潮
2. **汗水增加质感** — "汗水在皮肤上闪光""汗珠飞溅"是运动摄影的真实感密码
3. **运动器材要准确** — "Spalding篮球""Merino骑行服""碳纤维冲浪板"
4. **角度强化力量** — 仰拍（巨人感/爆发力）、侧面（速度感）、俯拍（战术/全场）
5. **极限运动的危险感** — "悬崖边缘""巨浪包围""高空坠落"，危险让画面更刺激
