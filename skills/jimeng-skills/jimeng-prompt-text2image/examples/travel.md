# Travel — 旅行/探险

## Instructions

旅行类提示词的核心是**目的地独特性和探索感**。每个目的地有其标志性视觉元素——冰岛的极光与黑沙滩、日本的樱花与神社、摩洛哥的马赛克与市场。关键是捕捉"在场感"——让观者感觉自己就在那里。

## Scenario Patterns

| 子场景 | 配方侧重 | 典型触发词 |
|--------|---------|-----------|
| 自然奇观 | 标志地貌+光线+季节+规模感 | 极光, 大峡谷, 冰岛, 班夫 |
| 城市漫游 | 地标建筑+街景+咖啡馆+行人 | 巴黎, 东京, 罗马, 旅行 |
| 公路旅行 | 汽车+公路+旷野+天空+自由 | 公路, 自驾, 66号公路, road trip |
| 极地探险 | 冰雪+极地动物+科考站+极端 | 南极, 北极, 冰川, 极地 |
| 徒步/登山 | 背包客+山径+日出+帐篷 | 徒步, 登山, 露营, hiking |
| 宗教圣地 | 寺庙+朝圣者+香火+庄严 | 寺庙, 朝圣, 西藏, 耶路撒冷 |

## Examples

### Example: 冰岛极光

**用户需求**：写一个冰岛极光下的旅行画面

**公式拆解**：
- 主体：极光 + 绿色/紫色光带在夜空舞动
- 场景：冰岛黑沙滩+玄武岩柱+雪地
- 前景：旅行者背影仰望
- 光线：极光本身的绿光 + 星光
- 构图：广角 + 人小景大
- 氛围：震撼奇幻

**最终提示词**：
```
冰岛黑沙滩上空舞动的北极光，绿色和紫色的光带在星空中如帷幔般飘动，
前景是六边形的玄武岩柱和黑色沙滩上残留的白雪，一个旅行者的小小背影站在沙滩上
仰望极光，广角构图突出天地之大，旅行摄影风格，震撼的自然奇观
```

### Example: 公路旅行

**用户需求**：写一个美国西部公路旅行的经典画面

**公式拆解**：
- 主体：老式敞篷车 + 车尾视角
- 场景：笔直公路 + 沙漠旷野 + 远山
- 天空：壮丽的日落天空
- 光线：金色夕阳
- 氛围：自由奔放
- 风格：旅行纪实摄影

**最终提示词**：
```
美国西部笔直的公路延伸向远方的红色砂岩山脉，一辆复古敞篷车行驶在路上，
从车尾视角看去公路向天际延伸，壮丽的橘红色日落天空占据画面上半部，金色夕阳
将沙漠染成暖橙色，公路两侧是稀疏的仙人掌和灌木，旅行纪实摄影风格，自由公路电影感
```

### Example: 日本神社

**用户需求**：写一个京都伏见稻荷大社的千本�的旅行画面

**公式拆解**：
- 主体：千本鸟居 + 红色鸟居隧道
- 场景：伏见稻荷大社 + 山林步道
- 光线：阳光透过鸟居间隙洒落
- 人物：穿和服的游客
- 氛围：神秘庄重
- 构图：纵深引导线

**最终提示词**：
```
京都伏见稻荷大社的千本鸟居隧道，朱红色的鸟居紧密排列向山上延伸形成视觉隧道，
阳光从鸟居之间的间隙射入在地面投下条纹光影，石阶上一位穿和服的背影缓缓前行，
纵深引导线构图，旅行摄影风格，神秘而庄重的日本传统美学
```

## Category-Specific Tips

1. **地标要写全称** — "巴黎埃菲尔铁塔""京都伏见稻荷大社""冰岛斯科加瀑布"，即梦对知名地标识别度高
2. **时间/季节定氛围** — 樱花季的京都 vs 红叶季的京都，同一个地点完全不同
3. **人小景大** — 旅行摄影中人物是比例尺，"小小的人影""背影""剪影"让观者代入
4. **当地元素是点睛之笔** — 日本的自动贩卖机、意大利的Vespa摩托、印度的突突车
5. **极端环境要写装备** — 极地要写"红色羽绒服"增加画面焦点，沙漠要写"遮阳帽和头巾"
