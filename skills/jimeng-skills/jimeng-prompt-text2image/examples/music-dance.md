# Music & Dance — 音乐/舞蹈/表演

## Instructions

音乐舞蹈类提示词的核心是**动态和节奏感**。音乐是听得见的，要通过视觉元素传达——演奏者的投入表情、乐器的材质光泽、舞者的动态姿态、舞台灯光的戏剧性。关键是捕捉"正在发生"的瞬间感。

## Scenario Patterns

| 子场景 | 配方侧重 | 典型触发词 |
|--------|---------|-----------|
| 乐器演奏 | 乐器细节+手部姿态+演奏表情 | 钢琴, 吉他, 小提琴, 古琴 |
| 演唱会/现场 | 舞台灯光+观众+歌手+能量 | 演唱会, 音乐节, 现场, live |
| 芭蕾舞 | 足尖+纱裙+形体+聚光灯 | 芭蕾, 天鹅湖, 舞者 |
| 街舞/hiphop | 动态pose+街头+宽松穿搭 | 街舞, breaking, 嘻哈 |
| 古典舞蹈 | 水袖+长绸+传统服饰+舞台 | 古典舞, 水袖, 霓裳 |
| 音乐静物 | 乐器+乐谱+暖光+怀旧 | 乐器静物, 唱片机 |

## Examples

### Example: 钢琴演奏

**用户需求**：写一个钢琴家演奏时的特写，要有情感

**公式拆解**：
- 主体：钢琴家双手 + 键盘特写
- 动作：手指在琴键上跳跃
- 场景：三角钢琴 + 昏暗音乐厅
- 光线：单束聚光灯从上方打下
- 氛围：深沉投入
- 构图：俯拍手部特写

**最终提示词**：
```
钢琴家双手在黑白琴键上跳跃的特写，修长的手指以精准而富有情感的力度按下琴键，
黑色三角钢琴的漆面反射着柔和的灯光，昏暗的音乐厅中单束聚光灯从上方照亮键盘和双手，
空气中漂浮着微尘在光束中舞动，音乐演奏摄影，深沉而动人的氛围
```

### Example: 芭蕾舞者

**用户需求**：写一个芭蕾舞者在舞台上的唯美瞬间

**公式拆解**：
- 主体：女芭蕾舞者 + 白色tutu纱裙+足尖鞋
- 姿态：arabesque（阿拉贝斯克）+ 单腿站立+手臂延伸
- 场景：空舞台 + 黑幕布
- 光线：追光聚光 + 纱裙透光
- 风格：舞蹈摄影
- 氛围：优雅空灵

**最终提示词**：
```
芭蕾舞者在空旷的黑暗舞台上做出arabesque姿态，白色tutu纱裙在追光中如云朵般轻盈透光，
足尖鞋踮起脚尖的精确平衡，手臂优雅地向前延伸，单束追光照亮舞者的身体轮廓，
纱裙的薄纱在光线中呈现半透明的层次，舞蹈摄影风格，优雅空灵的极致美感
```

### Example: 摇滚现场

**用户需求**：写一个摇滚演唱会现场的震撼画面

**公式拆解**：
- 主体：吉他手 + 电吉他+甩发动作
- 场景：大型舞台 + 灯光阵列
- 光线：彩色舞台灯光+频闪+烟雾
- 氛围：能量爆发
- 构图：仰拍

**最终提示词**：
```
摇滚演唱会现场，吉他手在舞台上甩动长发激情演奏电吉他，汗水飞洒在空中，
背后的巨型LED屏幕和彩色射灯阵列打出红色和蓝色的强光，舞台烟雾在灯光中形成光束，
仰拍视角突出摇滚巨星的张扬气势，演唱会现场摄影，狂热的能量瞬间
```

## Category-Specific Tips

1. **乐器要写品牌或类型** — "Steinway三角钢琴""Fender Stratocaster电吉他""斯特拉迪瓦里小提琴"
2. **舞蹈姿态用专有名词** — arabesque（阿拉贝斯克）、plié（普利耶）、grand jeté（大跳），专业术语更精准
3. **舞台灯光是灵魂** — 追光、聚光、频闪、烟雾光柱，舞台摄影的核心是光线的戏剧性
4. **动态要有"冻结感"** — 汗珠飞溅、裙摆飘扬、琴弦震动——这些冻结的瞬间传达运动和能量
5. **观众反应制造氛围** — 演唱会中加入"观众举手机闪光灯如星海"营造现场感
