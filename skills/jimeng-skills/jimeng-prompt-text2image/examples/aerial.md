# Aerial — 航拍/空中视角

## Instructions

航拍类提示词的核心是**鸟瞰视角和大地纹理**。从空中看，山川、城市、田野变成了抽象图案和纹理。关键是明确飞行高度和角度——正俯拍（90度）看图案，斜俯拍（45度）看立体和光影。

## Scenario Patterns

| 子场景 | 配方侧重 | 典型触发词 |
|--------|---------|-----------|
| 自然航拍 | 地貌纹理+光影+河流曲线+俯瞰 | 航拍, 鸟瞰, 俯瞰, 高空 |
| 城市航拍 | 路网+建筑群+夜景灯光+天际线 | 城市航拍, 城市俯瞰 |
| 海岸线 | 海陆交界+浪花+沙滩纹理 | 海岸, 海岛, 沙滩航拍 |
| 农田/茶园 | 几何纹理+色彩对比+农民 | 梯田, 茶园, 农田, 大地 |
| 山脉航拍 | 山脊线+云海+雪顶+光影 | 山脉, 群山, 云海之上 |
| 无人机自拍 | 人物+环境+垂直俯拍 | 无人机, 自拍, 高空自拍 |

## Examples

### Example: 梯田航拍

**用户需求**：写一个龙脊梯田的航拍画面

**公式拆解**：
- 主体：梯田 + 水镜面反射 + 曲线纹理
- 时间：日落时分
- 光线：金色夕阳斜照 + 水面反光
- 视角：45度斜俯拍
- 氛围：壮观
- 风格：航拍风光摄影

**最终提示词**：
```
龙脊梯田的航拍视角，层层叠叠的梯田在夕阳下形成优美的曲线纹理，每层水田如镜面
反射着金色天空，山间薄雾缭绕增加了层次感，45度斜俯拍展现梯田的立体落差，
航拍风光摄影风格，壮丽的大地纹理
```

### Example: 城市路网

**用户需求**：写一个立交桥的航拍夜景

**公式拆解**：
- 主体：复杂立交桥 + 车轨光流
- 时间：蓝调时刻向黑夜过渡
- 光线：车灯形成的光轨 + 路灯网
- 视角：正俯拍（90度）
- 风格：城市航拍
- 画质：长曝光

**最终提示词**：
```
立交桥的垂直航拍夜景，复杂的多层匝道在空中交织成几何图案，车流形成红色尾灯和
白色前灯的光轨在桥面上流动，路灯阵列构成发光的几何网格，正俯拍90度视角，
长曝光城市航拍，车轨光流如同城市的血管
```

### Example: 海岛航拍

**用户需求**：写一个热带海岛的航拍，蓝绿渐变色

**公式拆解**：
- 主体：海岛 + 环礁 + 白沙
- 海水：蓝绿渐变 + 透明浅滩
- 视角：斜俯拍 + 广角
- 点缀：小船
- 风格：旅行航拍

**最终提示词**：
```
热带海岛的航拍视角，翠绿的岛屿被白色沙滩环绕，浅滩海水呈现从透明到蒂芙尼蓝
再到深蓝的渐变层次，珊瑚礁的暗影在浅水区隐约可见，一艘白色小船漂浮在碧蓝海面上，
斜俯拍广角构图，旅行航拍摄影风格，人间天堂
```

## Category-Specific Tips

1. **角度决定画面性质** — 90度正俯拍=抽象图案感，45度斜俯拍=立体空间感
2. **光影在高空更明显** — 低角度阳光在山脉、建筑上投下的长阴影是航拍的关键层次
3. **云/雾增加纵深感** — 云层或薄雾在山间制造的层次让航拍不扁平
4. **大地纹理是航拍独有优势** — 梯田曲线、农田色块、河流蜿蜒、沙丘波纹
5. **季节颜色** — 春天的油菜花田（金黄）、秋天的稻田（金黄）、雪后（纯白+线条）
