# Street Photography — 城市/街拍/纪实

## Instructions

街拍类提示词的核心是**真实感和瞬间感**。捕捉城市生活的真实瞬间——行人、光影交错的街道、老建筑和现代元素的碰撞。关键是"不完美"的真实感：不对称构图、自然抓拍、环境中的偶然元素。

## Scenario Patterns

| 子场景 | 配方侧重 | 典型触发词 |
|--------|---------|-----------|
| 城市街拍 | 行人+建筑+光影+抓拍感 | 街拍, 城市, 扫街, street |
| 老城小巷 | 石板路+老建筑+晾衣绳+居民 | 老城, 巷子, 古城, 胡同 |
| 地铁/公交 | 车厢+乘客+窗影+通勤 | 地铁, 公交, 通勤 |
| 市场/集市 | 摊位+商品+买卖+烟火气 | 市场, 集市, 菜市场, 夜市 |
| 涂鸦/街头艺术 | 壁画+墙面+色彩+行人 | 涂鸦, 街头艺术, 壁画 |
| 城市建筑 | 建筑立面+几何+光影+对称 | 建筑, 摩天楼, 立面 |

## Examples

### Example: 雨夜街拍

**用户需求**：写一个雨夜城市的街拍画面，要有电影感

**公式拆解**：
- 场景：雨夜都市街道 + 湿润路面+霓虹倒影
- 人物：穿风衣的行人过马路
- 光线：霓虹灯+车灯+路灯混合
- 风格：纪实街拍+电影感
- 氛围：孤独都市感

**最终提示词**：
```
雨夜都市十字路口，湿润的沥青路面反射着霓虹灯和车灯的彩色光芒，一位穿卡其色风衣的行人
撑黑色雨伞走过斑马线，路灯在雨幕中形成柔和光晕，远处高楼灯火通明，
纪实街拍风格，电影感冷暖色调对比，都市孤独氛围
```

### Example: 老城小巷

**用户需求**：写一个欧洲老城小巷的晨光画面

**公式拆解**：
- 场景：欧洲老城狭窄小巷 + 鹅卵石路+老建筑
- 光线：清晨低角度阳光 + 长阴影
- 细节：自行车靠墙 + 窗台鲜花 + 猫
- 风格：旅行纪实摄影
- 氛围：静谧安逸

**最终提示词**：
```
欧洲老城狭窄的鹅卵石小巷，清晨低角度的阳光在石板路上投下长长的阴影，一辆复古自行车
靠在斑驳的石墙上，窗台上的红色天竺葵正在盛开，一只橘猫蹲在门前台阶上，
旅行纪实摄影风格，温暖的金色晨光，静谧安逸
```

### Example: 夜市烟火

**用户需求**：写一个亚洲夜市的烟火气场景

**公式拆解**：
- 场景：夜市摊位 + 蒸汽+灯光+人群
- 主体：小吃摊位 + 升腾的白色蒸汽
- 光线：暖黄灯泡串 + 霓虹招牌
- 氛围：热闹市井烟火气
- 构图：中景 + 纵深引导线

**最终提示词**：
```
亚洲夜市的热闹场景，小吃摊位上升腾着白色蒸汽，暖黄的灯泡串横跨街道，
霓虹招牌的红蓝光映在来往行人的脸上，摊位老板正在翻炒铁锅中的食物，
纵深构图展现街道的延伸感，热闹市井烟火气，纪实街拍风格
```

## Category-Specific Tips

1. **抓拍感 > 完美构图** — 街拍的魅力在于真实，不要过度追求完美对称
2. **光影对比是街拍灵魂** — 强光与深影的交界处往往是最佳拍摄点
3. **善用"路人"制造生活感** — "匆匆走过的行人""依偎等待的情侣"让画面有故事
4. **天气增加氛围** — 雨后的湿润地面、雾天的朦胧、雪中的脚印
5. **城市要有可辨识元素** — 红电话亭=伦敦, 黄色出租车=纽约, 霓虹招牌=东京/香港
