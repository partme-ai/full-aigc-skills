# Cinematic — 电影感/氛围

## Instructions

电影感类提示词的核心是**用光线讲故事**。不同于写实摄影的"记录"，电影感强调情绪、叙事和氛围。关键是确立一个明确的"情绪锚点"——忧郁、浪漫、孤独、紧张、怀旧——然后用光线和色彩来承载这个情绪。

## Scenario Patterns

| 子场景 | 配方侧重 | 典型触发词 |
|--------|---------|-----------|
| 雨夜都市 | 湿润地面+霓虹倒影+冷暖对比 | 雨夜, 霓虹, 孤独, 都市 |
| 旅途/火车 | 车窗+运动模糊+怀旧色调 | 旅途, 火车, 窗外, 在路上 |
| 酒吧/爵士 | 昏暗暖光+烟雾+乐器+颓废 | 酒吧, 爵士, 威士忌, 深夜 |
| 旧书店/时光 | 书架+尘埃光束+暖金+静谧 | 书店, 时光, 怀旧, 温暖 |
| 水下/梦幻 | 水波光+蓝调+漂浮感+静谧 | 水族馆, 水下, 梦幻, 蓝调 |
| 公路/旷野 | 公路+天空+孤独+长焦压缩 | 公路, 旷野, 旅行, 自由 |

## Examples

### Example: 雨夜都市

**用户需求**：写一个雨夜东京街头的氛围画面，要有王家卫的感觉

**公式拆解**：
- 场景：东京涩谷 + 霓虹灯招牌 + 湿润柏油路
- 人物点缀：穿风衣行人 + 透明伞
- 光线：居酒屋暖橙光 vs 夜色冷蓝 + 霓虹灯
- 风格：赛博朋克电影感 + 王家卫色调
- 氛围：冷暖强烈对比
- 画质：电影级画质

**最终提示词**：
```
雨夜的东京涩谷，霓虹灯招牌的彩色倒影在湿漉漉的柏油路上，穿风衣的行人撑透明伞
匆匆走过，温暖的橘色灯光从居酒屋透出，冷蓝与暖橙的强烈对比，赛博朋克电影感
```

**为何这样写**：雨夜都市的灵魂在"湿润地面+霓虹倒影+冷暖对比"三件套；行人只是一个剪影点缀，不抢主体

### Example: 列车旅途

**用户需求**：想要行驶中列车窗外的画面，有旅行的诗意

**公式拆解**：
- 场景：列车车厢内 + 窗外田野风光
- 细节：窗玻璃雨滴 + 模糊风景快速后退
- 光线：车厢昏黄灯光
- 风格：复古胶片质感
- 氛围：怀旧旅行感
- 色彩：暖木色+窗外绿色

**最终提示词**：
```
行驶中的列车窗外的田野风光，窗玻璃上有雨滴，模糊的绿色田野和远山在窗外快速后退，
温暖的木质车厢内饰和昏黄的灯光，复古胶片质感，旅行的诗意和怀旧氛围
```

**为何这样写**：窗户作为"画中画"框架构图；雨滴+运动模糊制造"在路上"的流动感；暖冷对比（车厢暖vs窗外冷）

---

### Example: 爵士酒吧

**用户需求**：写一个深夜爵士酒吧的氛围，颓废而浪漫

**公式拆解**：
- 场景：深夜爵士酒吧
- 主体/道具：吧台 + 半杯威士忌 + 冰块折射
- 后景：乐队剪影 + 萨克斯
- 光线：蓝紫昏暗灯光
- 氛围：烟雾袅袅 + 颓废浪漫
- 风格：王家卫电影风格

**最终提示词**：
```
深夜的爵士酒吧，蓝色和紫色的昏暗灯光，吧台上放着半杯威士忌，冰块在酒液中折射光芒，
后景的乐队剪影在演奏萨克斯，烟雾袅袅，王家卫电影风格，颓废而浪漫
```

### Example: 水族馆

**用户需求**：水族馆玻璃墙前的梦幻场景

**公式拆解**：
- 场景：巨大水族馆玻璃墙
- 主体元素：蓝色海水 + 游动鱼群
- 人物：人物剪影站在玻璃前仰望
- 光线：水面投射的波光粼粼
- 色彩：蓝紫色
- 氛围：梦幻静谧

**最终提示词**：
```
巨大的水族馆玻璃墙前，蓝色的海水和游动的鱼群，光线从水面投射下来形成波光粼粼的效果，
人物剪影站在玻璃前仰望，蓝紫色的神秘氛围，梦幻而静谧
```

## Category-Specific Tips

1. **光线即情绪** — 暖光=温馨/怀旧，冷光=孤独/神秘，冷暖对比=冲突/故事感
2. **不要过度描述人物** — 电影感人像中，人物常是"剪影""背影""模糊的身影"，让环境承载情绪
3. **天气是情绪放大器** — 雨=忧伤/浪漫，雾=神秘/迷茫，雪=纯净/孤独
4. **引用导演风格有效** — "王家卫风格""韦斯·安德森风格""是枝裕和风格"即梦能理解其色调和构图偏好
5. **电影感≠暗调** — 也可以做明亮电影感（韦斯·安德森的对称构图+高饱和粉色），不要默认"电影感=暗"
