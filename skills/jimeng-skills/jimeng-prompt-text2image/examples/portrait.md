# Portrait — 人像摄影

## Instructions

人像类提示词的核心是**人物塑造**。主体（人物特征+服装+表情+姿态）占提示词约50%的比重，然后用环境、光线和风格来完成氛围。即梦对人像的理解力很强，关键在于用具体的视觉细节替代抽象形容。

## Scenario Patterns

| 子场景 | 配方侧重 | 典型触发词 |
|--------|---------|-----------|
| 时尚写真 | 服装+光线+构图 | 写真, 街拍, 时尚, 杂志风 |
| 古装人像 | 服装+环境+国风 | 汉服, 古装, 旗袍, 侠客, 仙气 |
| 生活日常 | 环境+表情+自然光 | 日常, 生活感, 自然, 抓拍 |
| 职业肖像 | 服装+光线+构图 | 职业照, 商务, 头像, 正装 |
| 情绪表达 | 光线+表情+氛围 | 忧郁, 开心, 孤独, 沉思 |
| 情侣/多人 | 互动+环境+氛围 | 情侣, 闺蜜, 家庭, 合照 |

## Examples

### Example: 时尚街拍

**用户需求**：帮我写一个都市女性雨后街头的时尚写真提示词

**公式拆解**：
- 主体：都市女性 + 米色风衣 + 透明雨伞
- 场景：雨后外滩 + 湿润街道 + 霓虹灯倒影
- 动作：撑伞行走
- 风格：电影感摄影
- 光线：冷暖对比（霓虹暖光 vs 夜色冷蓝）
- 构图：中景 + 引导线（街道延伸）
- 画质：超写实

**最终提示词**：
```
穿米色风衣的都市女性走在雨后外滩，撑一把透明雨伞，街灯在地面水洼中形成倒影，
霓虹灯招牌的暖光反射在湿润的柏油路上，电影感冷暖对比色调，湿发纹理清晰，超写实
```

**为何这样写**：风衣+透明雨伞建立时尚感；水洼倒影增加画面层次；冷暖对比是电影感的核心手法

### Example: 汉服古装

**用户需求**：想要一个明制汉服女性在宫墙前的画面

**公式拆解**：
- 主体：东方女性 + 红色圆领袍+马面裙 + 团扇
- 场景：朱红宫墙前
- 动作：手持团扇 + 微风拂动衣袂
- 风格：写实摄影
- 光线：金色夕阳逆光 + 发丝光晕
- 构图：中景 + 居中
- 画质：8k超清 + 浅景深

**最终提示词**：
```
一位身着明制汉服的东方女性，红色圆领袍配马面裙，手持团扇站在朱红宫墙前，
微风拂动发丝和衣袂，金色夕阳逆光，发丝边缘勾勒出光晕，写实摄影，浅景深，8k超清
```

**为何这样写**：逆光是古风人像的灵魂，发丝光晕营造仙气；朱红宫墙 + 红色汉服的同色系搭配，用金色逆光分离主体

### Example: 情绪肖像

**用户需求**：写一个老人面部的黑白特写，要有故事感

**公式拆解**：
- 主体：老人 + 岁月痕迹的皮肤 + 深邃双眼
- 场景：无特定（浅景深虚化）
- 动作：静态凝视
- 风格：黑白摄影
- 光线：硬朗侧光强调纹理
- 构图：面部特写
- 画质：超高细节 + 极浅景深

**最终提示词**：
```
老人的面部特写，布满岁月痕迹的皮肤纹理，深邃的双眼饱含故事，黑白摄影风格，
硬朗的侧光强调纹理和轮廓，极浅景深，超高细节
```

**为何这样写**：黑白去掉色彩干扰，聚焦纹理和光影；硬侧光制造高反差，强调皮肤质感；极浅景深让视线聚焦双眼

### Example: 职业商务

**用户需求**：需要一张专业商务人像，男性，在办公室

**公式拆解**：
- 主体：白领男性 + 深蓝西装 + 自信微笑
- 场景：落地窗前 + 城市天际线
- 动作：双手插兜站立
- 风格：商业人像摄影
- 光线：柔和办公顶光
- 构图：中景 + 干净利落
- 画质：4k

**最终提示词**：
```
穿深蓝西装的白领男性站在落地窗前，双手插兜，自信微笑，窗外是城市天际线，
柔和的办公环境顶光，商业人像摄影风格，干净利落的构图，4k
```

## Category-Specific Tips

1. **眼神是关键** — 即梦对眼神的渲染较好，善用"深邃的眼神""温柔的凝视""警觉的目光"等描述
2. **手部可能变形** — 避免复杂的手部动作描述（如"手指摆出V字"），简单的手部姿态（手持物品、自然下垂、插兜）更稳定
3. **服装材质可用比喻** — "丝绸般的光泽""皮革的肌理""棉麻的松弛感"比直接说"好看的衣服"有效得多
4. **多人场景需注意** — 两人以上时，用"一对情侣""母子""一群朋友"等概括性描述比逐一描写每个人更可靠
5. **表情用场景暗示** — 不说"她很高兴"，而写"她嘴角微微上扬，眼中有光"，把情绪转化为视觉线索
