# 即梦图片4.0模型 提示词手册

> 来源: 即梦官方飞书知识库 | 最后更新: Jan 14, 2026

## 核心更新

🔹 即梦在图片4.0模型**首次支持多模态生图**，同一模型实现：
- **文生图**（Text-to-Image）
- **图像编辑**（Image Editing）
- **组图生成**（Multi-image Generation）

能够通过自然语言灵活控制画面细节。

## 模型亮点

| 亮点 | 说明 |
|------|------|
| **1. 精准指令编辑** | 自然语言指令精确控制编辑效果 |
| **2. 高度特征保持** | 编辑后保持原图特征一致性 |
| **3. 深度意图理解** | 理解复杂语义需求 |
| **4. 多图输入输出** | 支持多图输入与组图生成 |
| **5. 超高速超高清** | 快速生成 + 高清画质 |

## 提示词写作建议（4.0 特有）

1. **多模态指令**：4.0 支持自然语言编辑指令，可以直接说"把背景换成沙滩"而不是用复杂的参数
2. **元素增减时保持一致性**：使用"保持原图的..."等描述引导特征保留
3. **风格转换**：3.0→4.0 风格迁移效果更自然，大胆尝试跨界风格
4. **组图生成**：可为一组图片统一风格或主题，适合系列内容创作
5. **编辑与生成结合**：先文生图再编辑，实现精细控制
