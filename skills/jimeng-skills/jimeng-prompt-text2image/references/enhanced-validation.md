# Validation Checklist (Enhanced)

In addition to `rules/validation-checklist.md`, verify:

- [ ] No hex color codes (`#RRGGBB`) anywhere in the prompt text — use Chinese color names (大红, 胭脂, 金色) only
- [ ] Color names are natural Chinese, not literal translations (e.g. "大红" not "非常大的红色")
- [ ] For 3.0: text content wrapped in `"引号"` is only for the actual text to render, not for color names
