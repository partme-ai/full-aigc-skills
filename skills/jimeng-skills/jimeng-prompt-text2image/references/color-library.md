# Color Library — 色彩参考库

> 整合中国传统色与国际标准色两大体系，为即梦文生图提示词提供精准的色彩描述词汇。
> 色码格式均为十六进制 (#RRGGBB)，可直接用于设计参考和色彩搭配。

---

## 一、中国传统色

> 近 160 种传统色，按其在中国古典文化中的色相归属分为九大色系。每一色名皆源于自然、生活与诗词，凝结了古人对天地万物的观察与诗意想象。
>
> "云髻不饶荷叶绿，罗裙未减石榴红。" —— 古人以色入诗，以色传情。

## 二、国际标准色

> 涵盖 Pantone、RAL 两大工业色标体系，以及莫兰迪高级灰调、国际通用专业色名、配色方案与电影色彩风格。

### 1. 国际标准色彩体系总览

| 体系 | 全称 / 来源 | 特点 | 适用领域 |
|------|-----------|------|---------|
| Pantone | Pantone Matching System (PMS) | 国际通用专色标准，数千种编号色，每年发布年度色 | 品牌设计、时尚、印刷 |
| RAL | Reichsausschuß für Lieferbedingungen（劳尔色卡） | 德国工业色标：RAL Classic 213 色 + RAL Design 1688 色 | 工业涂料、建筑、冶金、产品设计 |
| Munsell | Munsell Color System | 基于色相(Hue)/明度(Value)/彩度(Chroma)三维科学体系 | 科学、教育、土壤学 |
| NCS | Natural Color System | 基于黑度/白度/彩度/色相四维，北欧标准 | 建筑、室内、工业设计 |
| Morandi | 乔治·莫兰迪 (Giorgio Morandi) | 低饱和高级灰调，柔和内敛 | 极简设计、侘寂、北欧风格 |

### 3. RAL 国际色卡（劳尔色卡）

> **RAL**（Reichsausschuß für Lieferbedingungen，德国交货条件委员会）成立于 1925 年，是全球最权威的工业色彩标准之一。中文译为**劳尔色卡**，在国际上广泛通用，尤其适用于冶金、涂料、建筑和工业设计。
>
> RAL 包含两大体系：
> - **RAL Classic**：213 种经典工业色，以 4 位数字编码（RAL 1xxx-9xxx），按色相分组
> - **RAL Design**：1688 种设计色，7 位数字编码，基于 CIE 国际标准色彩空间，按色度(H)、亮度(L)、色品(C) 排列，适合专业色彩设计

#### RAL Design 系统 (RAL D)

> RAL Design 系统包含 **1688 种颜色**，使用 7 位数字编码（如 `RAL 270 30 20`），基于 CIE 国际标准色彩空间，编码含义为：
>
> `RAL H LL CC`：**H** = 色度角 (0°–360°，0=红, 60=黄, 120=绿, 180=青, 240=蓝, 300=紫) · **LL** = 明度 (0=黑, 100=白) · **CC** = 彩度/饱和度

| 参数 | 范围 | 说明 |
|------|------|------|
| 色度角 H | 000–360 | 绕色环的角度，决定色相 |
| 明度 LL | 00–99 | 越高越亮，越低越暗 |
| 彩度 CC | 00–99 | 越高越鲜艳，越低越灰 |

**RAL Design 阅读示例**：`RAL 270 30 20` = 色相角 270°（蓝紫色）、明度 30（偏暗）、彩度 20（中低饱和度/灰调蓝紫）

#### RAL 在 AI 提示词中的使用

- **直接引用色号**：写 `RAL 5002 群青蓝`、`RAL 3000 火焰红` 指定精确工业色
- **材质+RAL**：写 `RAL 9005 墨黑色哑光金属外壳` 或 `RAL 1013 浅灰色混凝土墙面`
- **工业/建筑风格必备**：描述包豪斯、工业风、极简空间时使用 RAL 色名可精准控制色板
- **德国工业美学**：`RAL 7016 煤灰 + RAL 2004 纯橙` 的组合暗示包豪斯/德国功能主义配色

### 4. 莫兰迪色系 (Morandi Palette)

> 意大利画家乔治·莫兰迪的标志性用色——所有颜色都掺入了一层灰度，低饱和、柔和、高级。非常适合极简、侘寂、北欧和文艺风格。

`灰粉` · `灰绿` · `灰蓝` · `烟灰紫` · `燕麦色` · `豆沙色` · `雾霾蓝` · `干枯玫瑰色` · `奶茶色` · `灰米色` · `鼠尾草绿` · `烟熏紫` · `灰橘` · `灰杏` · `灰棕`

**莫兰迪配色要诀**：在任意颜色的描述前加"灰调""烟熏""雾感"等前缀，即可获得莫兰迪美感。例如"灰调珊瑚""雾感蓝""烟熏玫瑰"。

### 7. 专业配色方案 (Color Schemes)

| 方案 | 英文 | 色轮关系 |
|------|------|---------|
| 单色调 | Monochromatic | 单一色相的不同明度/饱和度变体 |
| 互补色 | Complementary | 色轮正对面（蓝-橙、紫-黄、红-绿） |
| 分裂互补 | Split-Complementary | 一色 + 对面两邻色，冲突弱于纯互补 |
| 类似色 | Analogous | 色轮相邻三色（蓝-青-绿），和谐统一 |
| 三色调 | Triadic | 色轮三等分（红-黄-蓝），均衡活泼 |
| 四色调 | Tetradic | 两组互补色，丰富而平衡 |
| 暖色调 | Warm Palette | 红、橙、黄为主 |
| 冷色调 | Cool Palette | 蓝、绿、紫为主 |
| 大地色 | Earth Tones | 棕、赭、橄榄、陶土等自然色 |
| 宝石色 | Jewel Tones | 深饱和高贵色（祖母绿、宝石蓝、紫水晶、红宝石） |
| 粉彩色 | Pastel | 柔和低饱和浅色，轻盈雅致 |
| 霓虹色 | Neon | 高饱和荧光感，冲击力强 |
| 灰度 | Grayscale | 纯黑-白-灰 |
| 棕褐色调 | Sepia | 暖棕复古，如旧照片 |
