## 模型亮点

**专业实拍、风格精准、细节大师、高效设计**

### 1. 专业实拍：每一帧都是专业级质感
- 电影级画面：场景更丰富完整、光影层次细腻，色调统一
- 电影感和故事性增强
- 典型场景：**电影感、爱情片、史诗片、公路片**

### 2. 风格精准：突破创作想象边界
### 3. 细节大师：以假乱真的视觉冲击
### 4. 高效设计：兼具创意和多样性

## 提示词建议

- 3.1 模型在**电影感画面**和**故事性**方面有突出表现
- 推荐使用电影类型关键词（爱情片、史诗片、公路片等）引导画面风格
- 风格精准度高，可大胆尝试风格化描述
- 结合 3.0 的词库使用效果更佳（3.1 与 3.0 的响应词典结构相似）
