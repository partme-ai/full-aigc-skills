# 即梦图片4.5模型 提示词手册

> 来源: 即梦官方飞书知识库 | 最后更新: Dec 12, 2025

## 核心更新

🔹 即梦图片4.5相较于4.0有**整体提升**，在以下方面得到显著改善：
- **人像场景**：4.0高频反馈问题得到改善
- **美观度**：画面美感提升
- **推理能力**：语义理解与逻辑推理增强

## 模型亮点

与 4.0 结构一致，但各维度均有提升：

| 亮点 | 说明 |
|------|------|
| **1. 精准指令编辑** | 自然语言指令精确控制编辑效果 |
| **2. 高度特征保持** | 编辑后保持原图特征一致性 |
| **3. 深度意图理解** | 理解复杂语义需求，推理能力更强 |

| **5. 超高速超高清** | 快速生成 + 高清画质 |

## 提示词写作建议（4.5 特有）

1. **人像优先**：4.5 在人像场景改善显著，人像类提示词效果更佳
2. **美观度提升**：画面美感更优，可减少额外美观度修饰词
3. **推理能力增强**：支持更复杂的语义描述和逻辑关系
4. **兼容4.0写法**：4.0 的提示词写法在 4.5 上效果更好，无需大幅调整
5. **长提示词可用**：支持 100+ 字符的详细描述，充分利用公式
