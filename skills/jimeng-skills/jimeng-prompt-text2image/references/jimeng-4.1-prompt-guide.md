## 提示词手册

### 图片参考编辑

#### 1. 海报排版
- **文案编辑**：修改海报文字内容、字体、排版
- **多图排版**：多张图片组合排版

#### 2. 人像场景
- **美化**：人像美化、修肤、美颜
- **妆容**：妆容迁移、调整
- **发型**：发型变换
- **表情**：表情调整
- **服装**：服装更换、风格调整

#### 3. 通用编辑词
- **画质**：清晰度、画质增强
- **光影**：光照调整、色调变化
- **环境**：背景/环境替换
- **消除修复**：去除瑕疵、修复缺陷
- **添加合成**：添加元素、合成画面
- **调整**：整体画面调整

#### 4. 多图一致性玩法
多张图片保持统一风格/主题

#### 5. 多轮编辑能力
支持连续多轮编辑，质量不压缩

## 完整提示词示例（4.1 编辑工作流）

### 新年拜年海报 — 多轮编辑工作流

```
【第1轮 文生图】传统中式庭院新年场景，两个穿大红唐装的小孩子拱手拜年，大红灯笼高挂，春联福字贴满大门，暖色调喜庆氛围，电影级画质

【第2轮 海报排版编辑】在画面正上方添加文字"百年好合"，金色优雅手写体，适当大小不遮挡人物，字体居中排版

【第3轮 人像美化】对两个孩子进行美化：脸色更红润，笑容更自然灿烂，去除脸部噪点，增强眼神光

【第4轮 风格迁移】将整体风格迁移为国潮插画风，保留人物特征和文字不变，背景简化更有设计感，画面更干净
```

### 人像编辑 — 妆容迁移

```
【文生图】一位年轻女性在樱花树下，身穿白色连衣裙，自然光线，清新风格
【编辑】将模特妆容改为桃花妆，粉色眼影和腮红，红唇，保留五官特征不变
```

### 海报迁移

```
【上传参考海报】保持原海报的文案和布局
【编辑】将海报风格从极简风迁移为复古国潮风，红色和金色为主色调，加入祥云纹样和中国结元素
```

## 提示词写作建议（4.1 特有）

1. **多轮编辑**：4.1 支持多轮编辑，可分步调整（先改背景→再加文字→再调色调），每轮质量不压缩
2. **海报场景优先**：4.1 在海报编辑和海报迁移方面提升显著，适合营销海报制作
3. **人像增强**：妆容迁移、服装编辑、表情调整效果更好，适合电商模特图/时尚摄影
4. **一致性玩法**：利用参考图一致性提升，可批量生成系列图片
5. **影视风格**：影视风格美感提升，适合电影感海报/短片封面
