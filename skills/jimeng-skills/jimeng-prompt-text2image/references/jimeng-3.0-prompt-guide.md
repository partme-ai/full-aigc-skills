# 即梦图片3.0模型 提示词手册

官方提示词攻略（来源：ByteDance Feishu Wiki）

## 模型核心亮点

| 亮点 | 说明 |
|------|------|
| **真实** | 去AI味、以假乱真 |
| **高清** | 默认1k、可支持2K |
| **专业** | 专业级的画面效果 |
| **文字** | 更准确的字体响应、专业的设计排版 |

### 1. 真实：去AI味、以假乱真

**质感真实：** 去磨皮感、假脸感、AI油腻感
**情绪真实：** 去僵硬感、摆拍感、AI眼神空洞感，情绪细腻有感染力

### 2. 高清：默认1K、可支持2K

图像清晰度和结构准确性显著提升，2K时画面结构更准确

### 3. 专业：专业级的画面效果

**影像场景：** 更准确的电影类型片、镜头语言响应
**动漫场景：** 风格更多元、细节更丰富、色调更统一，不再有2.1的"抠图感"

### 4. 文字：更准确的字体响应、专业的设计排版

- 大字小字都能响应、支持不同字体（对比2.1模型只能响应标题大字）
- 3.0解锁了更多有设计感的**艺术字体**、细致区分了不同**字重**
- 多样字体搭配设计排版和高美感底图，有效降低后期成本

## 提示词秘籍

### 核心原则

1. **用简单直接的语言表达** — 短提示词也能出惊艳效果。长提示词并不一定更好，自己输入的提示词可能比AI生成的更有效
2. **用连贯的自然语言描述画面内容**（主体+行为+环境等），用**短词语**描述画面美学（风格、色彩、光影、构图等）
3. **文字内容用`"引号"`包裹**，可以提升文字准确率
4. **专业词汇推荐用原文/英文**，效果更准确
5. **推荐写明图像用途和类型** — 例如：用途=`PPT封面背景图、背景素材图` / 类型=`广告海报设计、纪实摄影`

### 示例

```
广告海报设计，卡通Q版风格，文字为"呼伦贝尔肉业集团春季推广"，位于左下角，还有日期和地点信息，字体为小号。
左上角和右侧有新鲜多样的食材图片，动态显示春季生机。左下角有一组新上市的牛肉产品。
暖色调，高饱和度，橙黄色和绿色搭配。广角侧面动态构图，仰视视角，中景拍摄，中景到全景，光影强烈对比，强烈反射折射效果。
超高质量渲染，强烈反射折射效果。3D渲染，数字艺术，C4D与OC渲染器。
```

## 响应词典分类结构

即梦3.0模型的提示词响应词典按以下结构组织：

1. **热门词** — 高频常用风格词（胶片、闪光灯、过曝、液态金属、ins风等）
2. **美学词** — 色彩、光线、构图、视角、景别
3. **潮流风格** — 插画、动漫、卡通、像素、3D风格等
4. **影像风格** — 电影类型、摄影流派
5. **设计风格** — 字体设计、版式构图
6. **艺术风格** — 国画、版画、水彩、岩彩等传统/现代艺术
7. **材质** — 毛绒、毛毡、玻璃、青铜、粘土、果冻、烫金等

See `references/jimeng-3.0-vocabulary.md` for the full categorized vocabulary list.
