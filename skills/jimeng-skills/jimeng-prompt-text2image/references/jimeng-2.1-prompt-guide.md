# 即梦图片2.1模型 提示词手册

> 来源: 即梦官方飞书知识库 | 最后更新: Jul 14, 2025

## 模型概述

2.1 模型相比 2.0 各维度提升：

| 维度 | 提升内容 |
|------|----------|
| **构图** | 景别、视角变化多样性提升，画面层次感增强 |
| **光影** | 去除刻意光线表达，画面更真实自然 |
| **色彩** | 色调统一性增强，去除高对比混乱杂色 |
| **质感** | 人物皮肤磨皮感及油腻感减弱，纹理更真实细腻 |
| **细节** | 简化过于繁琐元素，画面更有层次感、秩序感 |
