# 《中国传统色：故宫里的色彩美学》384 色体系

> 文化学者郭浩与故宫文创设计师李健明合著（中信出版社，2020 年），从 24 节气、72 物候、96 件手绘故宫文物中考据出 384 种中国传统色。
> 每种颜色均标注 CMYK 与 RGB 色值，附有文献考证。这是目前最权威的中国传统色体系著作，豆瓣评分 9.0+。
>
> **体系结构**：24 节气 × 4 组物候 × 4 色（起·承·转·合） = 384 色
>
> **四季主色调**：春青 · 夏赤 · 秋金 · 冬墨
>
> **使用场景**：当提示词需要中国传统色时优先查此文件。按节气检索——知道用户要什么季节/月份的意境，找到对应节气的色组，色名本身即是一句诗。

## 一、色彩体系基础

### 1.1 五正色与五间色

| 五正色 | 青 | 赤 | 黄 | 白 | 黑 |
|--------|-----|-----|-----|-----|-----|
| **五行** | 木 | 火 | 土 | 金 | 水 |
| **方位** | 东 | 南 | 中 | 西 | 北 |
| **季节** | 春 | 夏 | 季夏 | 秋 | 冬 |
| **五间色** | 绿 | 红 | 流黄 | 碧 | 紫 |

> 正色为纯正之色，间色为混合之色。古人以正色为尊，间色次之。《礼记·玉藻》："衣正色，裳间色。"

### 1.2 色彩命名五源

| 来源 | 命名逻辑 | 典型色名 |
|------|---------|---------|
| **天象** | 天空、日月星辰、时辰变幻 | 天缥（晴空初解冻的浅青）、窃蓝（悄悄借来的一点蓝）、东方既白（黎明蓝朦透白）、暮山紫（傍晚山岚的紫霭）、月白（月光之淡蓝） |
| **自然风物** | 花木果蔬、鸟兽虫鱼 | 桃夭（桃花烂漫的粉红）、石榴裙（石榴花般的艳红）、芰荷（荷花的粉紫）、棠梨（叶枯后的红褐）、庭芜绿（暮春庭院的深绿） |
| **诗词典故** | 文人笔墨、经典名句 | 海天霞（海上霞光）、千山翠（越窑青瓷）、藕丝秋半（藕丝白中透秋意）、苍烟落照（暮色苍茫）、青青（《诗经》"青青子衿"） |
| **器物工艺** | 礼器、陶瓷、织物、颜料 | 黄琮（礼祭玉器的大地色）、瓷秘（越窑秘色瓷）、胭脂水（趋近胭脂的釉色）、螺子黛（波斯进口画眉颜料，墨绿价比黄金） |
| **历史人事** | 帝王将相、名士佳人 | 太师青（蔡京的道袍青灰）、杨妃（杨贵妃的醉色）、魏红/魏紫（洛阳牡丹名品）、逍遥游（庄子哲思之色） |

## 三、384 色在 AI 提示词中的使用要诀

### 3.1 核心原则

1. **色名即画面**：每个传统色名本身就是一个微缩的场景。"暮山紫"承载了王勃的滕王阁、"东方既白"承载了苏轼的赤壁、"天河碧"承载了李煜的南唐。引用色名时，颜色+意境同时进入提示词
2. **物候锚定时空**：写"立夏·朱颜酡"两个词，就同时指定了季节（初夏）+ 时辰（傍晚）+ 情绪（微醺娇羞）+ 色相（暖橙粉）。同理，"白露·藕丝秋半" = 初秋 + 午后 + 温柔 + 藕白
3. **典故自动带入美学体系**："石榴裙"不仅是一种红，还带入了盛唐的视觉系统（丰腴、华丽、开放）。"太师青"带入北宋士大夫审美（清雅、克制、文人化）。选对颜色，就选对了整套视觉风格
4. **五正色构建色彩框架**：青主春/东/木 → 赤主夏/南/火 → 黄主季夏/中/土 → 白主秋/西/金 → 黑主冬/北/水。大场景用正色定基调，细节用间色点缀

### 3.2 实战技法

- **配色方案**：选取同一节气的"起承转合"四色作为画面主色板，天然和谐。例如立春组：黄白游(起) + 天缥(承) + 流黄(转) + 海天霞(合) = 早春的完整色谱
- **跨节气组合**：需要时间跨度感时，选取相邻节气的颜色渐变——"惊蛰桃夭 → 春分洛神珠 → 清明凝夜紫" 描绘春日渐深的色彩演进
- **色彩+材质叠加**：传统色名搭配材质词——"千山翠釉面""胭脂色丝绸""象牙白茶盏""藕丝秋半纱裙"
- **作为光线描述**："暮山紫色的夕阳""东方既白的天光""朱颜酡般的暖光洒在脸上"
