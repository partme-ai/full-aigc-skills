## 通用结构模板

所有示例遵循相同的信息层级结构：

```
[主体/产品视觉] + [背景/色调] + [主标题] + [副标题/标签] + [底部促销信息] + [装饰元素]
```

**标准布局**（从上到下）：
1. **趣味标签**（粉色/彩色色块）— 如"奖励型人格专属""发疯学大师"
2. **主标题**（粗体无衬线，上方中央）— 趣味文案
3. **副标题**（右上角）— "好东西代表：{产品名}"
4. **主体视觉**（画面中央）— 产品/玩偶插画，多有白色描边
5. **品牌标注**（下方）— "{拼音}" + "Good Things Conference"
6. **底部促销**（底部区域）— "抖音商城 618好东西大会" + 搜索框图标
7. **背景点缀** — 二维码、英文词、趣味标语

## 风格规律总结

### 色彩体系
| 产品类型 | 主色调 | 点缀色 |
|---------|--------|--------|
| 毛绒玩偶/服饰 | 绿色系 | 蓝色+黄色 |
| 清新可爱 | 蓝色系/浅蓝渐变 | 黄色+绿色 |
| 反差萌/解压 | 白色+彩色卡片 | 多彩 |
| 趣味玩乐 | 黄色系 | 绿色+粉色 |
| 文化祈福 | 蓝色系 | 红色/玫红 |
| 个性态度 | 紫色系 | 绿色+黄色 |

### 人物标签体系
- **奖励型人格** / 奖励型人格专属 — 满足感/犒劳自己类产品
- **发疯学大师** / 发疯学大师专属 — 解压/搞笑类产品
- **尝新乐子人** — 新奇/创意类产品
- **玄学爱好者** — 文化/祈福/好运类产品
- **脆皮省电人** — 轻松/懒人/治愈类产品
- **自信放光芒** — 态度/个性类产品

### 标语文案风格
- 谐音梗：蒜鸟都不涌易（算鸟都不容易）、「豆」你玩儿
- 网络梗：精神状态belike、接接接接接、always精神工伤
- 态度宣言：一般丑的我看不上、微笑是我的保护色
- 祝福/玄学：下一站上岸、信则有不信也得信
- 反差：好迷但好喜欢、对症下药乐到病除
