## 9. 推荐完整提示词模板

> 从即梦3.0官方推荐模板中提取的完整示例，可作为同类场景的参考框架。

### 电商/促销海报
```
广告海报设计，甜美清新风格，文字为"蜜桃果肉 乌龙茶香"位于画面中央，黑色动感字体。
右侧绿底白字"生财冰淇淋 新品来袭"。左上角品牌标识，右上角拼音。
主视觉为深棕色冰淇淋蛋筒，质感绵密，周围点缀粉嫩蜜桃。
左下角黑色数字"12"标注价格，下方产品描述"清香甜蜜 双重口味 搭配100%严选生牛乳"。
整体粉色主色调，暖光照射，浅景深虚化背景，水平构图，中景拍摄。
3D渲染，超高清质感，甜蜜氛围。
```

### 劳动节/田园主题
```
劳动节主题海报设计，田园清新风格，画面主体为绿意盎然的草地（高饱和度翠绿色），点缀红/橙/白色小花。
一条蜿蜒白色小径贯穿画面，两位劳动者行走其中。
右上方书法字体"不负耕耘"（行书风格，加引号修饰）。
整体采用水平线构图，晨光光线（45度侧光），浅景深虚化远景。
色彩浓郁鲜亮，暖色调为主（黄/橙/绿），搭配紫色形成补色对比。
中景拍摄范围，微仰视角增强劳动者形象。手绘水彩质感，笔触可见。
类似宫崎骏动漫风格的自然生机感。超高清渲染，8K分辨率。
```

### 电影海报/科幻
```
电影海报, 科幻片, 量子隧道贯穿的悬浮城市场景，建筑表面实时刷新数据流，
中央破碎玻璃字"未来之城"悬浮重组动画，底部像素风小字"2025年上映"投影于地面，
故障艺术风格，动态模糊处理，荧光绿与深空蓝撞色，硬表面建模细节，
异形建筑结构，赛博格美学元素。
```

### 品牌宣传/科技极简
```
广告海报设计，现代简约风格，蓝白渐变主色调，科技感与清新氛围。
用途：企业品牌宣传海报，适用于数字媒体及印刷。
```

### 节日/国潮
```
"中秋节"，拟人化的玉兔身着月白汉服手提荷花灯笼，青瓦飞檐古建筑群倒映在镜面湖泊，
漫天荷花灯海环绕金色满月，水面波纹扰动"月满人团圆"瘦金体投影，
古风仙侠插画，工笔重彩技法，绢本设色质感，星轨光效，琉璃盏反光，HDR渲染。
```

### 电影感/情绪人像
```
电影感,，纪录片，浅景深中景，老人蜷坐窗边泪珠悬垂未落，
雨水划过玻璃倒映枯树，青灰色调高对比，电影级光线。
```

### 国风/瓷器
```
中国风，绢本画质感，禅意新中式，三维青花瓷瓶表面展开山水长卷，
釉下钴蓝皴擦出层峦叠嶂，"千年传承"瘦金体悬浮于瓷胎开光处，
小字"非遗中心"隐于云头纹饰，绢本设色质感，散点透视布局，
釉面反光细节，矿物颜料颗粒感，虚实相生留白，宋代极简美学，青灰调性渲染。
```

### 珠宝摄影
```
珠宝设计，黄金戒指镶嵌椭圆形切割蓝宝石，戒臂雕刻古典藤蔓花纹，
微距摄影展现金属抛光纹理与宝石火彩，深蓝色渐变背景，
珠宝摄影，高光反射，焦散效应，景深控制，HDR渲染。
```
