
- **直接引用色号**：写 `RAL 5002 群青蓝`、`RAL 3000 火焰红` 指定精确工业色
- **材质+RAL**：写 `RAL 9005 墨黑色哑光金属外壳` 或 `RAL 1013 浅灰色混凝土墙面`
- **工业/建筑风格必备**：描述包豪斯、工业风、极简空间时使用 RAL 色名可精准控制色板
- **德国工业美学**：`RAL 7016 煤灰 + RAL 2004 纯橙` 的组合暗示包豪斯/德国功能主义配色

### 4. 莫兰迪色系 (Morandi Palette)

> 意大利画家乔治·莫兰迪的标志性用色——所有颜色都掺入了一层灰度，低饱和、柔和、高级。非常适合极简、侘寂、北欧和文艺风格。

`灰粉` · `灰绿` · `灰蓝` · `烟灰紫` · `燕麦色` · `豆沙色` · `雾霾蓝` · `干枯玫瑰色` · `奶茶色` · `灰米色` · `鼠尾草绿` · `烟熏紫` · `灰橘` · `灰杏` · `灰棕`

**莫兰迪配色要诀**：在任意颜色的描述前加"灰调""烟熏""雾感"等前缀，即可获得莫兰迪美感。例如"灰调珊瑚""雾感蓝""烟熏玫瑰"。

### 7. 专业配色方案 (Color Schemes)

| 方案 | 英文 | 色轮关系 |
|------|------|---------|
| 单色调 | Monochromatic | 单一色相的不同明度/饱和度变体 |
| 互补色 | Complementary | 色轮正对面（蓝-橙、紫-黄、红-绿） |
| 分裂互补 | Split-Complementary | 一色 + 对面两邻色，冲突弱于纯互补 |
| 类似色 | Analogous | 色轮相邻三色（蓝-青-绿），和谐统一 |
| 三色调 | Triadic | 色轮三等分（红-黄-蓝），均衡活泼 |
| 四色调 | Tetradic | 两组互补色，丰富而平衡 |
| 暖色调 | Warm Palette | 红、橙、黄为主 |
| 冷色调 | Cool Palette | 蓝、绿、紫为主 |
| 大地色 | Earth Tones | 棕、赭、橄榄、陶土等自然色 |
| 宝石色 | Jewel Tones | 深饱和高贵色（祖母绿、宝石蓝、紫水晶、红宝石） |
| 粉彩色 | Pastel | 柔和低饱和浅色，轻盈雅致 |
| 霓虹色 | Neon | 高饱和荧光感，冲击力强 |
| 灰度 | Grayscale | 纯黑-白-灰 |
| 棕褐色调 | Sepia | 暖棕复古，如旧照片 |
