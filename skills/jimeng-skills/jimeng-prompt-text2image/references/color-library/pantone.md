## 二、国际标准色

> 涵盖 Pantone、RAL 两大工业色标体系，以及莫兰迪高级灰调、国际通用专业色名、配色方案与电影色彩风格。

### 1. 国际标准色彩体系总览

| 体系 | 全称 / 来源 | 特点 | 适用领域 |
|------|-----------|------|---------|
| Pantone | Pantone Matching System (PMS) | 国际通用专色标准，数千种编号色，每年发布年度色 | 品牌设计、时尚、印刷 |
| RAL | Reichsausschuß für Lieferbedingungen（劳尔色卡） | 德国工业色标：RAL Classic 213 色 + RAL Design 1688 色 | 工业涂料、建筑、冶金、产品设计 |
| Munsell | Munsell Color System | 基于色相(Hue)/明度(Value)/彩度(Chroma)三维科学体系 | 科学、教育、土壤学 |
| NCS | Natural Color System | 基于黑度/白度/彩度/色相四维，北欧标准 | 建筑、室内、工业设计 |
| Morandi | 乔治·莫兰迪 (Giorgio Morandi) | 低饱和高级灰调，柔和内敛 | 极简设计、侘寂、北欧风格 |
