## 一、3.0 模型提示词秘籍

### 1.1 核心原则

- **简短精准 > 冗长啰嗦**：用简单直接的语言表达，短提示词也能发挥惊艳效果。自己写的关键词可能比 AI 生成的更有效
- **自然语言 + 短词语组合**：主体/行为/环境用连贯自然语言描述，美学效果（风格/色彩/光影/构图）用短词语描述
- **文字用引号包裹**：把想要生成的文字内容插入 `"引号"` 中可显著提升文字准确率
- **专业词汇用原文/英文**：专业术语推荐使用英文或原文，效果更准确
- **写明用途与类型**：推荐注明图像用途（如 `PPT封面背景图`、`背景素材图`）和类型（如 `广告海报设计`、`纪实摄影`）

### 1.2 3.0 模型能力边界

| 能力 | 说明 |
|------|------|
| **文字** | 大字小字都能响应；支持多种字体、字重；图文排版专业 |
| **影像** | 电影类型片、镜头语言响应更准确 |
| **动漫** | 风格更多元、细节更丰富、色调更统一，无"抠图感" |
| **分辨率** | 默认 1k，可支持 2K（2K 时画面结构更准确） |
| **真实感** | 质感真实（去磨皮感/假脸感/AI油腻感）、情绪真实（去僵硬感/摆拍感） |

## 二、文字设计（3.0 专属能力）

> 3.0 模型在文字生成上大幅超越 2.1，不仅支持标题大字，小字也能响应，且解锁了更多艺术字体和字重区分。

```
细体         — 纤细轻盈，适合文艺/清新风格

特粗体       — 强烈视觉冲击，适合大字报/海报主标题
```
