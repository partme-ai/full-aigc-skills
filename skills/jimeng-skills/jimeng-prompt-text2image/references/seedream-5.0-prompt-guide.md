# Seedream 5.0 Lite 用户手册

> 来源: 即梦官方飞书知识库 | 最后更新: Feb 12, 2026

## 模型亮点

| 亮点 | 说明 |
|------|------|
| **1. 联网实时检索** 🌐 | 首次支持检索生图功能，与热门资讯话题深度绑定。满足强时效知识、名人识别、海外本土效果等场景。由模型自行判断是否需要检索，时效词或极长尾的词可高概率激发 |
| **2. 编辑精准可控** 🎯 | 编辑指令响应更精准 |
| **3. 智能逻辑推理** 🧠 | 逻辑推理能力增强，理解复杂需求 |

**额外优势：** 2K、4K 清晰度即梦首发并限免

> ⚠️ Preview 版真实感/美感存在一定效果劣化，相关生成需求可使用 4.5

## 各版本使用建议

| 版本 | 推荐场景 |
|------|----------|
| **5.0 Lite** | 需要**联网检索**（时效性内容、名人、海外本土化）、需要**逻辑推理**、需要**2K/4K高清** |
| **4.5** | 需要**准确自然视觉效果**，preview版效果不如4.5的场景 |
| **4.0** | **极致耗时与性价比**，快速生成场景 |
| **3.1** | **卓越美感与风格化**，追求艺术品质 |
| **3.0** | **文字生成与图文排版**，海报文字场景 |

## 联网检索功能详解

首次支持检索生图功能，核心特性：

- **强时效性**：创作与热门资讯话题深度绑定
- **适用场景**：强时效知识、名人识别、海外本土效果
- **触发机制**：由模型自行判断是否需要检索，时效词或极长尾的词可高概率激发
- **使用示例**：
  - "参考'多邻国'App界面，设计背单词打卡页，有单词和天数，用那个绿色鸟元素"
  - "生成北欧冬奥海报：挪威极光背景，滑雪运动员身穿国家服，要求有冬奥元素和吉祥物"

## 提示词写作建议（5.0 特有）

1. **利用联网检索**：提示词中可包含具体品牌、名人、时事话题、海外文化元素，模型会自动检索补充知识
2. **高清输出**：直接指定 2K/4K 输出，获得超高清画质
3. **逻辑推理**：可写需要逻辑关系的提示词（如因果关系、空间关系、时间顺序）
4. **编辑精准**：编辑类提示词可以直接用自然语言精确描述修改内容
5. **预览版限制**：如果效果不理想，可降级到 4.5 模型重试
