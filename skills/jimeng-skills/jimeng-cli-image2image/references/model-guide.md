# Model Selection Guide — I2I 模型选择指南

> 图生图编辑的模型版本选择。I2I 要求模型 ≥ 4.0。

---

## 快速选择

```
I2I 编辑场景 → 选模型：
├── 风格迁移（照片→油画/水墨/二次元）→ 5.0 或 4.5
├── 背景替换 → 5.0（综合最强）
├── 细节增强 → 4.6 或 5.0
├── 色彩调整 → 5.0（默认）
├── 人物编辑/换装 → 4.6
├── 多图合成 → 5.0
└── 快速测试 → 4.0（稳定基线）
```

## I2I 模型限制

| 模型 | I2I 支持 | 分辨率 | 说明 |
|------|---------|--------|------|
| 5.0 | ✅ | 2k, 4k | 默认推荐 |
| 4.6 | ✅ | 2k, 4k | 人像/产品强 |
| 4.5 | ✅ | 2k, 4k | 艺术风格强 |
| 4.1 | ✅ | 2k, 4k | 稳定基线 |
| 4.0 | ✅ | 2k, 4k | 最低要求 |
| 3.1 | ❌ | — | 不支持I2I |
| 3.0 | ❌ | — | 不支持I2I |

---

## 分辨率-场景建议

| I2I 场景 | 推荐分辨率 | 理由 |
|----------|-----------|------|
| 风格迁移 | 2k（迭代）/ 4k（最终） | 先用2k测试效果 |
| 背景替换 | 4k | 边缘处理需要高分辨率 |
| 细节增强 | 4k | 细节本身就要求高分辨率 |
| 色彩调整 | 2k | 纯色调变化不需要高分辨率 |
| 多图合成 | 4k | 复杂合成需要高精度 |
