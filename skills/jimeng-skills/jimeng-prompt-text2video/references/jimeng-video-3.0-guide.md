# 即梦视频3.0 & 3.0 Pro 官方提示词创作指南

> 来源: 即梦官方飞书知识库「即梦视频模型创作指南」
> 模型: 即梦视频3.0 & 3.0 Pro | 更新日: Mar 13

## 模型概述

即梦视频3.0 Pro 具有强大的语义遵循能力，写好 prompt 可以生成丰富多样的视频。
即梦视频Prompt写作的核心是**"直观表达出你想要的效果"**，可以使用自然语言自由书写出动作/运镜/风格/视角，从而有效控制视频效果。

> 💡 如果遇见响应不好的情况，可尝试将 prompt 改得更加直白易懂。

## Prompt 构建公式

### 文生视频 (T2V)
```
主体 + 动作 + 场景 + 镜头 + 风格 + (情绪演绎) + (照明)
```

### 图生视频 (I2V / I+T2V)
```
动作 + 镜头 + (情绪/照明)
```

### 核心思路

- **文生视频**: 提示词写得越明确、细节越多，画面还原度就越高。可以指定主体、动作、运镜、光照、情绪等各类信息
- **简单场景探索**: 即使没有具体画面构想，也可以给出一个简单的场景描述，让模型自动补充内容，快速探索创意方向
- **图生视频**: 更多依赖图像本身的信息，提示词主要用来引导人物动作、镜头语言或情绪变化。虽然风格大致由原图决定，但仍然可以控制角色移动、镜头推进或光照变化等动态效果

## 8 大控制维度详解

### 1. 动作指令

#### 1.1 单一动作（简单动作）
单个明确动作的描述，适合短时（4-6s）视频。

示例：挥手、转身、跳跃、奔跑、微笑

#### 1.2 时序组合动作（复杂动作）
按时间序列描述的多个连贯动作，适合较长（7-15s）视频。

写法：`[动作A] → [动作B] → [动作C]`

### 2. 情绪演绎
描述人物或场景的情绪状态，影响整体氛围。

示例：忧郁地望向远方、兴奋地手舞足蹈、安静地思考

### 3. 场景描述
明确的环境、时间、天气描述。

示例：黄昏时分的老街、雨后的森林、阳光明媚的海滩

### 4. 风格描述

#### 4.1 风格直出 (T2V)
直接通过文生视频生成特定风格。

#### 4.2 风格延续 (I+T2V / I2V)
基于参考图延续风格生成视频。

### 5. 镜头语言

#### 5.1 基础运镜
- **推 (Push in)**: 镜头向前推进，聚焦主体
- **拉 (Pull out)**: 镜头向后拉远，展示环境
- **摇 (Pan)**: 镜头左右转动
- **移 (Track/Dolly)**: 镜头横向移动
- **跟 (Follow)**: 跟随主体移动
- **升/降 (Boom up/down)**: 镜头垂直升降

#### 5.2 进阶运镜
- 环绕 (Orbit)
- 穿梭 (Fly-through)
- 一镜到底 (One-take)

#### 5.3 景别和视角控制
- **景别**: 远景/全景/中景/近景/特写
- **视角**: 平视/仰视/俯视/主观视角/过肩镜头

### 6. Prompt 控制美感
- **人物外形**: 外貌、服饰、身材等视觉特征描述
- **画面美感**: 色调、光影、构图等美学控制

### 7. 切镜能力
镜头切换和转场控制。

### 8. 创意特效

#### 8.1 首尾帧类型模块（基础分类）
基于首尾帧的不同类型特效。

#### 8.2 首尾帧创意玩法模块（玩法应用）
首尾帧的创意应用方式。

#### 8.3 I+T2V 创意特效模块（扩展）
文生图+图生视频组合的创意特效。

## 提示词构建总结表

| 模式 | 公式 | 关键要素 |
|------|------|----------|
| 文生视频 | 主体 + 动作 + 场景 + 镜头 + 风格 + 情绪 + 照明 | 所有要素都可自由组合 |
| 图生视频 | 动作 + 镜头 + (情绪/照明) | 图像提供主体和场景 |

> 本指南基于即梦官方视频模型 3.0 & 3.0 Pro。不同视频模型版本（Seedance 1.5/2.0 等）的提示词风格可能有所不同，请结合 `jimeng-prompt-text2video` 主技能的相机词库和示例文件使用。
