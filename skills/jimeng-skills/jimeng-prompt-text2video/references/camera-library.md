# Camera Movement Library — 运镜大全

> 运镜是视频提示词区别于图片提示词的核心要素。即梦 Seedance 2.0 对中文运镜术语的识别优于英文，**务必使用中文术语**。

---

## 一、七种基本运镜

### 1. 推 (Dolly In / Push In)

| 属性 | 内容 |
|------|------|
| **中文提示词** | 镜头缓缓推近、镜头向前推进、画面逐渐拉近、镜头推近至特写 |
| **视觉效果** | 画面从远景/中景逐渐聚焦到主体，背景范围缩小，主体在画面中变大 |
| **情感效果** | 引导注意力、制造紧张感、揭示细节、强调重要性 |
| **适用场景** | 人物表情特写、产品细节展示、剧情高潮、揭示关键信息 |

### 2. 拉 (Dolly Out / Pull Out)

| 属性 | 内容 |
|------|------|
| **中文提示词** | 镜头缓缓拉远、画面逐渐后退、镜头向后拉开、从特写拉至全景 |
| **视觉效果** | 画面从主体特写逐渐扩大到全景，展现主体所在的环境全貌 |
| **情感效果** | 揭示全貌、制造孤独感、结束感、让主体显得渺小 |
| **适用场景** | 故事结尾、展示环境规模、人物孤寂感、空间揭示 |

### 3. 摇 (Pan)

| 属性 | 内容 |
|------|------|
| **中文提示词** | 镜头水平横摇、从左向右摇镜、镜头缓缓扫过、环视四周 |
| **视觉效果** | 相机位置不动，镜头水平旋转，像转头一样扫视场景 |
| **情感效果** | 探索感、跟随视线、连接空间关系、叙事过渡 |
| **适用场景** | 风景展示、室内环视、跟随移动物体、空间关系交代 |

### 4. 移 (Truck / Track)

| 属性 | 内容 |
|------|------|
| **中文提示词** | 镜头平移跟随、镜头横移、侧面跟拍、平移展示 |
| **视觉效果** | 相机沿水平方向移动，保持与被摄主体的相对位置 |
| **情感效果** | 陪伴感、沉浸感、空间穿越感 |
| **适用场景** | 人物行走跟拍、街道巡游、展示空间宽度 |

### 5. 升 (Pedestal Up / Boom Up)

| 属性 | 内容 |
|------|------|
| **中文提示词** | 镜头缓缓上升、镜头向上抬升、从低处升到高处 |
| **视觉效果** | 相机垂直上升，视野从低到高，地面逐渐远离 |
| **情感效果** | 壮丽感、升华感、权力感、希望升起 |
| **适用场景** | 建筑全貌、日出场景、人物站立起身、英雄登场 |

### 6. 降 (Pedestal Down / Boom Down)

| 属性 | 内容 |
|------|------|
| **中文提示词** | 镜头缓缓下降、镜头向下沉降、从高处降至低处 |
| **视觉效果** | 相机垂直下降，视野从高到低，地面逐渐接近 |
| **情感效果** | 压迫感、落地感、揭示低处细节、沉重 |
| **适用场景** | 人物坐下、落叶飘落、揭露地面细节、压抑氛围 |

### 7. 跟 (Tracking Shot / Follow)

| 属性 | 内容 |
|------|------|
| **中文提示词** | 镜头跟随主体移动、镜头紧跟、跟拍、追随镜头 |
| **视觉效果** | 相机与被摄主体同步移动，保持主体在画面中的位置不变 |
| **情感效果** | 代入感、参与感、不放弃的追随、沉浸式体验 |
| **适用场景** | 人物行走、奔跑、骑行、驾驶、任何移动主体的拍摄 |

## 二、复合/高级运镜

### 8. 推+升 (Dolly In + Rise)

`镜头边推进边缓缓抬起` — 从低角度逐渐接近主体同时升高，营造从仰视到平视的尊重感

### 9. 拉+摇 (Pull Out + Pan)

`镜头拉远的同时缓缓横摇` — 一边揭示全景一边扫视环境，常用于影片结尾

### 10. 跟+环绕 (Track + Orbit)

`镜头围绕主体环绕拍摄` — 以主体为圆心环绕移动，展示主体的立体感和气场

### 11. 360度环绕 (360° Orbit)

`镜头围绕主体360度环绕一周` — 完整的环绕视角，用于展示产品全貌或人物气场

### 12. 希区柯克变焦 (Dolly Zoom / Vertigo)

`镜头前进同时变焦拉远，背景透视急剧变化` — 主体大小不变但背景透视剧变，制造不安/眩晕感

### 13. 俯冲 (Crash Zoom)

`镜头快速俯冲向主体` — 极速接近，制造冲击感和戏剧性

## 三、运镜与情感映射

| 想传达的情感 | 推荐运镜 | 提示词示例 |
|-------------|---------|-----------|
| 紧张/悬疑 | 推 (Dolly In) | "镜头缓缓推进，逼近主角的面部" |
| 孤独/渺小 | 拉 (Dolly Out) | "镜头渐渐拉远，人物在茫茫人海中越变越小" |
| 壮丽/震撼 | 升 (Boom Up) | "镜头缓缓升起，展现出整座城市的壮阔" |
| 探索/发现 | 摇 (Pan) | "镜头从左向右缓缓扫过书架上的古籍" |
| 陪伴/参与 | 跟 (Tracking) | "镜头跟随主角穿过熙攘的菜市场" |
| 沉浸/穿越 | 移 (Truck) | "镜头横向平移，穿过一道道拱门" |
| 终结/离别 | 拉+升 (Pull Out + Rise) | "镜头缓缓拉远并升起，主角的身影消失在远方" |
| 压迫/不安 | 降 (Boom Down) | "镜头缓缓下降，天花板越来越低" |
| 气场/展示 | 环绕 (Orbit) | "镜头环绕主角缓缓旋转一周" |

## 四、运镜组合与使用要诀

### 推荐组合模式
- **单一运镜（最稳定）**: 仅用一种运镜，搭配场景和动作。成功率高，适合大多数场景
- **一主一次**: 一种主要运镜 + 一种轻微辅助（如"镜头缓缓推近，同时微微上摇"）。适合需要丰富画面感的场景
- **避免三合一**: 三种以上运镜叠加极易导致画面混乱，不推荐

### 运镜节奏描述
`缓慢/缓缓` — 抒情、沉思、优雅
`平稳/匀速` — 纪录片、客观叙述
`快速/急速` — 紧张、动作、冲击
`先慢后快` — 蓄力爆发
`先快后慢` — 高潮后归于平静

### 核心规则
1. **中文优先**："镜头缓缓推近" 优于 "dolly in slowly"
2. **放在提示词后半段**：主体+动作+场景 在前，运镜 在后
3. **Keep it simple**: 1 种运镜最安全，最多 2 种
4. **速度要写出来**：不是"推"，而是"缓缓推"或"快速推近"
