# Sports & Action — 运动/动作视频

## Instructions

运动视频的核心是**冻结高速动态中的力量瞬间**——扣篮的腾空、冲浪的穿越、冲刺的爆发。运动视频的动作描述要比其他类别更具体、更有爆发力，运镜多用跟拍来保持与运动员的同步感。

## Scenario Patterns

| 子场景 | 动态核心 | 推荐运镜 | 典型触发词 |
|--------|---------|---------|-----------|
| 篮球/足球 | 腾空扣篮+射门+对抗 | 跟/推 | 篮球, 足球, 比赛 |
| 跑步/田径 | 冲刺+步伐+汗水 | 跟/摇 | 跑步, 马拉松, 冲刺 |
| 冲浪/滑板 | 穿浪+滑行+跳跃 | 跟/摇 | 冲浪, 滑板, 极限 |
| 健身/举重 | 肌肉收缩+举重+汗水 | 推/固定 | 健身, 举重, 健美 |
| 瑜伽/柔韧 | 缓慢体式转换+呼吸 | 固定/推 | 瑜伽, 冥想, 柔韧 |

## Examples

### Example: 扣篮爆发

**用户需求**：写一个篮球运动员飞身扣篮的视频

**公式拆解**：
- 主体：篮球运动员 + 球衣
- 动作：运球冲刺 → 起跳腾空 → 单手抓球暴扣
- 场景：篮球场 + 观众沸腾
- 运镜：镜头跟随运动员从运球到起跳
- 光影：体育馆顶光照亮肌肉线条
- 时长：5秒
- 风格：赛事转播

**最终提示词**：
```
篮球运动员从三分线外运球冲刺，大步跨入罚球线后猛地起跳腾空，身体在空中舒展，
单手抓球高高举起砸向篮筐，汗水在空中飞溅，肌肉线条在体育馆顶光下紧绷清晰，
镜头从侧面跟拍运动员从冲刺到起跳扣篮的全过程，5秒，赛事转播风格，高速动态
```

### Example: 冲浪穿管

**用户需求**：写一个冲浪者在巨大卷曲海浪中穿行的视频

**公式拆解**：
- 主体：冲浪者 + 站在冲浪板上
- 动作：压低身体在卷曲的浪管中穿行 + 手轻触浪壁
- 场景：翡翠绿色巨浪 + 水花飞溅
- 运镜：镜头从浪管外跟拍至浪管内
- 时长：6秒
- 风格：极限运动

**最终提示词**：
```
冲浪者压低身体站在冲浪板上在巨大的卷曲海浪中穿行，翡翠绿色的浪壁在头顶卷成管状，
他伸出手轻触浪壁保持平衡，白色水花在浪尖飞溅，阳光穿透海水呈现半透明的绿色层次，
镜头从浪管侧面跟拍，6秒，极限运动摄影风格
```

### Example: 瑜伽晨光

**用户需求**：写一个海边日出的瑜伽视频

**公式拆解**：
- 主体：女性瑜伽者 + 白色瑜伽服
- 动作：缓慢流畅地从山式过渡到树式 + 双手合十举过头顶
- 场景：海边岩石 + 日出
- 运镜：镜头缓缓环绕（极慢）
- 时长：10秒
- 风格：治愈慢生活

**最终提示词**：
```
女性瑜伽者在海边岩石上迎着日出做瑜伽，她缓慢流畅地从山式过渡到树式体式，
双手合十缓缓举过头顶，白色瑜伽服在海风中轻轻飘动，初升的太阳金色光芒逆光
勾勒出她的身体轮廓，远处海面波光粼粼，镜头极慢地环绕拍摄，10秒，治愈慢生活风格
```

## Category-Specific Tips

1. **运动视频的关键词是"爆发"** — 用"猛地""瞬间""腾空""爆裂"等有力动词
2. **汗水/水花/灰尘=动态增强器** — 这些飞溅的微粒让运动动作更有冲击力
3. **跟拍=第一人称参与感** — 运动视频中跟拍的效果远好于固定机位
4. **动作链条要完整** — "运球→冲刺→起跳→扣篮" 比单独的"扣篮"更有叙事性
5. **短时长=强爆发** — 5-6秒是运动视频的最佳时长，太长反而削弱冲击力
