# Portrait & People — 人物/人像视频

## Instructions

人物视频的核心是**动作自然 + 运镜叙事**。区别于人物图片，视频需要描述具体的动作过程——不只是"站着"而是"缓缓转身、回眸一笑"。运镜选择决定了观众与人物的情感距离。

## Scenario Patterns

| 子场景 | 动作侧重 | 推荐运镜 | 典型触发词 |
|--------|---------|---------|-----------|
| 日常/生活感 | 自然动作（行走/看书/喝咖啡） | 跟/摇 | 日常, vlog, 生活 |
| 情绪/氛围 | 微表情+缓慢动作 | 推/拉 | 情绪, 孤独, 思念 |
| 时尚/穿搭 | 走秀/展示/转身 | 跟/环绕 | 穿搭, 展示, 模特 |
| 运动/户外 | 跑步/跳跃/伸展 | 跟/摇 | 运动, 户外, 活力 |
| 回眸/瞬间 | 转身+回眸（经典镜头） | 推 | 回眸, 转身, 一眼万年 |

## Examples

### Example: 森林漫步

**用户需求**：写一个女孩在森林里散步的视频，要有阳光和自然感

**公式拆解**：
- 主体：年轻女孩 + 白色长裙 + 长发
- 动作：缓缓行走 + 伸手轻触路边野花 + 裙摆随步伐摇曳
- 场景：清晨森林 + 阳光穿透树冠形成光束
- 运镜：镜头侧面平移跟随
- 光影：阳光透过树叶形成斑驳光影，在她身上流动
- 时长：8秒
- 风格：清新治愈、电影级画质

**最终提示词**：
```
年轻女孩穿着白色长裙在清晨的森林中缓缓行走，长发和裙摆随步伐轻轻摇曳，
她伸手轻触路边的一朵野花，阳光透过茂密的树冠形成丁达尔光束洒在她身上，
光影在她身上流动变幻，镜头侧面平移跟随，8秒，清新治愈风格，电影级画质
```

### Example: 回眸一笑

**用户需求**：写一个经典的回眸视频，要有电影感

**公式拆解**：
- 主体：女性背影 + 风衣 + 长发
- 动作：背对镜头缓步前行 → 听到呼唤 → 转身回眸 → 嫣然一笑
- 场景：秋日黄昏街道 + 梧桐落叶
- 运镜：镜头缓缓推近（推）
- 光影：金色夕阳光逆光，发丝勾勒出光晕
- 时长：6秒
- 风格：电影感

**最终提示词**：
```
穿卡其色风衣的女性背对镜头在秋日黄昏的街道上缓步前行，梧桐叶飘落，
她仿佛听到呼唤般缓缓转身回眸，嫣然一笑，金色夕阳逆光在她发丝上勾勒出温暖的光晕，
镜头缓缓推近她的面庞，6秒，电影感，温暖胶片色调
```

### Example: 晨跑

**用户需求**：写一个晨跑的视频，要有活力和城市感

**公式拆解**：
- 主体：年轻男性跑者 + 运动服
- 动作：匀速奔跑 + 呼吸有节奏 + 汗水滑落
- 场景：清晨城市河边跑道 + 旭日初升
- 运镜：镜头前方跟拍（跟）
- 光影：金色晨光从侧面照亮跑者
- 时长：6秒
- 风格：运动纪实

**最终提示词**：
```
年轻男性跑者穿着深蓝色运动服在清晨城市的河边跑道上匀速奔跑，呼吸有节奏，
汗水从额头滑落，旭日初升的金色晨光从侧面照亮他的面庞和身体轮廓，
镜头在前方跟拍，6秒，运动纪实风格，充满活力的清晨氛围
```

## Category-Specific Tips

1. **人物动作要写"过程"而非"结果"** — "转身"是一帧，"缓缓转身"是一个过程
2. **发丝/衣物的飘动是"天然的动作指示器"** — 即使主体不动，风吹动发丝和衣物也能让画面有生命
3. **跟拍最常用** — 人物视频中，"镜头跟随"是最安全也最自然的运镜
4. **回眸是经典公式** — 背影→转身→回眸微笑，这个动作链在视频中效果出奇好
5. **情绪用推镜** — 想表现人物内心时，镜头缓缓推进面部是电影百年验证的手法
