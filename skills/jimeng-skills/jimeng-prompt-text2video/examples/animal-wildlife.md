# Animal & Wildlife — 动物视频

## Instructions

动物视频的核心是**捕捉动物的标志性动态**——鸟类的飞翔、猎豹的奔跑、水母的漂浮。关键是写出该物种最具辨识度的运动方式，配以适合的运镜。

## Scenario Patterns

| 子场景 | 动态核心 | 推荐运镜 | 典型触发词 |
|--------|---------|---------|-----------|
| 鸟类飞翔 | 展翅+翱翔+俯冲+鸟群 | 跟/摇 | 鸟, 飞翔, 鹰, 鸽群 |
| 陆地奔跑 | 奔跑+跳跃+肌肉运动 | 跟/摇 | 奔跑, 猎豹, 马, 鹿 |
| 水中游动 | 摆尾+穿梭+鱼群同步 | 跟/推 | 鱼, 鲸, 海豚 |
| 宠物日常 | 玩耍+摇尾+舔舐 | 固定/推 | 猫, 狗, 宠物 |
| 昆虫微距 | 振翅+爬行+采蜜 | 推/微距 | 蝴蝶, 蜜蜂, 蜂鸟 |

## Examples

### Example: 鹰击长空

**用户需求**：写一只雄鹰在雪山之上翱翔的视频

**公式拆解**：
- 主体：金雕（雄鹰）+ 展开的巨大翅膀
- 动作：在天空中盘旋翱翔 + 翅膀微调利用气流
- 场景：雪山之巅 + 蓝天 + 云层
- 运镜：镜头仰拍跟摇，跟随鹰的飞行轨迹
- 光影：阳光穿透翅膀边缘
- 时长：8秒
- 风格：野生动物纪录片

**最终提示词**：
```
一只金雕在雪山之巅的蓝天中盘旋翱翔，巨大的翅膀完全展开，翼尖的羽毛在气流中
微微上翘，阳光穿透翅膀边缘形成半透明的金色光晕，背景是白雪覆盖的山峰和翻滚的云海，
镜头仰拍跟摇追随鹰的飞行轨迹，8秒，野生动物纪录片风格
```

### Example: 骏马奔跑

**用户需求**：写一匹黑马在草原上奔跑的视频

**公式拆解**：
- 主体：黑色骏马 + 鬃毛飞扬
- 动作：四蹄腾空奔跑 + 鬃毛和尾巴在风中飘动
- 场景：金色草原 + 夕阳
- 运镜：镜头侧面平移跟拍
- 时长：6秒
- 风格：自然纪实

**最终提示词**：
```
一匹黑色骏马在金色草原上全力奔跑，鬃毛和尾巴在风中飞扬，四蹄有力地踏击地面
扬起尘土，夕阳将马匹的轮廓染成金色，镜头侧面平移跟拍保持与马匹同步，
6秒，自然纪实风格，自由奔放的力量感
```

### Example: 水母漂浮

**用户需求**：写一个深海发光水母的视频

**公式拆解**：
- 主体：发光水母 + 透明伞状体+细长触手
- 动作：一张一合地收缩前进 + 触手如丝带飘舞 + 发出生物荧光
- 场景：幽暗深海
- 运镜：镜头缓缓推进
- 时长：8秒
- 风格：水下自然纪录片

**最终提示词**：
```
一只发光的深海水母在幽暗的海水中一张一合地收缩前进，透明的伞状体边缘发出
柔和的蓝色生物荧光，数十条细长的触手如发光丝带般在身后飘舞，光点在深蓝的
背景中如星空般闪烁，镜头缓缓推近水母展示其精致的内部结构，8秒，水下纪录片风格
```

## Category-Specific Tips

1. **每个物种有独特的"运动签名"** — 鹰的盘旋、马的腾跃、水母的脉动，抓住这个标志性动作
2. **速度决定运镜** — 快速移动的动物用跟拍，慢速浮动用推镜
3. **羽毛/毛发/鳞片的飘动是视频独有的生命力** — 利用这些细节让画面"活"起来
4. **鸟群/鱼群有集体运动的韵律** — "鸟群同时转向""鱼群同步穿梭"，群体运动是绝佳视频素材
