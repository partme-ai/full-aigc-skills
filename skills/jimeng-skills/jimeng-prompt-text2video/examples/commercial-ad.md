# Commercial & Ad — 商业/广告视频

## Instructions

商业视频的核心是**产品展示 + 品牌调性**。视频需要在有限时长内清晰展示产品并传达品牌气质。运镜多为推进+环绕，突出产品细节和质感。

## Scenario Patterns

| 子场景 | 动态核心 | 推荐运镜 | 典型触发词 |
|--------|---------|---------|-----------|
| 产品展示 | 360度旋转展示+光影流转 | 环绕/推 | 产品, 展示, 开箱 |
| 品牌大片 | 模特+产品+场景氛围 | 推/升 | 品牌, 广告, 大片 |
| 宣传/推广 | 文字+场景+行动号召 | 多种 | 宣传, 推广, 促销 |
| 开箱/unboxing | 打开包装+取出产品+特写 | 推/固定 | 开箱, 拆箱 |

## Examples

### Example: 手表广告

**用户需求**：写一个高端手表的产品展示视频

**公式拆解**：
- 主体：瑞士机械表 + 精钢表壳+皮质表带
- 动作：手表在光线下缓缓旋转 + 秒针平滑走动 + 表盘玻璃反射光芒
- 场景：纯黑背景 + 渐变柔光
- 运镜：镜头环绕手表旋转+缓缓推近
- 时长：8秒
- 风格：奢侈品牌广告

**最终提示词**：
```
一只瑞士机械表在纯黑背景中缓缓旋转，精钢表壳在渐变的柔光下呈现出优雅的金属光泽，
蓝宝石玻璃表盘反射着迷人的蓝光，秒针平滑无声地走动着，皮质表带的缝线纹理
在光线下清晰可见，镜头环绕手表缓缓旋转并逐渐推近表盘，8秒，奢侈品牌广告质感
```

### Example: 香水大片

**用户需求**：写一个高端香水广告的品牌大片视频

**公式拆解**：
- 主体：琥珀色香水瓶 + 金色液体
- 动作：香水瓶从暗处缓缓升起 + 光线在瓶身切割面上流动 + 香水喷雾散开
- 场景：暗调空间 + 岩石台座
- 运镜：镜头从瓶底缓缓上升到瓶顶（升）
- 时长：8秒
- 风格：高奢品牌广告

**最终提示词**：
```
琥珀色香水瓶从暗调空间的岩石台座上缓缓升起，金色液体在瓶中微微晃动，
一束柔光从上方打在瓶身上，在切割面的棱角间流动出金色和琥珀色的光影层次，
细密的香水喷雾在空中散开形成朦胧的雾状光晕，镜头从瓶底缓缓上升到瓶顶，
8秒，高奢品牌广告质感
```

### Example: 汽车广告

**用户需求**：写一个新能源跑车的外观展示视频

**公式拆解**：
- 主体：银色流线型跑车
- 动作：跑车从静态中缓缓启动 + 车身线条在光影中流转
- 场景：未来感影棚 + 黑色湿地面
- 运镜：镜头环绕车身一圈
- 时长：8秒
- 风格：汽车广告

**最终提示词**：
```
银色流线型新能源跑车停放在未来感影棚中，黑色湿地面上倒映出车身的完美镜像，
柔和的渐变光在流畅的车身曲线上缓缓流动展现出每一处精密的线条，轮毂开始
缓缓旋转，镜头环绕车身一周展示360度全貌，8秒，汽车广告质感
```

## Category-Specific Tips

1. **产品旋转+环绕运镜=经典公式** — 这是产品展示最安全也最有效的组合
2. **光线在产品上的流动是视觉核心** — "光影在曲面上的流转"远胜于"产品被光照亮"
3. **简约背景+精密光影** — 商业视频的背景要干净到近乎不存在，让产品成为唯一主角
4. **液体/喷雾/粉末=产品视频的动态加分项** — 香水喷雾、化妆品粉末飞扬、饮料注入冰块——这些动态让产品"活"起来
