# Urban & City — 城市/人文视频

## Instructions

城市视频的核心是**人流、车流、光影流动**——城市本身就是一座永不停歇的运动机器。延时摄影是城市视频的经典手法，但即梦更适合中短时长的城市切片。

## Scenario Patterns

| 子场景 | 动态核心 | 推荐运镜 | 典型触发词 |
|--------|---------|---------|-----------|
| 街景/人流 | 行人穿梭+车流+霓虹 | 固定/摇 | 街景, 城市, 熙攘 |
| 市场/集市 | 买卖互动+热气+烟火气 | 跟/摇 | 菜市场, 夜市, 集市 |
| 地铁/通勤 | 列车进站+人流上下 | 跟/推 | 地铁, 通勤, 上班 |
| 雨夜都市 | 霓虹倒影+雨滴+伞流 | 固定/摇 | 雨夜, 霓虹, 都市夜 |
| 老城/胡同 | 慢生活+老人+猫+光影 | 摇/移 | 老城, 胡同, 弄堂 |
| 城市天台 | 天际线+日出日落+云 | 升/摇 | 天台, 天际线, 俯瞰 |

## Examples

### Example: 雨夜霓虹

**用户需求**：写一个雨夜东京街头霓虹灯闪烁的视频

**公式拆解**：
- 场景：东京涩谷十字路口 + 霓虹灯招牌
- 动态：密密麻麻的行人撑伞过马路 + 雨滴落在路面激起水花 + 霓虹灯在湿润地面上倒影流动
- 运镜：镜头固定或微微上摇
- 光影：霓虹冷暖色交织 + 车灯流动
- 时长：8秒
- 风格：赛博朋克电影感

**最终提示词**：
```
雨夜的东京涩谷十字路口，霓虹灯招牌闪烁着蓝色、粉色和橙色的光，密密麻麻的行人
撑着透明雨伞匆匆穿过斑马线，雨滴不断落在湿润的柏油路面上激起细小的水花，
霓虹灯的倒影在地面水洼中流动变色，远处有车辆驶过留下红色的尾灯光轨，
镜头固定微微上摇展现霓虹灯海的璀璨，8秒，赛博朋克电影感
```

### Example: 清晨菜市场

**用户需求**：写一个清晨菜市场热闹开市的视频

**公式拆解**：
- 场景：清晨菜市场 + 摊位整齐排列
- 动态：摊贩整理蔬果+蒸笼冒白汽+顾客挑选+讨价还价的手势
- 运镜：镜头平移穿过市场走道
- 光影：清晨暖黄灯光+从顶棚透入的晨光
- 时长：10秒
- 风格：烟火气纪实

**最终提示词**：
```
清晨的菜市场中，摊贩们正在将新鲜的蔬菜水果整齐码放在摊位上，蒸笼冒着白色蒸汽，
暖黄的灯泡串横跨狭窄的走道，早起的顾客在摊位前挑选蔬菜和讨价还价，
镜头平移穿过熙攘的市场走道，顶棚缝隙透入的晨光与灯光交织，10秒，烟火气纪实风格
```

### Example: 城市天台日落

**用户需求**：写一个城市天台看日落的视频

**公式拆解**：
- 场景：城市天台 + 天际线建筑群全景
- 动态：夕阳缓缓沉入地平线 + 城市灯光渐次亮起 + 晚风吹动天台上的晾晒衣物
- 运镜：镜头缓缓上升
- 时长：10秒
- 风格：都市治愈

**最终提示词**：
```
城市天台上的日落时分，橘红色的夕阳缓缓沉入远处的天际线建筑群中，天空从橙红
渐变到深蓝，城市的万家灯火渐次亮起如星河落地，晚风吹动天台上晾晒的白色床单轻轻飘动，
镜头缓缓上升展现从金色黄昏到蓝色夜晚的完整过渡，10秒，都市治愈风格
```

## Category-Specific Tips

1. **城市的主体是人流** — "人群"本身就是最好的运动元素
2. **灯光变化是城市的脉动** — "霓虹灯渐次亮起""路灯一排排亮起"，灯光从暗到亮是天然的视频叙事
3. **雨天的城市自带美感** — 湿润地面=天然反射板，霓虹+雨滴=赛博朋克标配
4. **延时压缩时间感** — 即使不是真正的延时摄影，用"快速流动的车流""如潮水般的人群"也能传达时间压缩感
