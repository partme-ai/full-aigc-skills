# Abstract & Creative — 抽象/创意视频

## Instructions

创意视频的核心是**视觉上的"不可能"在视频中变为可能**——水墨在虚空中晕染、粒子组成形体又消散、色彩在空间中流动。这类视频最自由，但需要用具体的视觉隐喻来承载抽象概念。

## Scenario Patterns

| 子场景 | 动态核心 | 推荐运镜 | 典型触发词 |
|--------|---------|---------|-----------|
| 水墨动画 | 墨色晕染+笔触流动+留白 | 拉/摇 | 水墨, 晕染, 国风动画 |
| 粒子/碎片 | 粒子聚合+散开+形成形体 | 推/环绕 | 粒子, 消散, 聚合 |
| 色彩流动 | 色彩在空间流淌+混合 | 跟/摇 | 流动, 色彩, 抽象 |
| 生长/绽放 | 植物快速生长+花开延时 | 推/升 | 生长, 绽放, 花开花落 |
| 几何变换 | 几何形状旋转+变形+重构 | 环绕/推 | 几何, 变幻, 抽象 |

## Examples

### Example: 水墨梅花

**用户需求**：写一个水墨风格的梅花在宣纸上晕染盛开的视频

**公式拆解**：
- 主体/场景：宣纸上的水墨梅花
- 动作：墨色从笔触中向外晕染扩散 + 花瓣由浓到淡层层展开 + 墨韵在纸面流动
- 风格：水墨动画
- 运镜：镜头缓缓拉远展现整体画面
- 时长：8秒
- 风格：国风水墨动画

**最终提示词**：
```
宣纸上水墨绘就的梅花，浓墨从枝干的笔触中向外晕染扩散，淡墨的花瓣由浓到淡
层层绽放，墨韵在粗糙的宣纸纹理中自然流动和渗透，画面大面积留白，
镜头缓缓拉远展现梅枝的整体构图，8秒，国风水墨动画风格
```

### Example: 粒子人形

**用户需求**：写一个金色粒子汇聚成人形又散开的创意视频

**公式拆解**：
- 主体：成千上万的金色发光粒子
- 动作：粒子从四面八方汇聚 → 形成一个奔跑的人形 → 人形奔跑几步后粒子散开飘散
- 场景：深黑虚空
- 运镜：镜头环绕粒子运动
- 时长：8秒
- 风格：概念艺术

**最终提示词**：
```
成千上万的金色发光粒子在深黑的虚空中从四面八方汇聚，逐渐形成一个奔跑的人形轮廓，
人形奔跑几步后粒子突然散开如金色星尘般飘散在空间中，每个粒子都拖着微弱的金色光尾，
镜头环绕着粒子运动轨迹缓缓旋转，8秒，概念艺术动画风格
```

### Example: 花朵绽放延时

**用户需求**：写一朵牡丹花从含苞到盛开的延时视频

**公式拆解**：
- 主体：红色牡丹花 + 花苞
- 动作：花苞缓缓展开 → 花瓣一层一层向外绽放 → 完全盛开的牡丹
- 场景：晨光中的花园
- 运镜：镜头固定+缓缓推近
- 时长：10秒
- 风格：自然延时摄影

**最终提示词**：
```
深红色的牡丹花苞在晨光中缓缓展开，外层花瓣先张开，内层花瓣紧接着一层一层
向外绽放，花蕊从中心逐渐显露出来，晨光在花瓣上流转映出丝绸般的质感，
整个过程从含苞到完全盛开，镜头固定并缓缓推近花心，10秒，自然延时摄影风格
```

## Category-Specific Tips

1. **抽象需要具体的视觉锚点** — 即使是抽象视频，也要有一个可辨识的视觉核心（粒子、墨色、花朵）
2. **粒子/碎片=天然的视频素材** — 粒子聚合和散开是Seedance表现最好的抽象效果之一
3. **水墨动画需要"渗透感"** — 关键动词：晕染、渗透、流动、扩散
4. **生长/绽放类自带叙事弧线** — 从无到有、从小到大、从含苞到盛开——这是最自然的视频叙事
