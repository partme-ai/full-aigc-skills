# Food & Culinary — 美食/烹饪视频

## Instructions

美食视频的核心是**过程的诱惑**——食材的转变、蒸汽的升腾、酱汁的流淌。美食视频的"动"在于烹饪的动作（切、炒、翻、倒、搅拌）和食物的自然反应（冒泡、融化、膨胀、上色）。

## Scenario Patterns

| 子场景 | 动态核心 | 推荐运镜 | 典型触发词 |
|--------|---------|---------|-----------|
| 烹饪过程 | 翻炒+切菜+倒酱汁+火焰 | 固定/推 | 烹饪, 炒菜, 火焰 |
| 成品展示 | 热气升腾+酱汁流淌+切开展示 | 推/环绕 | 成品, 摆盘, 特写 |
| 饮品制作 | 倒咖啡+拉花+冰块落入+气泡 | 推/固定 | 咖啡, 调酒, 倒茶 |
| 烘焙/甜点 | 面团发酵+巧克力融化+糖粉撒落 | 推/拉 | 烘焙, 蛋糕, 甜点 |
| 食材特写 | 水珠+新鲜切面+纹理质感 | 推/微距 | 食材, 新鲜, 微距 |

## Examples

### Example: 中式爆炒

**用户需求**：写一个中式厨房爆炒的视频，要有锅气和火焰

**公式拆解**：
- 主体/动作：厨师颠勺 + 食材在锅中翻飞 + 火焰窜起
- 场景：中式厨房灶台 + 铁锅+旺火
- 动态：食材在热油中翻滚 + 火焰从锅边窜起 + 白色蒸汽升腾
- 运镜：镜头缓缓推进（推）
- 光影：炉火照亮厨师面部 + 油烟在灯光中形成光柱
- 时长：6秒
- 风格：美食纪录片

**最终提示词**：
```
中式厨房灶台上铁锅中的食材在旺火中翻飞，厨师有力地颠勺，橙色的火焰从锅边窜起，
白色蒸汽升腾弥漫，油光在食材表面闪烁，炉火的光芒照亮厨师专注的面庞，
镜头缓缓推进聚焦锅中翻滚的美食，6秒，美食纪录片风格，极致的锅气和烟火气
```

### Example: 手冲咖啡

**用户需求**：写一个手冲咖啡的慢节奏视频

**公式拆解**：
- 主体/动作：手冲壶 + 热水注入 + 咖啡粉膨胀冒泡
- 场景：温暖的日式咖啡馆吧台
- 动态：热水缓慢均匀地注入 + 咖啡粉膨胀释放气体 + 咖啡液一滴一滴滴入分享壶
- 运镜：镜头缓缓推近（推）
- 光影：侧窗柔和的自然光
- 时长：10秒
- 风格：慢生活治愈

**最终提示词**：
```
手冲壶的细长壶嘴中流出的热水均匀地浇在咖啡粉上，咖啡粉遇水膨胀形成饱满的咖啡床，
细小的气泡不断冒出，深褐色的咖啡液一滴一滴透过白色滤纸滴入玻璃分享壶中，
侧窗柔和的自然光照亮升腾的水蒸气，镜头缓缓推近滤杯，10秒，慢生活治愈风格
```

### Example: 蛋糕切开展示

**用户需求**：写一个巧克力熔岩蛋糕切开瞬间的视频

**公式拆解**：
- 主体：巧克力熔岩蛋糕 + 白色瓷盘
- 动作：银色餐刀切入蛋糕 + 巧克力熔岩缓缓流出
- 场景：简约餐桌 + 蜡烛+红酒
- 运镜：镜头俯拍推进（推）
- 时长：6秒
- 风格：美食广告

**最终提示词**：
```
银色餐刀缓缓切入深褐色的巧克力熔岩蛋糕，温热浓郁的巧克力熔岩从切口处缓缓流出
在白色瓷盘上蔓延，蛋糕表面撒着的糖粉轻微飘起，旁边烛光摇曳，
镜头俯拍缓缓推进聚焦流淌的巧克力熔岩，6秒，美食广告质感
```

## Category-Specific Tips

1. **烹饪视频的节奏要快** — 中式爆炒用短促有力的动作描述，甜品用缓慢优雅的节奏
2. **蒸汽=画面生命力** — 几乎每个热食视频都应该有蒸汽，"升腾的热气"是美食视频的灵魂
3. **特写+推镜=食欲** — 食物视频最经典的组合：微距特写 + 缓缓推近
4. **食材的转变瞬间最美** — 巧克力融化、蛋液凝固、面包膨胀、焦糖上色——捕捉"正在变"的那个瞬间
5. **慢动作适合液体** — 咖啡滴落、酱汁流淌、蜂蜜拉丝——这些场景天然适合慢节奏
