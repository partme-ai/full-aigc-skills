# Cinematic & Narrative — 电影感/叙事视频

## Instructions

叙事视频的核心是**用镜头讲故事**。区别于其他类别，叙事视频需要人物互动、情绪发展和空间关系的变化。运镜是讲故事的工具——推镜揭示真相、拉镜制造孤独、跟镜建立陪伴。

## Scenario Patterns

| 子场景 | 叙事核心 | 推荐运镜 | 典型触发词 |
|--------|---------|---------|-----------|
| 相遇/邂逅 | 两人目光交汇+走近 | 推+摇 | 相遇, 初见, 邂逅 |
| 离别/告别 | 背影远去+挥手+目送 | 拉 | 离别, 告别, 背影 |
| 等待/思念 | 窗前凝望+时间流逝 | 固定+推 | 等待, 思念, 守候 |
| 追逐/逃亡 | 奔跑+回头+紧张感 | 跟+摇 | 追逐, 逃亡, 逃跑 |
| 回忆/闪回 | 画面色调切换+动作重复 | 推 | 回忆, 往昔, 从前 |

## Examples

### Example: 车站离别

**用户需求**：写一个火车站送别的感人视频

**公式拆解**：
- 主体：一对情侣 + 女生在站台上 + 男生在火车窗内
- 动作：女生挥手告别 + 火车缓缓启动 + 她的手臂慢慢放下
- 场景：老式火车站月台 + 蒸汽/黄昏
- 运镜：镜头从两人中景缓缓拉远
- 光影：黄昏金色阳光斜照
- 时长：10秒
- 风格：电影感叙事

**最终提示词**：
```
老式火车站的月台上，一对情侣在离别，火车缓缓启动，女生站在月台上向车窗内的
男生挥手告别，火车逐渐加速远去，她的手缓缓放下，黄昏的金色阳光从侧面斜照
在她身上投下长长的影子，镜头从两人中景缓缓拉远展现月台的空旷和她的孤独身影，
10秒，电影感叙事风格，温暖而忧伤的色调
```

### Example: 雨夜重逢

**用户需求**：写一个雨夜两人重逢的浪漫视频

**公式拆解**：
- 主体：男女两人
- 动作：远远看到对方 → 快步走近 → 在雨中相拥
- 场景：雨夜老街路灯下 + 湿润的石板路
- 运镜：镜头缓缓推近两人
- 光影：昏黄路灯 + 雨幕中光晕
- 时长：10秒
- 风格：电影感浪漫

**最终提示词**：
```
雨夜的古老石板街道上，昏黄的路灯在雨幕中形成温暖的光晕，男人和女人远远看到对方，
快步走近后在路灯下紧紧相拥，雨滴落在他们身上和周围的石板路上激起水花，
空气中弥漫着雾气，镜头缓缓推近两人相拥的身影，10秒，电影感浪漫风格
```

### Example: 老人与猫

**用户需求**：写一个老人午后窗前看书的温馨视频

**公式拆解**：
- 主体：白发老人 + 橘猫
- 动作：老人慢慢翻书 + 猫蜷缩在腿上打盹 + 老人轻抚猫的背
- 场景：午后窗边 + 老式扶手椅 + 阳光洒入
- 运镜：镜头固定，仅有轻微推近
- 光影：午后阳光缓缓移动 + 尘埃在光束中飘浮
- 时长：10秒
- 风格：电影感温馨

**最终提示词**：
```
白发老人坐在窗边的老式扶手椅上缓缓翻阅一本旧书，一只橘猫蜷缩在他的腿上打盹，
老人不时轻抚猫的背，午后的阳光透过白色薄纱窗帘洒入，光束中飘浮着细小的尘埃，
光影在木地板上缓缓移动，镜头固定微微推近捕捉这宁静的午后时光，10秒，
电影感温馨风格，治愈而宁静
```

## Category-Specific Tips

1. **叙事视频需要"情感弧线"** — 不是简单的动作，而是情绪的起承转合
2. **运镜=情感语言** — 拉远=孤独/离别、推近=揭示/亲密、固定=观察/沉思
3. **光线变化=时间流逝** — 叙事视频中，光影的移动暗示时间的经过
4. **人物互动要有"空间关系变化"** — 从远到近、从分离到相拥——空间距离的变化本身就是故事
5. **简单即力量** — 老人翻书、情侣相拥、独自撑伞——简单的动作承载最重的情感
