# Nature & Landscape — 自然风景视频

## Instructions

自然风景视频的核心是**捕捉自然的"动"**——水流、云涌、风吹草动、光影流转。与人像视频不同，风景中主体往往是水、云、光、植被，运镜多用来展示宏大空间。

## Scenario Patterns

| 子场景 | 动态核心 | 推荐运镜 | 典型触发词 |
|--------|---------|---------|-----------|
| 瀑布/水流 | 水流动+水雾+水花 | 摇/推 | 瀑布, 溪流, 海浪 |
| 山岳/云海 | 云雾翻滚+光影变化 | 升/摇 | 云海, 山峦, 日出 |
| 花海/草原 | 风吹植物起伏+花瓣飘落 | 移/摇 | 花海, 草原, 薰衣草 |
| 沙漠/戈壁 | 沙粒流动+热浪+风痕 | 拉/摇 | 沙漠, 戈壁, 沙丘 |
| 极光/星空 | 光带流动+星辰移动 | 升/固定 | 极光, 星空, 银河 |
| 雪景/冰封 | 雪花飘落+冰晶闪烁 | 摇/推 | 雪, 冰, 冬日 |

## Examples

### Example: 瀑布

**用户需求**：写一个壮观瀑布的视频，要有水雾和阳光

**公式拆解**：
- 主体/场景：巨大瀑布 + 翠绿山谷
- 动作/动态：水流从高处倾泻而下 + 水雾升腾弥漫
- 光影：阳光穿透水雾形成彩虹
- 运镜：镜头从瀑布顶端缓缓下摇至底部深潭
- 时长：8秒
- 风格：风光纪录片、电影级画质

**最终提示词**：
```
巨大的瀑布从翠绿的山谷崖壁上倾泻而下，白色水雾在底部升腾弥漫，
阳光穿透水雾形成一道淡淡的彩虹，水花飞溅在镜头前，
镜头从瀑布顶端缓缓向下摇至底部深潭，8秒，风光纪录片风格，电影级画质
```

### Example: 薰衣草花海

**用户需求**：写一个薰衣草田在风中起伏的视频

**公式拆解**：
- 场景：紫色薰衣草花田 + 蓝天白云 + 远处农舍
- 动态：微风拂过 + 薰衣草如波浪起伏 + 几只蝴蝶飞舞
- 光影：午后暖阳
- 运镜：镜头缓缓横移
- 时长：8秒
- 风格：唯美治愈

**最终提示词**：
```
辽阔的紫色薰衣草花田在午后微风中如波浪般起伏，蓝天白云下远处有一座石头农舍，
几只白色蝴蝶在花丛间飞舞，午后暖阳将花田染成温暖的紫色，
镜头缓缓横移扫过花田，8秒，唯美治愈风格
```

### Example: 极光

**用户需求**：写一个北极光在夜空中舞动的视频

**公式拆解**：
- 场景：雪原 + 针叶林剪影 + 结冰湖面
- 动态：绿色极光带在夜空中如帷幔般飘动摇曳
- 光影：极光本身的绿光照亮雪地
- 运镜：镜头缓缓上升
- 时长：10秒
- 风格：自然奇观纪录片

**最终提示词**：
```
绿色和紫色的北极光带在北极圈的夜空中如巨大的帷幔般飘动摇曳，光芒在雪原上
投下变幻的绿色光影，前景的针叶林形成黑色剪影，结冰的湖面反射着极光，
镜头缓缓上升展现天地之间的壮丽极光，10秒，自然奇观纪录片风格
```

## Category-Specific Tips

1. **自然风景的"动"源于自然力** — 水、风、光是大自然的三位动画师
2. **运镜节奏要慢** — 风景视频的运镜应缓慢、平稳，给观众时间去感受
3. **自然声音通感** — 虽然视频不包含声音描述，但用视觉暗示声音："水花飞溅""树叶沙沙"
4. **天气变化=天然的叙事弧线** — "雾气渐散，阳光穿透" 自带起承转合
5. **前景元素增加纵深感** — 透过树枝/岩石/花草拍摄，让风景不扁平
