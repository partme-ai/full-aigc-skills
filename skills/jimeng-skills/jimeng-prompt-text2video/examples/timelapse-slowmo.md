# Timelapse & Slow Motion — 延时/慢动作视频

## Instructions

延时和慢动作是视频独有的两种时间操控手法。**延时摄影**压缩时间（城市车流如光河、花朵快速绽放），**慢动作**拉长时间（水花飞溅的每一帧、运动员腾空的瞬间）。即梦 Seedance 2.0 可以通过文字描述来实现这两种效果。

## Scenario Patterns

| 子场景 | 时间操控 | 动态核心 | 推荐运镜 | 典型触发词 |
|--------|---------|---------|---------|-----------|
| 城市延时 | 压缩 | 车流如光+云层翻滚+日夜转换 | 固定/升 | 延时, 车轨, 日夜 |
| 自然延时 | 压缩 | 花开+日出+潮汐+星轨 | 固定 | 延时, 花开, 日出 |
| 人群延时 | 压缩 | 人流如潮+光影变幻 | 固定/摇 | 延时, 人流, 时间流逝 |
| 水花慢动作 | 拉长 | 水滴+浪花+液体飞溅 | 推/固定 | 慢动作, 水花, 飞溅 |
| 运动慢动作 | 拉长 | 腾空+转身+落地瞬间 | 跟/固定 | 慢动作, 腾空, 瞬间 |
| 表情慢动作 | 拉长 | 微笑绽放+泪水滑落+回眸 | 推 | 慢动作, 微表情 |

## Examples

### Example: 城市车流延时

**用户需求**：写一个城市立交桥车流如光河的延时视频

**公式拆解**：
- 场景：城市立交桥 + 高楼天际线
- 动态：车流压缩为红白两色光轨 + 路灯和建筑灯光渐次亮起 + 天色从黄昏变为深夜
- 运镜：镜头固定正俯拍
- 时长：10秒
- 风格：城市延时摄影

**最终提示词**：
```
城市立交桥的延时摄影，天色从金色黄昏渐变为深蓝夜晚，车流压缩为红色的尾灯光轨
和白色的前灯光轨在立交桥的曲线上快速流动如光河，路灯和摩天楼的灯光一排排渐次亮起，
镜头固定正俯拍，10秒，城市延时摄影风格，时间的诗意
```

### Example: 水滴慢动作

**用户需求**：写一滴水滴入水面的极慢动作视频

**公式拆解**：
- 主体：一滴透明水滴
- 动作：水滴在空气中缓慢下落 → 接触水面 → 水面凹陷 → 水花皇冠般溅起 → 涟漪扩散
- 场景：深色水面 + 微光
- 运镜：镜头极缓推近（推）
- 时长：8秒
- 风格：高速摄影慢动作

**最终提示词**：
```
一滴透明的水滴在空气中以极慢的速度下落，接触到深色的水面时表面先形成凹陷，
然后水花如皇冠般向上溅起，每一颗飞溅的微小水珠都清晰可见地悬浮在空中，
圆形的涟漪从落点向四周缓慢扩散，镜头极缓推近捕捉水滴的每一帧细节，
8秒，高速摄影慢动作风格
```

### Example: 花朵绽放延时

**用户需求**：写一朵红玫瑰从花苞到盛开的延时视频

**公式拆解**：
- 主体：深红玫瑰花苞
- 动作：花苞缓慢展开→外层花瓣打开→层层绽放→完全盛开
- 场景：晨光+露珠
- 运镜：镜头固定微微推近
- 时长：8秒
- 风格：植物延时摄影

**最终提示词**：
```
一朵深红色的玫瑰花苞在晨光中以延时摄影的速度缓缓展开，外层花瓣先向外翻开，
露出内层紧裹的花瓣，随后内层花瓣一层接一层地舒展开来，花蕊从中心绽放，
花瓣上的晨露在绽放过程中滑落，镜头固定并微微推近花心，8秒，植物延时摄影风格
```

## Category-Specific Tips

1. **延时=压缩时间的魔法** — 关键提示词："时间压缩""快速流动""渐次变化""光影流转"
2. **慢动作=放大瞬间的诗意** — 关键提示词："极慢""每一帧都清晰""缓慢下落""逐帧可见"
3. **延时适合宏观** — 城市、天空、人群、自然——延时表现的是"大尺度的时间"
4. **慢动作适合微观** — 水滴、表情、运动瞬间——慢动作表现的是"被放大的细节"
5. **固定机位+延时=经典组合** — 延时效果最好的运镜是固定机位，让时间本身成为动画师
6. **慢动作+推镜=加倍放大** — 慢动作本身已经在放大时间，加上推镜等于放大空间——双重聚焦效果极好
