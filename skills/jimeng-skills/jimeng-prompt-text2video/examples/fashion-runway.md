# Fashion & Runway — 时尚/T台视频

## Instructions

时尚视频的核心是**展示服装的动态美**——走秀的步伐、衣物的摆动、材质的光泽流转。运镜多用跟拍和环绕来展示服装的立体感和流动性。

## Scenario Patterns

| 子场景 | 动态核心 | 推荐运镜 | 典型触发词 |
|--------|---------|---------|-----------|
| T台走秀 | 模特步伐+衣摆飘动+灯光 | 跟/环绕 | 走秀, 时装周, runway |
| 穿搭展示 | 转身展示+细节特写 | 环绕/推 | 穿搭, 展示, lookbook |
| 街拍/日常 | 自然行走+抓拍感 | 跟/摇 | 街拍, 日常穿搭 |
| 配饰特写 | 首饰闪烁+包包摆动 | 推/微距 | 珠宝, 包包, 手表 |
| 后台/准备 | 化妆+整理服装+紧张 | 摇/固定 | 后台, 化妆, 准备 |

## Examples

### Example: T台走秀

**用户需求**：写一个时装周T台走秀的视频

**公式拆解**：
- 主体：高挑模特 + 黑色曳地长裙
- 动作：自信的步伐 + 裙摆随步伐流动 + 走到台前定点转身
- 场景：时装周T台 + 深色背景+聚光灯
- 运镜：镜头跟随模特走到台前
- 光影：聚光灯追光 + 裙摆面料的光泽流动
- 时长：8秒
- 风格：时尚摄影

**最终提示词**：
```
高挑模特穿着黑色曳地长裙在时装周T台上自信地向前走去，裙摆的丝绒面料随着步伐
流动出柔和的光泽变化，走到T台尽头她优雅地定点转身，聚光灯从上方追光打在她身上，
深色背景中只有她的身影被照亮，镜头从正面跟随模特走到台前，8秒，时尚摄影风格
```

### Example: 珠宝特写

**用户需求**：写一个钻石项链在光线下闪耀的视频

**公式拆解**：
- 主体：铂金钻石项链 + 梨形主钻
- 动作：项链在光线下缓缓转动 + 钻石折射出七彩火彩闪烁
- 场景：深蓝丝绒底座
- 运镜：镜头极缓推近（推）
- 时长：6秒
- 风格：奢侈珠宝广告

**最终提示词**：
```
铂金钻石项链在深蓝色丝绒底座上缓缓转动，梨形主钻和密镶碎钻在柔和的点光源下
折射出七彩火彩不断闪烁，铂金链节的精密纹理在光线中清晰可见，
镜头极缓推近聚焦主钻，6秒，奢侈珠宝广告质感
```

## Category-Specific Tips

1. **面料是时尚视频的主角** — 丝绒的光泽流动、雪纺的轻盈飘动、皮革的反光，材质的动态是时尚视频的视觉核心
2. **慢节奏=高级感** — 时尚视频的运镜和动作都应偏慢，快节奏会削弱奢侈感
3. **光源要少而精** — 奢侈品用单光源或两点光源突出质感，不要均匀布光
4. **定点转身=黄金3秒** — 模特走到台前转身的瞬间，是所有T台视频的高潮点
