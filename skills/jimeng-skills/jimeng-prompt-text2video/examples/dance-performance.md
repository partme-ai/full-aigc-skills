# Dance & Performance — 舞蹈/表演视频

## Instructions

舞蹈视频的核心是**身体律动与运镜的配合**。舞蹈本身就是最纯粹的运动艺术，运镜需要与舞蹈的节奏和情绪同步——舒缓的芭蕾用缓慢的推镜，爆发的街舞用快速的跟拍。

## Scenario Patterns

| 子场景 | 动态核心 | 推荐运镜 | 典型触发词 |
|--------|---------|---------|-----------|
| 芭蕾 | 足尖旋转+跳跃+纱裙飘动 | 推/环绕 | 芭蕾, 天鹅湖 |
| 街舞/hiphop | 爆发动作+身体wave+定格 | 跟/环绕 | 街舞, breaking |
| 中国古典舞 | 水袖+长绸+旋转+身韵 | 摇/跟 | 古典舞, 水袖 |
| 现代舞 | 即兴+地板动作+流动 | 跟/推 | 现代舞, 当代舞 |
| 演唱会/现场 | 歌手+观众+灯光+互动 | 摇/升 | 演唱会, 音乐现场 |

## Examples

### Example: 芭蕾独舞

**用户需求**：写一个芭蕾舞者在空舞台上的独舞视频

**公式拆解**：
- 主体：女芭蕾舞者 + 白色tutu纱裙
- 动作：连续旋转 + 纱裙如花朵绽放 + 最后定格在arabesque
- 场景：空无一人的黑暗舞台 + 单束追光
- 运镜：镜头缓缓环绕
- 时长：8秒
- 风格：艺术舞蹈摄影

**最终提示词**：
```
女芭蕾舞者穿着白色tutu纱裙在空无一人的黑暗舞台上独舞，她在单束追光中连续优雅地旋转，
纱裙如云朵般层层展开，足尖鞋在木质舞台上轻盈地点踏，最后定格在arabesque姿态，
手臂向前延伸如天鹅颈项，镜头缓缓环绕她旋转，8秒，艺术舞蹈摄影风格
```

### Example: 街舞battle

**用户需求**：写一个街舞舞者的power move视频

**公式拆解**：
- 主体：街舞舞者 + 宽松运动服
- 动作：快速的身体wave → 倒立定格 → 风车旋转
- 场景：街头篮球场 + 围观人群
- 运镜：镜头环绕+推近
- 时长：6秒
- 风格：街头运动纪实

**最终提示词**：
```
街舞舞者在街头篮球场上做出爆发力十足的动作，身体wave从指尖传递到脚尖如电流，
接着单手倒立定格在air freeze姿势，随后连续风车旋转身体如陀螺般在地面扫过，
围观的人群发出欢呼，镜头从前方环绕至侧面并推近，6秒，街头运动纪实风格
```

### Example: 水袖古典舞

**用户需求**：写一个中国古典水袖舞的视频

**公式拆解**：
- 主体：古典舞女舞者 + 汉服+长水袖
- 动作：水袖甩出→在空中画弧→缓缓收回
- 场景：园林水榭 + 月光
- 运镜：镜头横摇跟随水袖轨迹
- 时长：8秒
- 风格：国风唯美

**最终提示词**：
```
古典舞女舞者身着汉服在月光下的园林水榭中起舞，她将数米长的白色水袖向空中甩出，
水袖在空中画出优美的弧线如流云一般，然后缓缓收回缠绕在手臂上，
月光洒在水面反射出粼粼波光照亮她的身影，镜头横摇跟随水袖的运动轨迹，8秒，国风唯美风格
```

## Category-Specific Tips

1. **运镜节奏与音乐同步感** — 在提示词中用"跟随舞步""随节奏律动"来暗示运镜与舞蹈的配合
2. **纱裙/水袖/长发 = 舞蹈的动态放大器** — 这些飘逸元素让舞蹈动作更具视觉冲击力
3. **特写+远景交替** — 舞蹈视频最有魅力的组合：全身展示姿态+推近特写表情
4. **中国古典舞+水袖是即梦的强项** — 流畅的布料模拟是Seedance的拿手好戏
