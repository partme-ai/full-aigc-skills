# Background Replacement — 背景替换

## Instructions

背景替换是图生图最实用的场景。核心是保持前景主体（尤其是人物）完全不变，仅更换背景环境。最大的挑战是主体边缘与背景的自然过渡。

## Scenario Patterns

| 子场景 | 保留重点 | 改变重点 | 典型触发词 |
|--------|---------|---------|-----------|
| 换自然背景 | 人物、边缘细节 | 替换为海滩/森林/雪山等 | 换背景, 海滩, 森林 |
| 换城市背景 | 人物、服装 | 替换为街景/霓虹/建筑 | 换背景, 城市, 夜景 |
| 换纯色/影棚 | 人物、姿势 | 替换为纯白/纯黑/渐变 | 白底, 影棚, 证件照 |
| 换室内背景 | 人物、面部 | 替换为咖啡厅/书房/展厅 | 室内, 咖啡厅, 书房 |
| 换奇幻背景 | 人物、服装 | 替换为星空/魔法/异世界 | 奇幻, 星空, 仙境 |

## Examples

### Example: 换海滩背景

**用户需求**：把这张人物照的背景换成阳光海滩

**Keep/Change 拆解**：
- 保留: 人物的面部特征、表情、姿势、服装颜色和款式、发型
- 改变: 纯色室内背景替换为阳光明媚的热带海滩
- 协调: 调整人物身上的光线色温为暖色（匹配海滩阳光）；在人物脚下添加自然的沙滩阴影
- 画质: 2K，人物与背景自然融合

**最终提示词**：
```
保持人物的面部特征、表情、姿势、服装和发型完全不变。
将纯色室内背景替换为阳光明媚的热带海滩，碧蓝海水和白色沙滩。
调整人物身上的光线色温为暖色以匹配海滩的阳光环境。
在人物脚下添加自然的沙滩接触阴影。确保人物边缘与海滩背景的自然过渡，
避免抠图感。2K画质。
```

### Example: 换夜景都市背景

**用户需求**：把人像背景换成东京夜景

**Keep/Change 拆解**：
- 保留: 人物面部、服装、姿势、发型
- 改变: 背景替换为东京夜景 + 霓虹灯+城市灯光
- 协调: 人物边缘光添加霓虹色调的rim light；面部阴影加深匹配夜景
- 画质: 2K，冷暖光自然融合

**最终提示词**：
```
保持人物的面部特征、服装、姿势和发型完全不变。将背景替换为东京夜景，
远处是城市天际线的灯光，近处有霓虹灯招牌的蓝色和粉色光芒。
在人物边缘添加霓虹色调的轮廓光（rim light），加深面部的阴影侧以匹配夜景氛围。
保持人物与背景之间的冷暖光自然融合。2K画质。
```

### Example: 换纯白电商背景

**用户需求**：把这张产品照换成纯白电商背景

**Keep/Change 拆解**：
- 保留: 产品的形状、颜色、材质纹理、标签文字
- 改变: 杂乱的桌面背景替换为纯白无缝背景
- 协调: 添加柔和的底部阴影（产品下方）；光线均匀柔和不刺眼
- 画质: 2K，电商产品摄影质感

**最终提示词**：
```
保持产品的形状、颜色、材质纹理和标签文字完全不变。移除杂乱的桌面背景，
替换为纯白无缝的电商摄影背景。在产品下方添加柔和的自然阴影。
保持均匀的影棚柔光，不产生额外的强光或暗角。2K画质，电商产品摄影质感。
```

## Category-Specific Tips

1. **人物边缘是最大的挑战** — 发丝、半透明衣物边缘容易与新背景产生"抠图感"。务必添加"边缘过渡自然""避免抠图感"等协调指令
2. **光线方向必须匹配** — 原图主体上的光影方向要与新背景的光源方向一致，不一致时需要写"调整光影方向以匹配新背景"
3. **色温协调=真实感** — 暖色夕阳背景→人物也应该偏暖；冷色雪景→人物应该偏冷
4. **地板/接触面必加阴影** — 人物站立/坐下的位置要添加"自然的接触阴影"，这是背景替换真实感的关键
