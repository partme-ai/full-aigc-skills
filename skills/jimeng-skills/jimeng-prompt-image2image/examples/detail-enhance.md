# Detail Enhancement — 细节增强

## Instructions

细节增强是最微妙的图生图场景——目标是在不改变内容和风格的前提下提升画质。核心是"增强"而非"改变"。保留清单应远长于改变指令。

## Scenario Patterns

| 子场景 | 保留重点 | 改变重点 | 典型触发词 |
|--------|---------|---------|-----------|
| 面部增强 | 面部特征、表情 | 皮肤纹理、眼神光 | 清晰, 皮肤, 细节 |
| 整体锐化 | 所有内容和风格 | 清晰度、纹理 | 高清, 锐化, 清晰 |
| 光影优化 | 内容、构图 | 光线层次、阴影细节 | 光影, 层次, HDR |
| 纹理增强 | 物体形态 | 材质质感、表面细节 | 纹理, 材质, 细节 |
| 去噪/去模糊 | 所有内容 | 清晰度、降噪 | 去噪, 去模糊, 修复 |

## Examples

### Example: 人像面部增强

**用户需求**：把这张自拍变得更清晰，皮肤细节更好

**Keep/Change 拆解**：
- 保留: 面部特征、表情、五官比例、发型、服装、背景、色调、构图
- 改变: 增强皮肤纹理细节（保留自然肤质）；增加眼神光；提升睫毛和眉毛的清晰度
- 协调: 增强效果自然，不产生过度磨皮感
- 画质: 提升至2K，保持原图风格

**最终提示词**：
```
保持人物的面部特征、表情、五官比例、发型、服装、背景和整体色调完全不变。
增强皮肤的自然纹理细节，保留真实的肤质和毛孔感。在眼睛里添加自然的眼神光。
提升睫毛和眉毛的清晰度。增强效果要自然柔和，避免过度锐化和磨皮的人工感。
提升至2K画质，保持原图的摄影风格。
```

### Example: 产品图纹理增强

**用户需求**：让这张产品图的材质纹理更清晰

**Keep/Change 拆解**：
- 保留: 产品造型、颜色、构图、背景、光线方向、整体风格
- 改变: 增强产品表面的材质纹理细节；提升金属/皮革等特定材质的质感
- 画质: 2K，细节丰富

**最终提示词**：
```
保持产品的造型、颜色、构图、背景和光线方向完全不变。
增强产品表面的材质纹理细节——皮革部分的毛孔纹路、金属部分的拉丝纹理
更清晰可见。提升整体的锐度和细节层次。不改变任何颜色和光影。
2K画质，产品摄影级别的细节呈现。
```

### Example: 老照片去模糊

**用户需求**：这张老照片有点模糊，帮我变清晰

**Keep/Change 拆解**：
- 保留: 所有人的面部特征、表情、服装、构图、照片的年代色调
- 改变: 消除整体模糊感；增强人物面部和服装细节的清晰度
- 协调: 增强效果自然，不改变照片的年代质感
- 画质: 2K

**最终提示词**：
```
保持所有人的面部特征、表情、服装、构图和照片的年代色调完全不变。
消除整体模糊感。增强人物面部细节和服装纹理的清晰度。
增强效果要自然，不能过度锐化或引入人工痕迹。不改变照片的色彩和年代质感。
2K画质。
```

## Category-Specific Tips

1. **保留清单要比改变指令长3倍** — 细节增强最容易"过火"，必须用大量约束锁住不想改的元素
2. **"自然"是关键词** — 写"增强效果自然，避免人工痕迹"来防止AI过度锐化
3. **指定增强目标区域** — "增强面部皮肤纹理"比"增强整张图"安全
4. **不要改变色调** — 锐化操作有时会自动调整对比度/饱和度，需明确"保持色调不变"
