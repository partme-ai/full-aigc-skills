# Season & Time Change — 季节/时间变换

## Instructions

改变图像中的季节或一天中的时间。核心是让所有受季节/时间影响的元素同步变化——不仅是天空颜色，植被、阴影长度、光线色温都需要协调改变。

## Scenario Patterns

| 子场景 | 保留重点 | 改变重点 | 典型触发词 |
|--------|---------|---------|-----------|
| 夏→秋 | 建筑、构图、人物 | 绿色→金黄/红叶；光线变暖斜 | 秋天, 红叶, 金黄 |
| 冬→春 | 建筑、地形 | 雪→绿草/花；暖意回归 | 春天, 花开, 新绿 |
| 白天→黄昏 | 内容、构图 | 蓝天→橙红天空；影子拉长 | 黄昏, 夕阳, golden hour |
| 白天→夜晚 | 建筑、构图 | 日光→月光/灯光；暗部加深 | 夜景, 夜晚, 星空 |
| 晴→雨 | 内容、构图 | 添加雨滴；天空变灰；地面湿润 | 下雨, 雨后, 湿润 |

## Examples

### Example: 夏天→秋天

**用户需求**：把这张夏天的公园照片变成秋天的景色

**Keep/Change 拆解**：
- 保留: 公园的道路、长椅、树木位置、建筑物、整体构图
- 改变: 树叶从绿色变为金黄和红色；地面添加落叶；光线变得更斜、更暖
- 协调: 所有植被同步变为秋色；光影色温偏暖；空气感更通透
- 画质: 2K

**最终提示词**：
```
保持公园的道路、长椅、树木位置、建筑物和整体构图完全不变。
将场景从夏天转换为深秋。所有树叶从绿色变为金黄和红色。
地面上铺满落叶。阳光变得更斜、更暖（低角度的秋日阳光）。
空气感更通透清冽。整体色调偏暖金黄。2K画质。
```

### Example: 白天→蓝调黄昏

**用户需求**：把这张城市照片变成蓝调时刻

**Keep/Change 拆解**：
- 保留: 建筑结构、街道布局、构图
- 改变: 天空从白天蓝变为深蓝暮色（blue hour）；建筑窗户亮起暖黄灯光
- 协调: 地面反射暮光转为冷蓝调；路灯亮起暖色光点
- 画质: 2K

**最终提示词**：
```
保持建筑结构和街道布局完全不变。将时间从白天转换为日落后蓝调时刻（blue hour）。
天空从白天的蓝色变为深邃的暮色蓝。建筑的窗户渐次亮起暖黄色灯光。
地面和建筑立面反射暮光转为冷蓝调。路灯亮起形成暖色光点。
冷暖光对比营造城市黄昏的静谧氛围。2K画质。
```

### Example: 晴天→雨景

**用户需求**：把这张晴天街景变成下雨天

**Keep/Change 拆解**：
- 保留: 街道结构、建筑、构图、主要物体位置
- 改变: 天空从晴转灰；地面变湿润并出现水洼；添加细密的雨丝；行人撑伞
- 协调: 整体色温变冷；建筑立面颜色因湿润而加深；水洼中有建筑的倒影
- 画质: 2K

**最终提示词**：
```
保持街道结构、建筑、构图和主要物体位置完全不变。将天气从晴天转换为下雨天。
天空变为灰暗的雨云。地面变为湿润的深色并出现大大小小的水洼反射着
建筑和灯光的倒影。添加细密的雨丝斜落在画面中。远处的行人撑起了雨伞。
整体色温变冷，建筑立面颜色因湿润而加深。2K画质。
```

## Category-Specific Tips

1. **季节变换=植被+光线+色调三重改变** — 只改植被颜色是不够的，光线和色温必须同步
2. **时间变换=天空+光源+阴影三重改变** — 不仅是天色，所有光源和阴影长度都要匹配
3. **蓝调时刻只有约20分钟** — 日落后天色快速变化，描述中要体现"暮色渐深"的过渡感
4. **秋天=最容易被AI理解的季节变换** — 绿色→金黄/红色 的色彩转换即梦执行得非常好
