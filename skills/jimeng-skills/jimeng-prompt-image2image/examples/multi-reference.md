# Multi-Reference Compositing — 多图合成

## Instructions

即梦 4.0+ 支持 6-10 张参考图同时输入。多图合成的核心是**明确每张图的角色**——哪张提供主体、哪张提供背景、哪张提供风格。

## Scenario Patterns

| 子场景 | 图像角色分配 | 典型触发词 |
|--------|------------|-----------|
| 人物+背景合成 | 图一=人物, 图二=背景 | 把这个人放到... |
| 换装参考 | 图一=人物, 图二=服装参考 | 换上这件衣服 |
| 材质替换 | 图一=物体, 图二=材质参考 | 换成这个材质 |
| 风格+内容 | 图一=内容, 图二=风格 | 用这个风格画... |
| 多人合成 | 图一=人物A, 图二=人物B | 把这两个人放在一起 |
| 多元素组图 | 图一/二/三=不同元素参考 | 综合这几张图 |

## Examples

### Example: 人物换背景（双图）

**用户需求**：把图一的人物放到图二的海滩上

**Keep/Change 拆解**：
- 图一（人物照）提供：人物面部、姿势、服装
- 图二（海滩照）提供：海滩背景的风格、色调、光线方向
- 协调：人物光影匹配图二海滩的阳光色温和方向

**最终提示词**：
```
参考图一的人物面部、姿势和服装，配上图二的海滩背景环境和光线氛围。
保持图一人物的所有特征完全不变。调整人物身上的光影方向和色温，
使其与图二海滩的阳光环境自然匹配。在人物脚下添加沙滩的接触阴影。
画面自然融合，无拼接感。2K画质。
```

### Example: 材质替换（双图）

**用户需求**：用图二的牛仔布材质替换图一陶罐的表面

**Keep/Change 拆解**：
- 图一（陶罐）提供：物体的形状、轮廓、光影结构
- 图二（牛仔布）提供：牛仔布的纹理、颜色、材质特征
- 协调：牛仔布纹理沿着陶罐的曲面自然弯曲；光影保持图一的侧顶光

**最终提示词**：
```
参考图一陶罐的形状和轮廓，将表面材质替换为图二的牛仔布纹理和颜色。
保持图一的物体轮廓、光影结构（侧顶强光）和背景墙上的陶罐阴影完全不变。
牛仔布纹理沿着陶罐的曲面自然弯曲变形，匹配三维立体感。
突出牛仔布料的编织肌理与光影结构。2K画质。
```

### Example: 三图合成

**用户需求**：图一的人物，穿上图二的衣服，站在图三的场景中

**Keep/Change 拆解**：
- 图一：提供人物的面部特征和基本姿势
- 图二：提供服装的款式、颜色和材质
- 图三：提供场景的环境、光线和氛围
- 协调：所有元素统一光源方向为图三的光线；色调全局统一

**最终提示词**：
```
参考图一的人物面部特征和姿势，穿上图二的服装款式和颜色，
站在图三的场景环境中。保持图一人物的面部特征和基本姿势不变。
图二服装的材质褶皱与图一人物的姿势自然匹配。
所有元素统一采用图三的光源方向和色调氛围。全局色彩协调统一。
画面自然融合如一张原始照片。2K画质。
```

## Category-Specific Tips

1. **每张图必须有明确的角色** — 含糊的"结合这两张图"会产生不可预测的结果
2. **光源统一是多图合成的第一挑战** — 务必写"所有元素统一光源方向"或指定哪张图的光线为准
3. **色调全局协调** — 不同来源的图像色温不同，需要"全局色彩协调统一"指令
4. **即梦 4.0 最多 6-10 张** — 超过此数量会失败，且图越多协调难度越大
5. **角色分工用"参考图X的Y"句式** — 清晰、不可误解
