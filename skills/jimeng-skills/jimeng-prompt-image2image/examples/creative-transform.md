# Creative Transformation — 创意变形

## Instructions

创意变形是最自由的图生图场景——将照片转换为创意/幻想版本。从人物动物化到建筑风格奇幻化，想象力是唯一的边界。

## Scenario Patterns

| 子场景 | 保留重点 | 改变重点 | 典型触发词 |
|--------|---------|---------|-----------|
| 人物动物化 | 面部特征、表情 | 身体/头部变为动物特征 | 变成猫, 动物化 |
| 年龄变换 | 面部特征、身份 | 年龄增减（变老/变年轻） | 变老, 变年轻, 童年 |
| 材质变身 | 轮廓、姿态 | 身体/物体变为特殊材质 | 变成金属, 水晶化 |
| 奇幻化 | 面部、姿态 | 添加奇幻元素（翅膀/光环/魔法） | 天使, 精灵, 魔法 |
| 微缩/巨人 | 人物特征 | 尺度变化+环境匹配 | 变小, 巨人, 小人国 |

## Examples

### Example: 人物猫咪化

**用户需求**：把我的照片变成猫娘风格

**Keep/Change 拆解**：
- 保留: 面部特征、五官辨识度、表情、发型
- 改变: 添加猫耳朵和猫尾巴；眼睛变为猫眼竖瞳；肤色略微柔化
- 协调: 猫耳的颜色与发色协调；整体风格保持原图的二次元/写实基调
- 画质: 2K

**最终提示词**：
```
基于上传的照片进行萌系猫娘创意变形。保留原图的面部特征、五官辨识度、
表情和发型。添加与发色同色系的猫耳朵和毛茸茸的长尾巴。
眼睛变为猫眼竖瞳，更大更圆。肤质柔化。整体保持原图的风格基调。
创意变形效果自然可爱。2K画质。
```

### Example: 人物精灵化

**用户需求**：把我变成奇幻精灵

**Keep/Change 拆解**：
- 保留: 面部特征、五官、表情、姿势、服装风格
- 改变: 耳朵变为精灵尖耳；添加发光的透明翅膀；周围添加魔法光点
- 画质: 2K，奇幻风格

**最终提示词**：
```
基于上传的照片进行奇幻精灵创意变形。保留原图的面部特征、五官、表情和姿势。
耳朵变为精灵式的尖长耳朵。背后添加半透明的发光翅膀，翅膀上有细腻的脉络纹理。
周围漂浮着细小的魔法光点。整体色调偏向神秘的银蓝色。
服装保持原图风格但添加精灵式的细节装饰。2K画质，奇幻风格。
```

### Example: 年龄变换

**用户需求**：看看我老了以后的样子

**Keep/Change 拆解**：
- 保留: 面部骨骼结构、五官位置比例、表情、发型款式
- 改变: 添加自然的皱纹和皮肤老化痕迹；发色变灰白
- 协调: 老化效果自然写实，不夸张变形
- 画质: 2K

**最终提示词**：
```
基于上传的照片进行自然的年龄增加创意变形。保留面部骨骼结构和五官位置比例。
添加自然的细纹、鱼尾纹和皮肤老化痕迹。发色变为自然的灰白。
老化效果写实自然，不夸张不扭曲。保持表情和神态不变。2K画质。
```

## Category-Specific Tips

1. **创意变形的灵魂是"辨识度"** — 变形后仍要能认出原人物，保留"面部特征"是核心
2. **添加元素要有"归属感"** — 猫耳的毛色要匹配发色，翅膀的风格要匹配服装风格
3. **奇幻元素要写物理属性** — 翅膀是"透明发光的"还是"羽毛质感的"，细节决定真实感
4. **年龄变换=最难的创意变形** — 即梦对人脸年龄的处理精度有限，效果可能不如预期
