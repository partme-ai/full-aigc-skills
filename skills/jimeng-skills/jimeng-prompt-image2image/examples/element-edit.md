# Element Add/Remove — 元素添加/移除

## Instructions

在图像中添加新物体或移除不需要的元素。核心挑战是添加的元素要"融入"场景，移除后的空白区域要"合理填充"。

## Scenario Patterns

| 子场景 | 保留重点 | 改变重点 | 典型触发词 |
|--------|---------|---------|-----------|
| 添加物品 | 除目标位置外的所有区域 | 在指定位置添加新元素 | 添加, 加上, 放一个 |
| 移除杂物 | 除目标外的所有内容 | 移除指定元素+填补空白 | 移除, 去掉, 删除 |
| 替换物品 | 场景、光线 | 将一个物体替换为另一个 | 替换, 换成, 改成 |
| 添加人物/动物 | 背景和场景 | 在场景中添加人物/动物 | 加个人, 放只猫 |

## Examples

### Example: 添加元素

**用户需求**：在桌面咖啡旁边添加一支花

**Keep/Change 拆解**：
- 保留: 咖啡杯、桌面纹理、光线方向、整体色调、桌面上其他所有物品
- 改变: 在咖啡杯右侧添加一支白色马蹄莲插在小玻璃瓶中
- 协调: 花瓶的阴影方向与原图光源一致；花瓣的亮度与场景中其他白色物体匹配
- 画质: 2K，添加痕迹不可见

**最终提示词**：
```
保持咖啡杯、桌面纹理、光线方向、整体色调和桌面上所有其他物品完全不变。
在咖啡杯右侧添加一支白色马蹄莲插在透明的小玻璃瓶中，瓶中有少量清水。
花瓶投射的阴影方向与原图的光源方向一致。花瓣的亮度和色温与场景中
其他白色物体的光照匹配。添加效果自然，无人工痕迹。2K画质。
```

### Example: 移除元素（去路人）

**用户需求**：把照片背景里的路人去掉

**Keep/Change 拆解**：
- 保留: 主体人物、建筑、街道、光线、整体构图
- 改变: 移除背景中的路人；用合理的场景内容（地面纹理、建筑立面）自然填充空白
- 画质: 2K，编辑痕迹不可见

**最终提示词**：
```
保持主体人物、建筑、街道和整体构图完全不变。移除背景中所有路人。
移除后的空白区域用周围的场景内容自然填充——地面纹理延续、建筑立面完整。
不产生模糊或涂抹的痕迹。光线和色调在编辑区域与周围完全一致。
2K画质，编辑痕迹不可见。
```

### Example: 替换物品

**用户需求**：把桌上的水杯换成咖啡杯

**Keep/Change 拆解**：
- 保留: 桌面纹理、光线方向、整体色调、桌上其他所有物品
- 改变: 透明水杯替换为一个白色陶瓷咖啡杯，杯中有深色咖啡
- 协调: 咖啡杯的阴影方向与原图光源一致；杯子的材质反射与桌面的光线匹配
- 画质: 2K

**最终提示词**：
```
保持桌面纹理、光线方向、整体色调和桌上所有其他物品完全不变。
将透明水杯替换为一个白色陶瓷咖啡杯，杯中盛有深色热咖啡，杯口有细微的蒸汽。
咖啡杯投射的阴影方向与原图的光源方向一致。杯子的陶瓷材质反射
与桌面光线环境自然匹配。替换效果天衣无缝。2K画质。
```

## Category-Specific Tips

1. **添加元素要写"物理属性"** — 新物体的大小、位置、与光源的关系、与周围物体的遮挡关系
2. **移除后用场景内容填充** — 明确写"用周围的地面纹理/建筑立面填充空白"，否则AI可能留下模糊痕迹
3. **阴影/反光是融入关键** — 添加的物体必须投射与场景光源一致的阴影，否则像P上去的
4. **即梦4.0不支持精确局部编辑** — 无法编辑"只改左下角"，删除/添加指令会全局生效。选择简单场景成功率更高
