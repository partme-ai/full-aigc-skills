# Color Adjustment — 色彩调整

## Instructions

色彩调整改变图像的色调和色彩氛围，同时保留所有内容和构图。核心是"重新调色"而非"重新绘制"。

## Scenario Patterns

| 子场景 | 保留重点 | 改变重点 | 典型触发词 |
|--------|---------|---------|-----------|
| 冷暖调转换 | 所有内容 | 色温从暖→冷 或 冷→暖 | 暖色调, 冷色调, 色温 |
| 电影调色 | 内容、构图 | 应用特定电影色调 | 电影调色, 胶片色, 色彩分级 |
| 季节色调 | 内容、构图 | 色调匹配某季节 | 秋色, 春色, 冬色 |
| 单色/黑白 | 内容、构图 | 转为黑白或单色调 | 黑白, 单色, 褪色 |
| 鲜艳/柔和 | 内容、构图 | 饱和度升降 | 鲜艳, 柔和, 低饱和 |

## Examples

### Example: 暖调转冷调

**用户需求**：把这张暖色调照片改成冷蓝调

**Keep/Change 拆解**：
- 保留: 构图、人物、场景、所有内容细节
- 改变: 色调从暖黄转为冷蓝；降低暖色的饱和度；增强冷色的比重
- 协调: 色彩过渡自然平滑；肤色在冷调中仍保持自然
- 画质: 2K

**最终提示词**：
```
保持构图和所有内容完全不变。将整体色调从暖黄色调整为冷蓝色调。
降低画面中暖色（橙、黄、红）的饱和度。增强冷色（蓝、青）的比重。
将白平衡向冷色端偏移。肤色在冷调中仍保持自然不偏青。色彩过渡平滑自然。
2K画质。
```

### Example: 电影感调色

**用户需求**：给这张照片加上王家卫电影风格的色调

**Keep/Change 拆解**：
- 保留: 所有内容、构图
- 改变: 应用王家卫电影的色彩分级——高光偏暖黄、暗部偏绿；增加轻微的胶片颗粒感
- 画质: 2K，电影质感

**最终提示词**：
```
保持构图和所有内容完全不变。应用王家卫电影风格的色彩分级——
高光区域偏暖黄色调，暗部偏墨绿色调。增加轻微的胶片颗粒感。
整体对比度柔和，色彩浓郁但不刺眼。营造颓废而浪漫的电影感色调。
2K画质，电影质感。
```

### Example: 鲜艳色调转莫兰迪灰调

**用户需求**：把这张饱和度很高的照片改成莫兰迪灰调

**Keep/Change 拆解**：
- 保留: 构图、人物、场景、所有内容细节
- 改变: 整体饱和度降低；所有颜色掺入一层灰度；形成莫兰迪式的高级灰调
- 协调: 色彩过渡自然；肤色在低饱和中仍保持自然柔和
- 画质: 2K

**最终提示词**：
```
保持构图和所有内容完全不变。将整体饱和度大幅降低，
所有颜色中掺入一层灰度，形成莫兰迪式的高级灰调。
暖色变为柔和的灰粉、灰橘、灰杏；冷色变为灰蓝、灰绿、烟灰紫。
肤色在低饱和中保持自然柔和。色彩过渡平滑自然，整体呈现静谧的文艺氛围。
2K画质。
```

## Category-Specific Tips

1. **色彩调整≠重新绘制** — 强调"保持所有内容不变"防止AI误改构图
2. **引用电影/胶片风格很有效** — "王家卫色调""Kodak Portra色彩""韦斯·安德森配色"
3. **肤色在调色中的保护** — 冷暖调转换时，务必写"肤色保持自然"防止人物变青或变黄
4. **饱和度变化要具体** — "降低暖色的饱和度"比"降低饱和度"精确
