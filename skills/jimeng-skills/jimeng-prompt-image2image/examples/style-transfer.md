# Style Transfer — 风格迁移

## Instructions

风格迁移是图生图最常用的场景之一。核心是保留原图的构图和主体，将艺术风格完全转换。即梦 4.0+ 支持照片→油画、写实→二次元、现代→水墨等多种风格转换。

## Scenario Patterns

| 子场景 | 保留重点 | 改变重点 | 典型触发词 |
|--------|---------|---------|-----------|
| 照片→油画 | 构图、人物特征、光影结构 | 笔触肌理、色彩饱和 | 油画风格, 梵高, 印象派 |
| 写实→二次元 | 人物特征、姿势、构图 | 上色方式、线条风格 | 二次元, 动漫化, 赛璐珞 |
| 照片→水墨 | 山水轮廓、空间关系 | 笔触、留白、墨色 | 水墨, 国画, 写意 |
| 照片→水彩 | 主体形态、构图 | 透明感、渲染、柔和 | 水彩, 清新, 淡雅 |
| 现代→复古 | 内容、构图 | 色调、颗粒感、做旧 | 复古, 胶片, 老照片 |
| 写实→3D卡通 | 人物特征、姿势 | 建模风格、材质 | Pixar, 3D卡通, C4D |

## Examples

### Example: 照片→水墨画

**用户需求**：把这张风景照变成水墨画风格

**Keep/Change 拆解**：
- 保留: 原图的山峦轮廓、水面位置、亭子结构、整体空间关系
- 改变: 色彩转为黑白灰墨色；笔触模仿传统水墨的皴擦点染；增加大面积留白
- 协调: 墨色浓淡与原图的光影逻辑一致（亮处淡墨、暗处浓墨）
- 画质: 宣纸质感，水墨韵味

**最终提示词**：
```
基于上传的风景照片，生成一幅水墨画风格版本。保留原图的山峦轮廓、水面位置
和亭子结构。色彩转为黑白灰墨色，笔触模仿传统水墨的皴擦点染技法。
画面增加大面积留白营造意境。墨色浓淡与原图的光影逻辑一致。
宣纸质感，水墨韵味，2K画质。
```

### Example: 照片→吉卜力动画

**用户需求**：把我的照片变成宫崎骏动画风格

**Keep/Change 拆解**：
- 保留: 人物的面部特征、姿势、服装款式、场景的空间关系
- 改变: 上色方式转为吉卜力风格的手绘水彩质感；线条变为柔和的铅笔线稿感
- 协调: 保留原图的光影氛围，但用动画风格重新演绎
- 画质: 动画电影画质

**最终提示词**：
```
基于上传的照片，生成吉卜力动画风格版本。保留人物的面部特征和姿势。
将上色方式转换为吉卜力风格的手绘水彩质感，线条柔和如铅笔稿。
保留原图的光影氛围，用温暖的动画色调重新演绎。画面有手绘的细腻感。
吉卜力动画电影画质，2K。
```

### Example: 照片→赛博朋克

**用户需求**：把这张街景照片改成赛博朋克风格

**Keep/Change 拆解**：
- 保留: 原图的建筑结构、街道布局、透视关系
- 改变: 整体色调转为蓝紫调；添加霓虹灯光效和全息投影元素
- 协调: 建筑的材质转为未来感金属和玻璃；天空变为暗夜
- 画质: 超写实，霓虹灯光影细腻

**最终提示词**：
```
将这张街景照片转换为赛博朋克风格。保留原图的建筑结构和街道布局。
整体色调转为深蓝紫色调。添加霓虹灯招牌和全息投影广告的光效。
建筑材质转为未来感的金属和玻璃质感，天空变为暗夜。
超写实渲染，霓虹灯光影细腻，2K画质。
```

## Category-Specific Tips

1. **风格迁移=保留构图+改变渲染方式** — 构图是骨架，风格是皮肤
2. **引用具体风格名效果更好** — "吉卜力风格"比"动画风格"精准
3. **光影逻辑要跨风格保持** — 原图的光源方向在新风格中应保持一致
4. **风格强度靠描述细节控制** — 详细描述笔触/上色方式=强风格转换；简单说风格名=弱风格转换
