# Restoration — 修复/复原

## Instructions

修复老照片的划痕、折痕、褪色和模糊。核心是清除损伤的同时保留照片的年代质感和历史氛围——修复≠现代化。

## Scenario Patterns

| 子场景 | 保留重点 | 改变重点 | 典型触发词 |
|--------|---------|---------|-----------|
| 去划痕/折痕 | 所有人脸、服装、构图 | 清除物理损伤痕迹 | 划痕, 折痕, 破损 |
| 色彩复原 | 内容、构图 | 恢复褪色的原始色彩 | 褪色, 发黄, 恢复色彩 |
| 去模糊 | 人脸特征 | 增强细节清晰度 | 模糊, 不清晰, 锐化 |
| 完整修复 | 年代质感 | 综合修复所有损伤 | 老照片修复, 翻新 |

## Examples

### Example: 老照片综合修复

**用户需求**：修复这张有划痕和褪色的老照片

**Keep/Change 拆解**：
- 保留: 人物的面部特征、表情、姿态、服装的年代款式、照片的年代氛围
- 改变: 清除划痕和折痕；恢复褪色的色彩；修复破损的边角
- 协调: 修复后的色彩饱和度要自然，不要过于鲜艳（匹配当年的胶片色彩）
- 画质: 2K，保留老照片的质感

**最终提示词**：
```
修复这张老照片。清除所有划痕、折痕和黄色污渍。修复破损的边角。
恢复褪色的原始色彩，但保持自然的饱和度——不要过于鲜艳，
应匹配当年胶片摄影的色彩特征。增强面部细节的清晰度。
保留照片的年代质感和历史氛围，不要过度现代化。2K画质。
```

### Example: 黑白照片上色

**用户需求**：给这张黑白老照片上色

**Keep/Change 拆解**：
- 保留: 人物的面部特征、表情、构图、光影层次
- 改变: 从黑白转为自然彩色
- 协调: 肤色自然（亚洲人肤色）；服装颜色符合年代特征
- 画质: 2K

**最终提示词**：
```
为这张黑白老照片进行自然上色。保持人物的面部特征、表情、构图和光影层次
完全不变。肤色上色为自然的亚洲人肤色。服装颜色应符合当时年代的
典型配色。整体色彩的饱和度和对比度自然柔和，不夸张。
上色后的照片仍保留老照片的年代氛围。2K画质。
```

### Example: 修复破损边角

**用户需求**：这张照片的边角破损了，帮我修复完整

**Keep/Change 拆解**：
- 保留: 照片中所有人物和内容、色调、年代氛围
- 改变: 修复破损缺失的边角区域；用合理的画面内容自然延伸填补缺失部分
- 画质: 2K

**最终提示词**：
```
保持照片中所有人物、内容和整体色调完全不变。修复破损缺失的边角区域。
用合理的画面内容自然延伸填补缺失的角——延续背景的纹理和色调。
修复区域与原有画面的过渡无缝自然，看不出修复痕迹。
保持照片的年代质感不因修复而改变。2K画质。
```

## Category-Specific Tips

1. **修复≠现代化** — 务必写"保留年代质感"防止AI把老照片变成现代数码照片
2. **色彩复原要克制** — 老照片即使修复后，饱和度也不应像现代照片一样鲜艳
3. **人脸是修复的第一优先级** — 明确写"保持面部特征不变"，防止AI在去模糊时改变五官
4. **上色要指定肤色和年代** — 不同年代服装颜色不同，写"符合年代特征的配色"防止穿越
