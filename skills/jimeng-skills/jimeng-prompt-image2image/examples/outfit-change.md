# Outfit Change — 换装/服装替换

## Instructions

保持人物身份（面部、发型、姿势）不变，更换服装。核心是服装与人物的自然贴合——材质褶皱要与姿势匹配，光线要与原图一致。

## Scenario Patterns

| 子场景 | 保留重点 | 改变重点 | 典型触发词 |
|--------|---------|---------|-----------|
| 换日常穿搭 | 面部、发型、姿势 | 更换日常服装 | 换衣服, 换穿搭 |
| 换正装/礼服 | 面部、发型、姿势 | 换为正装/晚礼服 | 西装, 礼服, 正装 |
| 换古装/汉服 | 面部、发型 | 换为汉服/古装 | 汉服, 古装, 旗袍 |
| 换制服/职业装 | 面部、发型、姿势 | 换为制服 | 制服, 白大褂, 军装 |
| 换奇幻服装 | 面部、发型 | 换为奇幻/科幻服装 | 铠甲, 魔法袍, 太空服 |

## Examples

### Example: 换汉服

**用户需求**：把我的照片换成明制汉服

**Keep/Change 拆解**：
- 保留: 面部特征、五官、表情、发型（但可加发簪）
- 改变: 现代服装更换为明制汉服——红色圆领袍+马面裙
- 协调: 服装的褶皱要匹配人物的站立姿势；汉服的面料光泽与原图光线方向一致
- 画质: 2K，服装材质细节清晰

**最终提示词**：
```
保持人物的面部特征、五官、表情和发型完全不变。
将现代服装更换为明制汉服——红色圆领袍搭配织金马面裙。
服装的褶皱要自然匹配人物的站立姿势。汉服丝绸面料的光泽方向
与原图光线方向一致。发型上添加一支金色发簪作为点缀。
2K画质，服装材质细节清晰。
```

### Example: 换晚礼服

**用户需求**：把我的日常照换成晚礼服造型

**Keep/Change 拆解**：
- 保留: 面部特征、五官、表情、发型款式
- 改变: 休闲服装更换为黑色丝绒曳地晚礼服
- 协调: 丝绒面料的光泽流动要与人物的身体曲线和姿态匹配
- 画质: 2K，高级时装质感

**最终提示词**：
```
保持人物的面部特征、五官、表情和发型款式完全不变。将休闲服装更换为
黑色丝绒曳地晚礼服，深V领设计，裙摆自然垂落在地面。丝绒面料的光泽流动
与人物的身体曲线和站姿自然匹配。保持原图的光线方向和色调。
2K画质，高级时装摄影质感。
```

### Example: 换职业装

**用户需求**：把我的休闲照换成白大褂医生造型

**Keep/Change 拆解**：
- 保留: 面部特征、五官、表情、发型
- 改变: 休闲服装更换为白大褂+听诊器
- 协调: 白大褂的衣领和褶皱要自然匹配人物的姿势
- 画质: 2K

**最终提示词**：
```
保持人物的面部特征、五官、表情和发型完全不变。将休闲服装更换为
医生的白大褂，左胸口袋中插着一支笔，脖子上挂着听诊器。
白大褂的衣领自然立起，衣身的褶皱与人物姿势匹配。
保持原图的光线方向和色调。2K画质，专业形象照质感。
```

## Category-Specific Tips

1. **服装褶皱=真实感的关键** — 写"服装褶皱自然匹配人物姿势"，僵硬的无褶皱服装会像纸片
2. **面料光泽方向要与光线一致** — 丝绸/丝绒/皮革的光泽方向必须匹配原图光源
3. **领口/袖口/腰线要写清楚** — 服装的结构决定最终效果，"深V领""收腰设计"比"好看的裙子"精准
4. **配饰随服装一起换** — 晚礼服需要配高跟鞋和手包，汉服需要配发簪和团扇
