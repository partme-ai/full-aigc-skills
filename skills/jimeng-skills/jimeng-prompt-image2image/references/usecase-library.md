# Use Case Library — 图生图编辑场景模板库

> 10 大图生图编辑场景的 Keep/Change 模板。每个场景定义了标准化的保留-改变-协调-画质公式。

---

## 1. 风格迁移 (Style Transfer)

**场景**：保留图像内容，改变艺术风格（照片→油画、写实→二次元、现代→水墨等）

**Keep/Change 模板**：
```
保留: 原图的构图、主体轮廓、空间关系、人物特征
改变: 艺术风格转换为 [目标风格]
协调: 保持原图的光影逻辑，在新风格中体现一致的明暗关系
画质: [2K/4K]，[目标风格的质感描述]
```

**提示词模式**：
```
基于上传的图片，生成一幅 [目标风格] 版本。保留原图的构图和主体轮廓。
色调转为 [新色调描述]。笔触模仿 [风格特定的技法]。增加 [风格特定的元素，如留白/笔触/颗粒感]。
```

---

## 2. 背景替换 (Background Replacement)

**场景**：保持前景主体不变，替换背景环境

**Keep/Change 模板**：
```
保留: 人物/主体的面部、姿势、服装、表情、边缘细节
改变: 背景替换为 [新背景描述]
协调: 调整主体身上的光影方向/色温，使其与新背景的环境光一致
画质: [2K/4K]，主体与背景的自然融合
```

**提示词模式**：
```
保持人物、姿势和服装完全不变。将背景替换为 [新背景描述]。
根据新背景的环境光方向调整人物身上的光影，确保人物与背景的自然融合。
注意人物边缘与背景的过渡要自然，避免抠图感。
```

---

## 3. 细节增强 (Detail Enhancement)

**场景**：提升画质、锐化纹理、增强光影，不改变内容和风格

**Keep/Change 模板**：
```
保留: 构图、内容、风格、色调、所有元素的位置关系
改变: 增强 [目标区域] 的细节和纹理清晰度；改善 [光线效果]
协调: 增强效果要自然，不能过度锐化或产生人工感
画质: 提升至 [2K/4K]，保持原图风格
```

**提示词模式**：
```
增强图像的整体细节和纹理清晰度，尤其是 [重点区域]。改善光影层次，
增加 [特定光线效果]。保持构图、内容、风格和色调完全不变。
增强效果要自然柔和，避免过度锐化。
```

---

## 4. 色彩调整 (Color Adjustment)

**场景**：改变整体或局部的色彩方案、色调、氛围

**Keep/Change 模板**：
```
保留: 构图、主体、内容、细节
改变: 色调从 [原色调] 调整为 [新色调]；色彩氛围从 [原氛围] 转变为 [新氛围]
协调: 确保色彩过渡自然，没有生硬的色块分割
画质: [2K/4K]，色彩过渡细腻
```

**提示词模式**：
```
保持构图和内容不变。将整体色调从 [原色调] 调整为 [新色调]。
增强 [特定颜色的饱和度/明度]。降低 [特定颜色] 的比重。
色彩过渡要自然平滑，营造 [新氛围] 的感觉。
```

---

## 5. 元素添加/移除 (Element Add/Remove)

**场景**：在图像中添加新物体或移除不需要的元素，并自然填补空白

**Keep/Change 模板**：
```
保留: 除 [要移除的元素] 外的所有内容
改变: [添加/移除] [目标元素] 在 [位置]
协调: 新元素与周围环境的材质/光影统一；移除后空白区域用合理内容填充
画质: [2K/4K]，编辑痕迹不可见
```

**提示词模式**：
```
在图像的 [位置] 添加 [元素描述]。新元素的光影方向要与场景中的光源一致，
材质的反射特性要与周围物体协调。保持图像其他部分完全不变。
```

---

## 6. 季节/时间变换 (Season/Time Change)

**场景**：改变图像中的季节或一天中的时间

**Keep/Change 模板**：
```
保留: 主体、构图、建筑/地形结构
改变: 季节从 [原季节] 变为 [新季节]；或 时间从 [原时间] 变为 [新时间]
协调: 所有受季节/时间影响的元素都要同步变化（植被、光影、天空、阴影长度）
画质: [2K/4K]，季节/时间变化的自然过渡
```

**提示词模式**：
```
保持主体和构图不变。将场景的季节从 [原季节] 转换为 [新季节]。
[新季节的特征描述：如春天的樱花和新绿、秋天的红叶和金黄、冬天的白雪和枯枝]。
同步调整光影色温和阴影长度以匹配 [新季节/新时间] 的特征。
```

## 7. 修复/复原 (Restoration)

**场景**：修复老照片的划痕、褪色、破损；复原模糊的旧照片

**Keep/Change 模板**：
```
保留: 原始人物的面部特征、表情、姿态、服装的年代感
改变: 修复划痕/折痕/褪色/模糊；增强清晰度和对比度
协调: 修复效果要保留照片的年代质感，不要过度现代化
画质: 提升清晰度的同时保留原始照片的年代氛围
```

**提示词模式**：
```
修复这张老照片。去除划痕、折痕和污渍。修复褪色区域，恢复原始的色彩和对比度。
增强面部细节的清晰度。保留照片的年代质感和历史氛围，不要过度现代化。
```

---

## 8. 创意变形 (Creative Transformation)

**场景**：将照片转换为创意/幻想版本——如人物动物化、建筑风格奇幻化等

**Keep/Change 模板**：
```
保留: 原图的基本构图和姿态/结构
改变: [元素A] 变形为 [创意版本]；[元素B] 添加 [奇幻特征]
协调: 变形后的元素需要与原图中未被改变的部分保持风格统一
画质: [2K/4K]，创意效果的精细呈现
```

**提示词模式**：
```
基于原图的构图和姿态，将 [目标元素] 创意变形为 [奇幻描述]。
保留 [必须保留的特征]。添加 [奇幻元素/特效]。整体风格统一，变形自然。
```

---

## 9. 换装/服装替换 (Outfit Change)

**场景**：保持人物身份不变，更换服装/造型

**Keep/Change 模板**：
```
保留: 人物面部特征、表情、发型、肤色、姿势
改变: 服装从 [原服装] 更换为 [新服装描述]
协调: 新服装的材质褶皱要与人物的姿势和体型匹配；光线方向与原图一致
画质: [2K/4K]，服装材质细节清晰
```

**提示词模式**：
```
保持人物的面部特征、表情、发型和姿势完全不变。将服装更换为 [新服装描述]。
新服装的材质和褶皱要与人物的姿势自然匹配。保持原图的光线方向和整体色调。
```

## 10. 多图合成 (Multi-Reference Compositing)

**场景**：从多张参考图中提取不同元素，合成为一张新图像（即梦 4.0+ 支持 6-10 张参考图）

**Keep/Change 模板**：
```
图一角色: 保留 [图一中的人物/主体特征]
图二素材: 使用 [图二中的元素/材质/风格]
图三素材: 使用 [图三中的元素/背景/光线]
协调: 所有元素的光影方向统一、色调协调、风格融合
画质: [2K/4K]，多元素合成的无缝融合
```

**提示词模式**：
```
参考图一的 [角色特征，如姿势、面部]，换上图二的 [元素，如发型/服装]，
配上图三的 [元素，如背景/道具]。所有元素的光影方向统一为 [光源描述]。
整体色调协调，风格统一，画面自然融合。
```

## 编辑强度控制

| 强度等级 | 描述 | 提示词中的体现 | 适用场景 |
|---------|------|-------------|---------|
| 轻 (20-30%) | 微妙调整 | "略微增强""稍作调整""保持原图95%不变" | 细节增强、色彩微调 |
| 中 (40-60%) | 明显但可控 | "将...替换为...""调整...同时保留..." | 背景替换、季节变换 |
| 强 (70-90%) | 大幅变换 | "完全改变风格""重新演绎""创意变形" | 风格迁移、创意变形 |

> 提示词的详细程度控制编辑强度：描述越具体详尽 = 编辑强度越高。模糊简短的描述 = 轻量调整。
