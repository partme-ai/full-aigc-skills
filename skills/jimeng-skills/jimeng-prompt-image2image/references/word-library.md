# Word Library — 图生图编辑词汇库

> 图生图专属词汇，聚焦"保留/改变/协调"三大编辑维度。常规描述词汇（人物/场景/风格等）请参考 text2image 的 `word-library.md`。

---

## 一、保留词库 (Keep Vocabulary)

### 1.1 保留动词

`保持...不变` · `保留` · `维持` · `不改变` · `确保...不被修改` · `锁定` · `固定` · `原样保留`

### 1.2 常保留的元素

`构图` · `主体轮廓` · `人物面部特征` · `表情` · `姿势` · `服装` · `发型` · `空间关系` · `光影方向` · `整体色调` · `分辨率` · `背景结构` · `建筑轮廓` · `色彩氛围`

### 1.3 保留指令模板

```
保持 [元素A]、[元素B] 和 [元素C] 完全不变
```

---

## 二、改变词库 (Change Vocabulary)

### 2.1 改变动词

`替换为` · `转换为` · `调整为` · `修改为` · `更改为` · `变成` · `转变为` · `更新为`

### 2.2 添加/移除动词

`添加` · `增加` · `加入` · `移除` · `删除` · `去掉` · `消除` · `抹去` · `替换掉`

### 2.3 增强/减弱动词

`增强` · `锐化` · `提升` · `加强` · `改善` · `优化` · `减弱` · `柔化` · `降低`

### 2.4 改变指令模板

```
将 [原元素] 替换为 [新元素描述]
在 [位置] 添加 [新元素描述]
移除 [目标元素]，用 [替代内容] 自然填充空白区域
```

## 三、协调词库 (Coordination Vocabulary)

### 3.1 光影协调

`调整光线方向以匹配` · `确保光影一致` · `统一光源方向` · `投射与场景匹配的阴影` · `环境光与主体融合` · `保持光线的自然过渡`

### 3.2 色彩协调

`保持整体色调的协调统一` · `色彩饱和度与 [参考] 一致` · `色调与 [参考] 和谐` · `统一白平衡` · `冷暖调与场景匹配`

### 3.3 材质/纹理协调

`材质反射特性与周围环境一致` · `纹理风格与整体画面融合` · `确保新元素与旧元素的材质风格统一` · `颗粒感/清晰度与原图匹配`

### 3.4 边缘/过渡协调

`主体边缘与背景过渡自然` · `避免抠图感和生硬边缘` · `确保元素之间无缝融合` · `自然过渡，无人工痕迹`

### 3.5 协调指令模板

```
调整 [新元素] 的 [光影/色彩/材质] 使其与原图中的 [参考元素] 协调统一。
确保 [新元素] 与 [旧元素] 之间的过渡自然流畅，无生硬的拼接感。
```

## 四、编辑强度词汇

### 4.1 轻度编辑

`略微` · `稍作` · `微调` · `轻量` · `小幅` · `淡淡地` · `保留原图95%以上`

### 4.2 中度编辑

`明显` · `适度` · `均衡地` · `自然过渡` · `可控范围内`

### 4.3 重度编辑

`完全` · `彻底` · `大幅` · `重新演绎` · `脱胎换骨` · `全面变换` · `只保留基本构图`

## 五、多图参考词汇

### 5.1 图像角色分配

`图一作为 [角色] 参考` · `图二提供 [元素] 参考` · `以图三的 [属性] 为准` · `综合图一和图二的 [元素]`

### 5.2 多图合成指令

`参考图一的 [特征]，换上图二的 [特征]，配上图三的 [背景]` · `提取图一中的 [元素]，与图二中的 [元素] 合成`
