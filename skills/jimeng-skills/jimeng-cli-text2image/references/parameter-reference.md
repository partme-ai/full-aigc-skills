# Parameter Reference — dreamina text2image 完整参数手册

> `dreamina text2image` 命令的完整参数参考。运行前先执行 `dreamina text2image -h` 以确认最新参数。

---

## 必需参数

| 参数 | 类型 | 说明 | 示例 |
|------|------|------|------|
| `--prompt` | string | 文生图提示词（中文效果最佳） | `--prompt="穿汉服的少女站在樱花树下"` |

## 可选参数

### --ratio（宽高比）

| 值 | 方向 | 适用场景 |
|----|------|---------|
| `21:9` | 超宽横幅 | 电影感风景、全景横幅 |
| `16:9` | 宽屏 | 风景、场景、电影感 |
| `3:2` | 标准横向 | 通用摄影比例 |
| `4:3` | 经典横向 | 通用、产品、室内 |
| `1:1` | 正方形 | 头像、社交媒体、产品特写 |
| `3:4` | 经典纵向 | 半身人像、时尚 |
| `2:3` | 纵向 | 全身人像 |
| `9:16` | 竖屏 | 手机壁纸、竖屏人像、故事 |

默认值：模型决定（通常 1:1 或 16:9）

### --model_version（模型版本）

| 值 | 代数 | 支持分辨率 | 特点 |
|----|------|-----------|------|
| `5.0` | 最新旗舰 | 2k, 4k | 综合最强，默认推荐 |
| `4.6` | 上代旗舰 | 2k, 4k | 人像/面部/产品强 |
| `4.5` | 上代 | 2k, 4k | 风景/插画/艺术风格 |
| `4.1` | 上代 | 2k, 4k | 通用稳定 |
| `4.0` | 第四代基础 | 2k, 4k | 图生图最低要求 |
| `3.1` | 第三代升级 | 1k, 2k | 简洁提示词 |
| `3.0` | 第三代基础 | 1k, 2k | 简洁提示词、快速 |

默认值：`5.0`

### --resolution_type（分辨率）

| 值 | 可用模型 | 说明 |
|----|---------|------|
| `1k` | 仅 3.0, 3.1 | 低分辨率，快速出图 |
| `2k` | 所有模型 | 标准分辨率 |
| `4k` | 4.0 及以上 | 高分辨率，细节丰富 |

默认值：模型决定（通常 2k）

### --session（会话ID）

| 值 | 说明 |
|----|------|
| `0` | 默认会话（无需创建） |
| 正整数 | 通过 `dreamina session create "name"` 创建的会话 ID |

### --poll（轮询秒数）

| 值 | 行为 |
|----|------|
| `0` | 异步模式：提交后立即返回 submit_id，不等待完成 |
| `30` | **推荐**：等待最多 30 秒，完成后自动返回结果 |
| `60` | 长轮询：等待最多 60 秒（4K 或复杂生成） |
| `120` | 超长轮询：等待最多 120 秒 |

## 完整命令示例

### 基础生成
```bash
dreamina text2image --prompt="一只布偶猫蹲在窗边，阳光洒在毛发上，浅景深，萌宠摄影"
```

### 指定比例和模型
```bash
dreamina text2image \
  --prompt="穿汉服的少女站在樱花树下，微风拂动发丝，金色夕阳逆光" \
  --ratio=3:4 \
  --model_version=5.0
```

### 全参数4K生成
```bash
dreamina text2image \
  --prompt="赛博朋克城市夜景，霓虹灯在雨夜中闪烁，湿漉漉的街道反射彩色光芒" \
  --ratio=16:9 \
  --model_version=5.0 \
  --resolution_type=4k \
  --poll=60
```

### 异步批量提交
```bash
# 提交不等待
dreamina text2image --prompt="..." --ratio=1:1 --poll=0
# 保存 submit_id，稍后查询
dreamina query_result --submit_id=<id>
```

### 指定会话
```bash
# 先创建会话
dreamina session create "product-shots"
# 在会话中生成（假设返回 session_id=42）
dreamina text2image --prompt="..." --session=42 --ratio=1:1 --poll=30
```

| 建议比例 1:1 | `--ratio=1:1` | 直接映射 |

| 2K 画质 | `--resolution_type=2k` | 直接映射 |
| 4K 画质 | `--resolution_type=4k` | 直接映射 |
| (未指定) | 省略参数 | 使用CLI默认值 |
