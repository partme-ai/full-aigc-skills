# Model Selection Guide — 模型选择指南

> 根据使用场景选择最合适的即梦图像模型版本。

---

## 快速选择

```
用户需要什么？
├── 最高质量，不关心细节 → 5.0（默认）
├── 人物/肖像/产品特写 → 4.6
├── 风景/插画/艺术风格 → 4.5
├── 通用稳定 → 4.0 或 4.1
├── 快速迭代/简洁提示词 → 3.0 或 3.1
└── 图生图编辑 → 4.0+（必须）
```

## 模型详细对比

### 5.0 — 最新旗舰（默认推荐）

| 属性 | 值 |
|------|-----|
| 分辨率 | 2k, 4k |
| 优势 | 综合能力最强，最新算法 |
| 适用 | **几乎所有场景** |
| 提示词 | 支持长短提示词 |
| 速度 | 中 |

**何时使用**: 默认选择。除非有特殊理由，否则使用 5.0。

### 4.6 — 人像/产品专家

| 属性 | 值 |
|------|-----|
| 分辨率 | 2k, 4k |
| 优势 | 人物面部、皮肤纹理、产品材质表现优异 |
| 适用 | 人像摄影、时尚、产品展示、电商 |
| 提示词 | 支持详细提示词 |
| 速度 | 中 |

**何时使用**: 用户需求明确是人像/产品时推荐。

### 4.5 — 风景/艺术专家

| 属性 | 值 |
|------|-----|
| 分辨率 | 2k, 4k |
| 优势 | 风景构图、艺术风格、插画效果好 |
| 适用 | 风景、国风、二次元、概念艺术 |
| 提示词 | 支持较长提示词（100+ tokens） |
| 速度 | 中 |

**何时使用**: 用户需求是风景/艺术/创意类时推荐。

### 4.0 / 4.1 — 稳定基线

| 属性 | 值 |
|------|-----|
| 分辨率 | 2k, 4k |
| 优势 | 稳定可靠，行为可预测 |
| 适用 | 通用场景、图生图最低要求 |
| 提示词 | 中等长度最佳 |
| 速度 | 中快 |

**何时使用**: 需要稳定输出或作为图生图的模型选择。

### 3.0 / 3.1 — 快速迭代

| 属性 | 值 |
|------|-----|
| 分辨率 | 1k, 2k |
| 优势 | 速度快，成本低 |
| 适用 | 简洁提示词（40-80字符）、快速测试 |
| 提示词 | 简短精炼效果更好 |
| 速度 | 快 |
| 限制 | 不支持图生图 |

**何时使用**: 快速迭代测试、简洁提示词、低预算场景。

## 分辨率选择建议

| 场景 | 推荐分辨率 | 理由 |
|------|-----------|------|
| 快速预览/迭代 | 2k（或省略） | 速度与质量平衡 |
| 最终出图/打印 | 4k | 最高细节 |
| 社交媒体分享 | 2k | 足够清晰 |
| 测试提示词效果 | 2k | 节省时间 |
| 人像特写 | 4k | 皮肤纹理更需要高分辨率 |
| 3.0/3.1 模型 | 1k 或 2k | 不支持 4k |
