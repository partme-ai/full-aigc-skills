# Parameter Reference — dreamina text2video 完整参数手册

> `dreamina text2video` 命令的完整参数参考。运行前先执行 `dreamina text2video -h` 确认最新参数。

---

## 必需参数

| 参数 | 类型 | 说明 | 示例 |
|------|------|------|------|
| `--prompt` | string | 视频生成提示词（含运动和运镜描述） | `--prompt="镜头缓缓推近，一只橘猫从沙发上跳下来"` |

## 可选参数

### --duration（时长，秒）

| 值 | 适用场景 |
|----|---------|
| `4`-`6` | 单一清晰动作（如"转身微笑"） |
| `7`-`10` | 动作有发展过程（如"从远到近走来"） |
| `11`-`15` | 2-3阶段动作进程（如"推门→环顾→走向窗边"） |

默认值：`5`

### --ratio（宽高比）

| 值 | 适用场景 |
|----|---------|
| `16:9` | 默认，横屏视频，最通用 |
| `9:16` | 竖屏视频，手机端/短视频 |
| `1:1` | 正方形视频，社交媒体 |
| `3:4` | 竖屏偏宽，适合人像 |
| `4:3` | 经典横屏 |
| `21:9` | 超宽电影感 |

默认值：`16:9`

### --video_resolution（分辨率）

| 值 | 可用模型 | 说明 |
|----|---------|------|
| `720P` | Seedance 2.0 全系列 | 标准分辨率 |

> 注意：官方文档中使用大写 `P`（`720P`）。Seedance 2.0 仅支持 720P。1080P 仅限旧版 3.0/3.5pro 模型。

### --model_version（模型版本）

| 值 | 速度 | 质量 | 说明 |
|----|------|------|------|
| `seedance2.0fast` | 快 | 好 | **默认**，适合迭代和日常使用 |
| `seedance2.0` | 慢 | 极佳 | 高质量最终输出 |
| `seedance2.0fast_vip` | 快 | 好 | VIP 快速版 |
| `seedance2.0_vip` | 慢 | 极佳 | VIP 高质量版 |

### --session（会话ID）

| 值 | 说明 |
|----|------|
| `0` | 默认会话 |
| 正整数 | 会话 ID（通过 `dreamina session create` 创建） |

### --poll（轮询秒数）

| 值 | 行为 |
|----|------|
| `0` | 异步：提交后立即返回 submit_id |
| `60` | **推荐**：视频生成较慢，至少60秒 |
| `120` | 长轮询：适合 seedance2.0 高质量模式 |
| `180` | 超长轮询：15秒视频+高质量 |

**Poll 行为**: 每秒轮询1次。超时返回 "querying" 中间结果（含 submit_id），可用 `query_result` 后续查询。

## 完整命令示例

### 基础视频生成
```bash
dreamina text2video \
  --prompt="镜头侧面跟拍，年轻跑者沿着河边步道匀速奔跑，金色晨光从侧面照亮" \
  --duration=6 \
  --poll=60
```

### 竖屏短视频
```bash
dreamina text2video \
  --prompt="镜头缓缓推近，一位女性在窗前缓缓转身回眸微笑，夕阳金色逆光，发丝飘动" \
  --duration=5 \
  --ratio=9:16 \
  --poll=60
```

### 高质量长视频
```bash
dreamina text2video \
  --prompt="镜头从左向右横摇，展现秋日森林的全景，金色和红色的树叶在风中轻轻摆动，阳光穿透树冠形成丁达尔光柱" \
  --duration=10 \
  --model_version=seedance2.0 \
  --poll=120
```

### 异步提交
```bash
dreamina text2video --prompt="..." --duration=8 --poll=0
# 保存 submit_id，稍后查询
```

## 参数映射速查（从提示词规格到CLI参数）

| 提示词输出 | CLI 参数 | 映射规则 |
|-----------|---------|---------|
| 时长建议 8s | `--duration=8` | 直接映射 |
| 比例 9:16 | `--ratio=9:16` | 直接映射 |
| 高质量要求 | `--model_version=seedance2.0` | 使用慢速高质量模型 |
| 快速迭代 | `--model_version=seedance2.0fast` | 使用默认快速模型 |
| 360度环绕运镜 | 已含在prompt中 | 运镜描述写在 --prompt 里 |
| (未指定时长) | 省略 `--duration` | 使用CLI默认值 5s |
| (未指定比例) | 省略 `--ratio` | 使用CLI默认值 16:9 |
