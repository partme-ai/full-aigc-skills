# Model Selection Guide — Seedance 2.0 模型选择指南

> 根据使用场景选择最合适的 Seedance 2.0 视频生成模型。

---

## 快速选择

```
用户需要什么？
├── 快速测试/迭代提示词 → seedance2.0fast（默认）
├── 最终高质量输出 → seedance2.0
├── VIP + 快速 → seedance2.0fast_vip
└── VIP + 高质量 → seedance2.0_vip
```

## 模型详细对比

### seedance2.0fast — 快速（默认推荐）

| 属性 | 值 |
|------|-----|
| 速度 | 约 2-5 分钟/10秒视频 |
| 质量 | 好 |
| 成本 | 标准 |
| 适用 | 日常使用、提示词迭代、测试 |

**何时使用**: 默认选择。开发和迭代阶段的日常使用。

### seedance2.0 — 高质量

| 属性 | 值 |
|------|-----|
| 速度 | 约 5-15 分钟/10秒视频 |
| 质量 | 极佳 |
| 成本 | 标准（与 fast 相同额度，但生成时间长） |
| 适用 | 最终输出、客户交付、高质量需求 |

**何时使用**: 提示词已经验证可行，需要最高质量的最终输出时。

### seedance2.0fast_vip / seedance2.0_vip — VIP版

| 属性 | 值 |
|------|-----|
| 速度 | 与对应非VIP版相同 |
| 质量 | 与对应非VIP版相同 |
| 成本 | VIP 额度 |
| 适用 | VIP 用户 |

## 场景-模型对照表

| 场景 | 首选模型 | 说明 |
|------|---------|------|
| 测试新提示词 | seedance2.0fast | 快速反馈，迭代效率高 |
| 人物日常视频 | seedance2.0fast | 日常场景用fast足够 |
| 自然风景视频 | seedance2.0 | 风景细节对画质要求高 |
| 产品展示视频 | seedance2.0 | 产品需要最高画质 |
| 电影感叙事 | seedance2.0 | 氛围和画质要求最高 |
| 抽象创意视频 | seedance2.0fast | 创意测试迭代频繁 |
| 最终交付 | seedance2.0 | 确保最高质量 |
| 批量生成 | seedance2.0fast | 需要较快的周转 |

---

## 时长-模型-轮询对照

| 时长 | 推荐模型 | 推荐 --poll | 预计等待 |
|------|---------|------------|---------|
| 5s | seedance2.0fast | 60 | 2-5 分钟 |
| 5s | seedance2.0 | 120 | 5-10 分钟 |
| 10s | seedance2.0fast | 120 | 3-8 分钟 |
| 10s | seedance2.0 | 180 | 8-15 分钟 |
| 15s | seedance2.0fast | 120 | 5-10 分钟 |
| 15s | seedance2.0 | 180 | 10-20 分钟 |

> 这些是估计值。实际时间取决于服务端负载。`--poll` 超时后使用 `query_result` 继续等待。
