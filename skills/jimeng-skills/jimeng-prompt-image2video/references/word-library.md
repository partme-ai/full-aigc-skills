# Word Library — 图生视频词汇库

> I2V 专属词汇——聚焦"运动/静止/过渡/风格一致"。常规运动词汇和运镜词汇请同时参考 text2video 的 `word-library.md` 和 `camera-library.md`。

---

## 一、I2V 核心原则词汇

### 1.1 从参考图开始

`从这张画面开始` · `基于参考图的场景` · `延续参考图的构图` · `从当前这一刻出发` · `画面中的人物开始...`

### 1.2 只描述变化

`接下来发生` · `随后` · `紧接着` · `画面中开始出现` · `渐渐地`

---

## 二、运动描述词汇（指定"什么在动"）

### 2.1 人物微动

`缓缓眨眼` · `嘴角微微上扬` · `发丝轻轻飘动` · `衣角在风中微摆` · `手指轻轻敲击` · `胸口随呼吸微微起伏` · `眼神流转`

### 2.2 自然元素运动

`水面波纹缓缓扩散` · `云层在天空中缓慢移动` · `树叶在微风中沙沙摆动` · `花瓣轻轻飘落` · `烛光摇曳闪烁` · `瀑布水流持续倾泻` · `草地在风中如波浪起伏` · `光影在地面慢慢移动`

### 2.3 物体运动

`车轮缓缓转动` · `烟雾袅袅升起` · `蒸汽从杯中升腾` · `旗帜在风中飘扬` · `时钟指针走动` · `液体在容器中微微晃动`

### 2.4 运动幅度控制

`微微` · `轻轻` · `缓缓` · `略微` · `极其细微的` · `几乎不可察觉的` · `明显的` · `剧烈的` · `大幅度的`

---

## 三、静止声明词汇（指定"什么不动"）

### 3.1 完全静止声明

`保持完全静止` · `纹丝不动` · `严格保持不变` · `锁定不动` · `完全冻结` · `不做任何运动`

### 3.2 部分静止声明

`除了 [运动元素] 以外，其他一切保持静止` · `只有 [运动元素] 在动` · `背景保持完全静止，仅前景中的 [元素] 在运动`

### 3.3 静止声明模板

```
[不需要动的元素列表] 保持完全静止不变。
只有 [需要动的元素] 在 [运动描述]。
```

## 四、过渡描述词汇（首尾帧专用）

### 4.1 过渡方式

`渐变过渡` · `平滑演变` · `由 [A] 逐渐变为 [B]` · `从 [A] 缓慢过渡到 [B]` · `逐步转变` · `层层推进`

### 4.2 过渡节奏

`匀速平滑过渡` · `前半段缓慢积蓄，后半段加速完成` · `由慢到快自然过渡` · `节奏均匀稳步推进`

### 4.3 生长/变形类过渡

`从内向外逐层展开` · `自下而上生长` · `从中心向四周扩散` · `由模糊到清晰显现` · `从含苞到完全盛开`

---

## 五、多帧叙事词汇（多帧故事专用）

### 5.1 帧间关系

`因为 [帧1事件] 所以 [帧2事件]` · `随着时间推移` · `紧接着` · `与此同时` · `在...之后`

### 5.2 时间流逝

`从清晨到正午` · `日影西斜` · `时光流转` · `季节悄然更替` · `岁月在画面中流淌`

### 5.3 镜头串联

`镜头跟随主体的运动轨迹从一个场景过渡到下一个` · `通过匹配剪辑将帧与帧之间自然连接` · `镜头在帧间平滑过渡`

---

## 六、风格一致性词汇

### 6.1 风格锁定

`保持参考图的风格和色调完全一致` · `延续原图的视觉风格` · `不改变参考图的任何视觉特征` · `输出视频与输入图像风格无缝衔接`

### 6.2 色彩/光影一致

`保持参考图的色彩饱和度和色调` · `延续原图的光影方向和强度` · `不引入新的光源或色彩偏移`

---

## 七、子模式特定提示词开头模板

### 单图生视频
```
基于这张参考图生成一段 [时长] 的视频。[运动元素] 开始 [运动描述]。
[静止元素] 保持完全静止。运镜采用 [运镜描述]。保持参考图的风格和色调一致。
```

### 首尾帧
```
以图一为起始帧，图二为结束帧，生成一段 [时长] 的过渡视频。
[元素] 从 [起始状态] 逐渐过渡到 [结束状态]。[过渡节奏描述]。
```

### 多帧故事
```
基于这 [N] 张关键帧生成一段 [时长] 的叙事视频。
[帧间因果关系和时间流逝描述]。镜头 [运镜描述] 串联各帧。
```

### 全能参考
```
综合以下参考素材生成一段 [时长] 的视频：[各素材分工描述]。
所有元素在统一的 [风格/光线/色调] 下自然融合。
```
