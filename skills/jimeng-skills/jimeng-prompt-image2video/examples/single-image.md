# Single Image to Video — 单图生视频

## Instructions

单图生视频是最常用的图生视频模式。核心是从一张静态画面出发，描述"接下来会发生什么运动"。提示词聚焦于运动元素和运动方式，不重复描述参考图已有的视觉内容。

## Scenario Patterns

| 子场景 | 运动核心 | 推荐运镜 | 典型触发词 |
|--------|---------|---------|-----------|
| 人物微动 | 眨眼+发丝飘动+呼吸起伏 | 推/固定 | 让人物活起来, 肖像动起来 |
| 风景微动 | 水流+云动+树叶摆动 | 摇/固定 | 风景动起来, 水面波纹 |
| 产品展示 | 旋转+光影流转 | 环绕/推 | 产品360, 展示视频 |
| 画作动画 | 笔触流动+墨色晕染 | 拉/摇 | 让画动起来, 名画活过来 |
| 光影流转 | 阴影移动+光线变化 | 固定/推 | 光影变化, 时间流逝 |

## Examples

### Example: 人物肖像活起来

**用户需求**：让这张人物肖像照片动起来

**运动拆解**：
- 运动元素: 头发在微风中轻轻飘动；眼睛缓慢眨动一次；嘴角微微上扬
- 静止元素: 身体姿势、服装、背景、光线方向保持完全静止
- 运镜: 镜头极缓推近面部
- 时长: 6秒
- 风格: 完全保持参考图的摄影风格

**最终提示词**：
```
基于这张人物肖像参考图生成一段6秒的视频。人物的头发在微风中轻轻飘动，
眼睛缓慢地眨动一次，嘴角微微上扬露出浅笑，胸口随呼吸有极其细微的起伏。
身体姿势、服装、背景和光线方向保持完全静止不变。
镜头极缓推近面部，逐渐聚焦眼神。完全保持参考图的摄影风格和色调。6秒。
```

## Category-Specific Tips

1. **只写运动，不写画面** — 参考图已经展示了所有视觉内容。提示词100%聚焦于"什么在动、怎么动"
2. **静止声明和运动描述同等重要** — 不声明静止=AI会让一切动起来
3. **微动最自然** — "微微""缓缓""极其细微"的效果远好于大幅度运动
4. **单图生视频最适合静态场景微动化** — 大幅度的运动变化建议用首尾帧或多帧故事
