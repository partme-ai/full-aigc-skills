# Multi-Frame Story — 多帧故事

## Instructions

多帧故事模式用 2-20 张关键帧作为故事板（storyboard），AI 生成连贯的叙事视频。核心是帧与帧之间的**因果关系**和**时间流逝感**——不只是排列顺序，而是有叙事逻辑的连接。

## Scenario Patterns

| 子场景 | 叙事逻辑 | 推荐帧数 | 典型触发词 |
|--------|---------|---------|-----------|
| 产品使用演示 | 步骤1→2→3因果关系 | 3-6帧 | 演示, 使用步骤 |
| 短剧/微剧情 | 起承转合叙事弧线 | 4-8帧 | 短剧, 剧情, 故事 |
| 品牌故事 | 情感递进+时间线 | 4-10帧 | 品牌, 宣传, 故事 |
| 教程/教学 | 步骤因果关系 | 3-8帧 | 教程, 教学, 步骤 |
| 多视角展示 | 不同角度/场景 | 3-6帧 | 多角度, 多场景 |
| 时光流逝 | 同一主体不同时间 | 3-6帧 | 时间流逝, 变化 |

## Examples

### Example: 产品使用演示（4帧）

**用户需求**：用4张产品使用步骤图生成一个演示视频

**叙事拆解**：
- 帧序列: 取出产品 → 涂抹在手背 → 推开吸收 → 肌肤光泽效果
- 帧间关系: 因果递进——每一步是下一步的前提
- 运镜: 镜头跟随手部动作在每个步骤间平滑过渡
- 时长: 12秒
- 节奏: 每个步骤约3秒，过渡自然不跳跃

**最终提示词**：
```
基于这4张产品使用步骤关键帧生成一段12秒的演示视频。
取出产品后，在手背上轻柔涂抹，产品推开后被肌肤吸收，最终展现肌肤的光泽效果。
每一步都是前一步的自然延续。镜头跟随手部动作在帧与帧之间平滑过渡。
整体节奏舒缓清晰，每个步骤停留足够时间供观看者理解。12秒。
```

### Example: 品牌故事（6帧）

**用户需求**：用6张图讲述一个从清晨到深夜的城市故事

**叙事拆解**：
- 帧序列: 黎明空城 → 早高峰人流 → 午后咖啡馆 → 黄昏天际线 → 夜晚霓虹 → 深夜寂静
- 帧间关系: 时间流逝——从清晨到深夜的完整一天
- 运镜: 镜头以缓慢的节奏横摇串联各场景
- 时长: 15秒
- 节奏: 早晨场景舒缓，正午节奏略快，夜晚逐渐放慢

**最终提示词**：
```
基于这6张城市故事关键帧生成一段15秒的叙事视频。
从黎明的空城开始，早高峰的人流涌入，午后在咖啡馆的片刻宁静，
黄昏的天际线逐渐暗下，夜晚霓虹灯闪烁，最后归于深夜的寂静。
镜头以缓慢的节奏横摇串联各场景，时间流逝的感觉贯穿始终。
早晨场景的节奏舒缓，正午略快，夜晚逐渐放慢。15秒。
```

### Example: AI短剧（8帧）

**用户需求**：用8张故事板帧生成一个微剧情短片

**叙事拆解**：
- 帧序列: 女主窗边沉思 → 收到信息 → 惊喜表情 → 匆忙换装 → 奔跑出门 → 街头寻找 → 看到对方 → 相视而笑
- 帧间关系: 情绪递进——沉思→惊喜→急切→喜悦
- 运镜: 跟随女主的情绪和运动节奏切换景别

**最终提示词**：
```
基于这8张故事板关键帧生成一段15秒的微剧情短片。
女主在窗边沉思时收到一条信息，表情从惊讶转为惊喜，匆忙换装后奔跑出门，
在街头焦急地寻找，最终在人群中看到对方，两人相视而笑。
镜头跟随女主的情绪节奏——沉思时缓慢推进，奔跑时快速跟拍，
相遇时缓缓推近两人的笑容。前段节奏紧凑，后段舒缓温暖。15秒。
```

### Example: 手工艺制作过程（5帧）

**用户需求**：用5张陶艺制作步骤图生成一个手工视频

**叙事拆解**：
- 帧序列: 揉泥 → 拉坯成型 → 修坯 → 上釉 → 烧制完成
- 帧间关系: 工艺递进——每一步是下一步的物质前提
- 运镜: 镜头以固定机位为主，在关键步骤轻微推近展示细节
- 时长: 15秒
- 节奏: 前两步舒缓展示手部动作，拉坯和修坯步骤节奏加快，最后烧制完成放慢

**最终提示词**：
```
基于这5张陶艺制作步骤关键帧生成一段15秒的手工艺视频。
陶艺师先揉泥排除气泡，然后在拉坯机上将泥团拉成花瓶形状，
接着修坯削薄器壁，再浸入釉料中均匀上釉，最后从窑中取出烧制完成的
青釉花瓶。每个步骤是下一步的物质前提。镜头以固定机位为主，
在拉坯和上釉步骤轻微推近展示手部细节。前段舒缓、中段紧凑、后段放慢。15秒。
```

## Category-Specific Tips

1. **帧与帧之间必须有叙事逻辑** — 不仅是时间先后，而是"因为A所以B"的因果或情感递进
2. **帧密度决定节奏** — 关键帧越密集=节奏越快，越稀疏=节奏越慢
3. **关键帧的视觉信息比提示词更重要** — 多帧故事模式中，帧本身承载了大部分画面信息，提示词聚焦于连接逻辑
4. **2-20帧范围大，但建议4-8帧最优** — 太少缺乏叙事、太多AI难以保持一致性
5. **每帧之间的一致性比单帧质量更重要** — 风格、色调、人物特征在帧间保持一致，否则输出视频会有跳跃感
