# Multi-Modal Reference — 全能参考

## Instructions

全能参考是即梦图生视频的最强模式——支持最多 9 张图片 + 3 段视频 + 3 段音频同时作为参考输入。核心是**明确每份参考素材的分工**，让不同素材分别控制视频的不同维度。

## Scenario Patterns

| 子场景 | 参考分工 | 典型触发词 |
|--------|---------|-----------|
| 人物+场景分离 | 图一=人物, 图二=场景 | 把这个人放到那个场景 |
| 风格迁移视频 | 图=内容, 视频=运动风格 | 用这个风格做视频 |
| 口型同步 | 图=人物, 音频=语音 | 对口型, 说话 |
| 多角度合成 | 多张图=不同角度参考 | 多角度, 综合参考 |
| 音乐MV | 图=画面, 音频=音乐节奏 | MV, 配乐, 节奏 |

## Examples

### Example: 人物+场景分离控制

**用户需求**：用图一的人物，配上图二的场景，生成视频

**参考分工**：
- 图一: 提供人物的面部特征、服装和姿势
- 图二: 提供场景的环境、色调和氛围（秋日森林）
- 合成: 人物站在森林中，光线从图二的树冠中穿透，人物身上投射斑驳光影
- 运动: 微风吹动人物的头发和衣角，树叶在风中轻轻摆动
- 时长: 8秒

**最终提示词**：
```
综合以下参考素材生成8秒视频。图一提供人物的面部特征、服装和姿势；
图二提供秋日森林的场景环境和暖金色调。人物站在图二的森林中，
阳光从树冠穿透形成斑驳光影投射在人物身上。微风吹动人
物的头发和衣角轻轻飘动，树叶在风中微微摆动。所有元素的光源方向
和色调统一于图二的暖秋氛围。8秒。
```

### Example: 风格+运动参考

**用户需求**：用图片作为画面参考，用视频的运动风格来驱动动画

**参考分工**：
- 图一: 提供画面内容和构图
- 视频参考: 提供运镜方式和运动节奏（手持纪录片风格）
- 合成: 图一的画面内容按照视频参考的运动风格动起来
- 时长: 8秒

**最终提示词**：
```
综合以下参考素材生成8秒视频。图一提供画面内容和构图；
视频参考提供运镜方式和运动节奏——手持纪录片风格的自然晃动和
缓慢的推近运动。将视频参考的运动风格应用到图一的画面内容上。
画面在图一的构图上，以手持摄影的呼吸感逐渐推近主体。
保持图一的色调和风格完全不变。8秒。
```

### Example: 音乐节奏驱动的多图MV

**用户需求**：用5张旅行照片和一首音乐做一个旅行MV

**参考分工**：
- 图1-5: 旅行照片——不同场景（海滩、山峦、古城、美食、日落）
- 音频: 提供音乐的情绪基调和节奏
- 合成: 5张图按照音乐节奏依次出现并微动，形成旅行回顾MV
- 时长: 15秒

**最终提示词**：
```
综合以下参考素材生成一段15秒的旅行回顾视频。图一至图五提供
五个不同旅行场景的画面。音频提供轻松愉悦的音乐情绪和节奏基调。
五张图按照音乐的节奏依次出现——每张图停留约3秒并进行微动：
海滩画面中浪花轻轻拍岸，山峦画面中云雾缓缓飘移，
古城中旗帜微微飘扬，美食画面中热气升腾，日落中光影缓缓变化。
各场景之间以柔和的过渡连接。整体色调温暖统一。15秒。
```

### Example: 口型同步（图+音频）

**用户需求**：用这张人物照片和一段语音，做一个说话的视频

**参考分工**：
- 图一: 提供人物的面部特征、表情和整体外观
- 音频: 提供语音内容和说话的情绪节奏
- 合成: 人物按照音频的节奏自然说话，口型与语音同步
- 运动: 嘴唇随语音开合、面部微表情、眨眼、轻微点头
- 时长: 音频长度决定

**最终提示词**：
```
综合以下参考素材生成口型同步视频。图一提供人物的面部特征和外观；
音频提供语音内容和说话的情绪节奏。人物按照音频的节奏自然说话，
口型与语音内容同步。伴随自然的眨眼、面部微表情和轻微的头部动作。
身体和背景保持相对静止。保持图一的人物特征和光线环境完全不变。
视频长度与音频长度一致。
```

## Category-Specific Tips

1. **每份素材必须有明确分工** — 含糊的"综合这些"几乎不会成功。必须写"图一提供X，图二提供Y"
2. **视频参考=运动模板** — 这是全能参考模式独有的优势。用视频告诉AI"像这样动"
3. **音频=节奏控制器** — 音乐的BPM直接决定视频的剪辑和运动节奏
4. **图+视频+音频=最强三件套** — 画面靠图、运动靠视频参考、节奏靠音频，三维分离控制实现最高精度
5. **素材数量有上限** — 最多9张图+3段视频+3段音频。超出限制会报错
6. **色调统一是关键** — 多来源素材的色调差异是最大挑战，务必写"所有元素的色调统一"
