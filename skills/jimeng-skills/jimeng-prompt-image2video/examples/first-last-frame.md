# First-Last Frame — 首尾帧生视频

## Instructions

首尾帧模式用两张图定义视频的起点和终点，AI 生成从首帧到尾帧的过渡动画。核心是描述"如何从图A变到图B"的过渡过程。

## Scenario Patterns

| 子场景 | 过渡核心 | 推荐节奏 | 典型触发词 |
|--------|---------|---------|-----------|
| 花开花落 | 从含苞→盛开 或 盛开→凋谢 | 缓慢积蓄+加速 | 花开, 绽放, 花落 |
| 季节更替 | 同一场景的夏→秋/冬→春 | 匀速渐变 | 季节变换, 春夏秋冬 |
| 日出日落 | 天色从暗→明 或 明→暗 | 先慢后快 | 日出, 日落, 昼夜 |
| 建筑/建造 | 从地基→建成 或 废墟→修复 | 均匀推进 | 建造, 变化, 施工 |
| 年龄变化 | 同人物从年轻→年老 | 均匀渐变 | 变老, 岁月, 时光 |
| 场景切换 | 从地点A→地点B（风格/空间过渡） | 变速过渡 | 切换, 转场, 穿越 |

## Examples

### Example: 花苞绽放

**用户需求**：图一是花苞，图二是盛开的花，生成绽放过程

**过渡拆解**：
- 起始状态: 深红色花苞紧裹（图一）
- 结束状态: 牡丹完全盛开，花瓣层层舒展（图二）
- 过渡方式: 外层的花瓣先展开，内层花瓣随后一层一层由外向内逐层绽放
- 过渡节奏: 前1/3缓慢积蓄力量，中间1/3加速展开，后1/3花瓣完全舒展
- 时长: 10秒

**最终提示词**：
```
以图一花苞为起始帧，图二盛开的牡丹为结束帧，生成一段10秒的绽放过渡视频。
外层花瓣先向外翻开，内层花瓣随后一层接一层由外向内逐层绽放，
花蕊从中心逐渐显露。过渡节奏为：前段缓慢积蓄，中段加速展开，
后段花瓣完全舒展到位。镜头固定微微推近花心。10秒。
```

### Example: 夏秋季节更替

**用户需求**：图一是夏天的树，图二是秋天的树，生成季节变换

**过渡拆解**：
- 起始状态: 夏季——满树翠绿（图一）
- 结束状态: 秋季——金黄和红色的树叶（图二）
- 过渡方式: 绿色从叶缘开始逐渐转为金黄，然后向红色过渡，最后几片叶子飘落
- 时长: 10秒

**最终提示词**：
```
以图一的夏季绿树为起始帧，图二的秋季红叶为结束帧，生成一段10秒的季节过渡视频。
树叶的绿色从叶缘开始逐渐转为金黄，随后部分叶子进一步转为红色。
过渡过程中几片叶子飘落到地面。整个季节转换平滑自然。
镜头固定，光影从夏季的高角度逐渐变为秋季的低角度暖光。10秒。
```

### Example: 日出场景

**用户需求**：图一是黎明前的黑夜，图二是日出后的金色天空，生成日出过程

**过渡拆解**：
- 起始状态: 黎明前的深蓝黑夜（图一）
- 结束状态: 日出后金色阳光照亮天际（图二）
- 过渡方式: 地平线先泛起鱼肚白 → 天空从深蓝渐变到橙红 → 太阳从地平线升起
- 时长: 10秒

**最终提示词**：
```
以图一黎明前的深蓝黑夜为起始帧，图二日出后金色天空为结束帧，生成一段10秒的
日出过渡视频。地平线先泛起鱼肚白，天空从深蓝缓慢渐变到橙红色，
太阳从地平线上缓缓升起，金色阳光逐渐洒满大地。光影从暗到明逐渐过渡，
前半段缓慢积蓄（夜色褪去），后半段加速（阳光涌现）。镜头固定。10秒。
```

### Example: 建筑从废墟到修复

**用户需求**：图一是废墟，图二是修复后的建筑，生成修复过程

**过渡拆解**：
- 起始状态: 残破的古建筑废墟，断壁残垣（图一）
- 结束状态: 完整修复的古建筑，雕梁画栋（图二）
- 过渡方式: 碎石从地面升起归位到墙体，梁柱从断裂到完整连接，彩绘从褪色到鲜艳重现
- 过渡节奏: 从底部到顶部逐层修复，节奏匀速稳步推进
- 时长: 12秒

**最终提示词**：
```
以图一古建筑废墟为起始帧，图二完整修复的建筑为结束帧，生成一段12秒的
修复过渡视频。碎石从地面升起归位到墙体中，断裂的梁柱逐渐连接完整，
墙面的彩绘从褪色斑驳恢复到鲜艳清晰。从建筑底部到顶部逐层修复，
节奏匀速稳步推进。镜头固定展现整个修复过程。12秒。
```

## Category-Specific Tips

1. **过渡方式是核心** — 不是"变成"，而是"花瓣从外向内逐层展开""绿色从叶缘开始渐变"
2. **节奏控制自然感** — 自然的过渡很少是匀速的。花开的节奏是先慢后快再慢
3. **两张图的风格/色彩必须协调** — 如果两张图的色调差异太大，过渡会不自然
4. **镜头通常固定** — 首尾帧模式下，建议镜头保持固定或极轻微运动，让画面内容的变化成为主角
