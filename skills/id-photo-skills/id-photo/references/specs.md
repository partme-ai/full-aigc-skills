## 拍照规范

1. **面部居中**：头部占画面高度约 45-50%
2. **正面免冠**：不戴帽子，露出双耳
3. **表情自然**：闭嘴，眼睛睁开，不露齿笑
4. **光线均匀**：无阴影遮挡面部
5. **背景纯净**：单一颜色无渐变
6. **服装整齐**：建议深色有领上衣
7. **无美颜滤镜**：保持真实面部特征
